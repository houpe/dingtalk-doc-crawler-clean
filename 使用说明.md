# 钉钉文档爬取项目

## 目录结构

```
dingtalk-doc-crawler/
├── src/                    # Python 脚本
│   ├── crawler.py          # 钉钉文档爬虫
│   ├── reformat_md.py      # MD 规则引擎优化
│   ├── optimize_md_deepseek.py  # AI 语义优化（DeepSeek）
│   └── pipeline.py         # 完整流程编排
├── assets/                 # 静态资源
│   └── favicon.png         # 网站图标
├── requirements.txt        # Python 依赖
│
├── ai/                     # AI 功能模块
│   ├── MaxKB/
│   └── sync-service/
│
├── docs/                   # 已生成的文档（VitePress 构建源）
├── output/                 # 原始下载数据
├── site/                   # VitePress 构建产物和配置文件
├── sync-service/           # MaxKB 文件同步服务
└── tests/                  # 测试脚本
```

## 快速使用

```bash
# 运行爬虫
python3 src/crawler.py

# 执行完整流程
python3 src/pipeline.py
```
