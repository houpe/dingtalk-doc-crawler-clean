# 系统访问及APP下载

## 一、适用场景

首次使用鲸天系统、鲸小宝或司机端小程序的用户，需要下载宝盒、访问系统或登录小程序时，可参考本文完成操作。

## 二、前置条件

- 上级已按你的手机号在鲸天系统开通账号（若未开通，登录宝盒时会提示“未办理入职”）。
- 鲸天系统添加员工账号后，会自动同步入职信息到宝盒，同步完成后才可登录宝盒及鲸天系统。

## 四、操作步骤

1. **安装宝盒**：根据上方“中通宝盒下载”任选一种方式完成安装。
2. **登录宝盒**：使用上级已开通的手机号登录宝盒。若提示“未办理入职”，请联系上级开通鲸天系统账号，并等待同步完成后重试。
3. **访问鲸天系统**：打开谷歌浏览器，访问鲸天系统链接，使用宝盒账号扫码或密码登录。
4. **下载鲸小宝**（如需）：安装鲸小宝后使用宝盒账号登录。
5. **登录司机端小程序**（如需）：微信扫描小程序码或搜索小程序，按指引完成登录。

## 五、操作结果

- 宝盒登录成功后，可在鲸天系统正常鉴权。
- 鲸天系统访问成功，显示系统首页。
- 鲸小宝、司机端小程序登录成功，进入对应工作界面。

## 六、注意事项

::: warning 账号同步
鲸天系统添加员工账号后，需等待宝盒自动同步入职信息（通常为即时同步），同步完成后才能登录宝盒。若长时间未同步，可联系上级或IT排查。
:::

::: danger 浏览器要求
请务必使用 **谷歌浏览器** 访问鲸天系统，避免因浏览器兼容性问题导致页面异常。
:::

## 七、常见问题

**Q：登录宝盒时提示“未办理入职”怎么办？**
A：说明你的手机号尚未在鲸天系统开通账号。请联系上级或对应人事在鲸天系统中添加员工账号，添加成功后等待宝盒同步信息，然后重新登录。
