# 司机端注册

## 一、适用场景

司机需要在网络货运平台接单时，先注册并认证成为平台的承运司机。

## 二、前置条件

- **账号与权限要求**：使用个人手机号码注册。
- **物理/环境准备**：准备一部 **安卓** 或 **iOS** 智能手机，确保网络正常（4G/5G/Wi-Fi）。
- **配套工具下载**：
  - 👉 [点击下载中通冷运司机端 APP（安卓）](https://zmas.zto.com/download/NTMS)
  - 下载鲸小宝 APP（iOS）：敬请期待。

## 三、操作入口

- **注册入口**：打开 APP → 使用 **手机号码 + 验证码** 登录即完成注册。
- **认证入口**：登录 APP → 点击 **我的** → 点击 **立即认证**。

## 四、操作步骤

### 4.1 场景一：司机注册

1. 打开 APP，输入手机号码，获取并填写验证码后登录，即完成注册。

### 4.2 场景二：司机认证

1. **填写认证信息**
   进入认证页面后，填写 **姓名、身份证、驾驶证** 等信息。上传 **身份证** 和 **驾驶证** 照片后，系统会自动识别并填入。请仔细核对，避免识别错误导致信息有误。

   ![Gemini_Generated_Image_bjl54nbjl54nbjl5.png](./images/司机端注册操作说明_1_ad777e38.png)

2. **完成人脸识别认证**（功能开发中）
   提交资料后，会进行人脸识别，验证提交证件与本人是否一致。

3. **填写车辆信息**
   上传车辆相关图片（如行驶证），系统自动识别后请仔细核对，避免识别错误。

   ![Gemini_Generated_Image_emte92emte92emte.png](./images/司机端注册操作说明_2_5d7f12d2.png)

4. **完成认证，等待平台审核**
   提交认证后，平台将在 **1～3个工作日** 内完成审核，请关注审核进度。

   ![image.png](./images/司机端注册操作说明_3_3281b260.png)

## 五、操作结果

注册并认证完成后，司机账号即具备在网络货运平台接单的资格。认证审核通过后，可在 APP 中查看认证状态为“已通过”。

## 六、注意事项

::: danger 重点提醒
- 上传的证件照片必须清晰、完整，系统自动识别后务必逐项核对，避免因识别错误导致审核不通过。
- 根据法规要求，平台需严格监管身份证、驾驶证、行驶证、从业资格证、道路运输许可证、道路运输经营许可证等证件的有效期；挂车需分别对车头和挂车进行双重监管。
:::

::: warning 注意事项
- 人脸识别功能当前为 **开发中**，上线时间请以 APP 实际版本为准。
- 认证审核周期为 **1～3个工作日**，请耐心等待，不要重复提交。
:::

## 七、常见问题

**Q：为什么要填写那么多信息？**
A：根据网络货运承运平台的国家规定，平台需要对在平台承运的司机进行多项证件（身份证、驾驶证、行驶证、从业资格证、道路运输许可证、道路运输经营许可证等）及有效期进行严格监管，挂车还需分别针对车头和挂车双重监管。

**Q：审核不通过怎么办？**
A：请检查提交的信息是否完整且与图片一致。若无误，请联系客服处理。

**Q：一直显示“审核中”怎么办？**
A：正常等待时间为1～3个工作日，超时请联系客服处理。
