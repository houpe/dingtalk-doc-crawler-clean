# 司机端注册

## 一、业务场景与名词解释

### 1.1 业务场景（为什么用？）

- 司机在网络货运接单，需要首先注册并认证成为一个网络货运平台的司机

### 1.2 核心名词解释（不迷路）

- <strong>司机</strong>：指在中通冷运网络货运平台上的承运方
- <strong>网络货运平台</strong>：即中通冷运网络货运平台，是专为网点、省公司、生态圈以及KA客户等解决整车业务场景的网络货运承运平台

## 二、前置准备与环境配置

- <strong>账号与权限要求</strong>：司机个人手机号码注册
- <strong>物理/环境准备</strong>：准备一部安卓或 iOS 系统的智能手机，确保网络连接正常（4G/5G/Wi-Fi）；

## 三、场景化标准操作步骤

### 3.1 场景一：[司机注册]

- <strong>系统功能路径</strong>：登录APP ->使用手机号码+验证码登录即完成注册

### 3.2 场景二：[司机认证]

- <strong>系统功能路径</strong>：登录APP ->我的->立即认证

### 3.3 核心操作步骤

1. <strong>[填写认证信息]</strong> 进入页面后，填写认证相关信息（姓名、身份证、驾驶证等），上传身份证、驾驶证照片后，系统会自动识别填写，请仔细核对填写内容，避免系统识别错误导致信息有误；

![Gemini_Generated_Image_bjl54nbjl54nbjl5.png](./images/司机端注册操作说明_1_ad777e38.png)

2. <strong>[完成人脸识别认证]</strong> 提交资料后，会进行人脸识别认证司机提交的证件是否与本人一致（开发中）

3. <strong>【填写车辆信息】</strong>

填写车辆信息，可以通过上传图片，系统自动识别，请仔细核对填写内容，避免系统识别错误导致信息有误；

![Gemini_Generated_Image_emte92emte92emte.png](./images/司机端注册操作说明_2_5d7f12d2.png)

<strong>4.【完成认证等待平台审核】</strong>

认证提交后，平台会在1-3个工作日内完成审核，请关注审核额进度

## 五、高频常见问题（FAQ）

- <strong>Q1：[为什么要填写那么多信息？]</strong>
- <strong>A</strong>：根据网络货运承运平台的国家规定，平台需要对在平台承运的司机进行身份证、驾驶证、行驶证、从业资格证、道路运输许可证、道路运输经营许可证等证件以及有效期进行严格监管，且挂车需要分别针对车头和挂车进行双重监管；
