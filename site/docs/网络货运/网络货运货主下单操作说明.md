# 网络货运货主下单

## 一、适用场景

货主注册后，在网络货运平台发布整车订单（录单操作）。

## 二、前置条件

- **账号与权限**：需包含中通冷运租户权限（如网点、省公司），若无权限请联系系统管理员。
- **设备与环境**：一台可联网的电脑，推荐使用 Chrome 浏览器。
- **系统入口**：[https://zc.ztocc.com/dashboard](https://zc.ztocc.com/dashboard)

## 三、操作入口

登录系统 → 顶部 tab 切换至 **中通冷运** → 左侧边栏 **货源管理** → **货源订单** → **整车订单**

## 四、操作步骤

1. **填写发件信息和收件信息**
   填写联系人、联系方式、省市区和详细地址。
   ![image.png](./images/司机端注册操作说明_1_ad777e38.png)

2. **填写服务信息**
   - 填写 **装货时间** 和 **卸货时间**（考核司机到达时间）。
   - 选择所需 **车型** 及 **温度要求**。
   - 司机调度：可选常用司机或线下约定司机直接派单（跳过抢单）。
   - 移动设备：如司机无温控设备，货主可提供便携温控设备（需单独授权）以监控温度和轨迹。
   - 保证金：为避免司机爽约，货主可要求司机缴纳保证金（**20~500 元**），运单完成后退还司机。
   ![image.png](./images/司机端注册操作说明_2_5d7f12d2.png)

3. **填写货物信息**
   填写货物类型、包装类型、**重量**、**体积** 和 **件数**。

   ::: danger 重点提醒
   货物重量不得超过所选车型的最大额定载重（4 米 2 车型除外）。
   :::
   ![image.png](./images/司机端注册操作说明_3_3281b260.png)

4. **财务信息**
   - **支付类型**：目前仅支持 **预付款**（后续会开启现付）。
   - **开票**：目前平台统一开票，支持开票业务。
   - **运费**：请录入 **不含税运费**。
   - **平台服务费**：平台向货主收取（根据单量比例不同，目前最高 **5%**）。
   - **保价**：平台为每单购买货险，按保价金额（货物价值）的 **万分之五** 收取。为保障货物安全，请足额填写货物价值。
   ![image.png](./images/省公司三方车下单操作说明_1_b723c91c.png)

5. **保存下单**
   点击 **保存**，系统自动扣除预付款账户余额，生成运输合同，代表下单成功，并生成订单号。
   ![image.png](./images/省公司三方车下单操作说明_2_ba076f40.png)

## 五、操作结果

下单成功后，系统生成订单号，并自动扣除预付款账户余额，视为平台确认承运。

## 六、注意事项

- 货物重量不得超过所选车型最大载重（4 米 2 除外）。
- 预付款余额不足时，下单会失败，需前往账户管理充值。
- 平台服务费最高 5%，保价费按货物价值的万分之五收取。
- 便携式温控设备需单独授权。
- 保证金（20~500 元）在运单完成后退还司机。
