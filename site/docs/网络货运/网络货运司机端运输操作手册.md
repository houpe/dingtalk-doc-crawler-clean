# 网络货运司机端运输操作手册

## 一、适用场景

司机在中通冷运网络货运平台完成运输任务时，使用司机端 APP 进行装货打卡、卸货打卡、上传凭证等操作，确保运输真实有效，以便结算运费。

## 二、前置条件

- **账号与权限**：使用司机个人手机号码注册并登录。
- **物理/环境准备**：一部安卓或 iOS 系统的智能手机，确保网络连接正常（4G/5G/Wi-Fi）。
- **配套工具下载**：
  - 👉 [点击下载中通冷运司机端 APP（安卓）](https://zmas.zto.com/download/NTMS)
  - 下载鲸小宝 APP（iOS）：敬请期待

## 三、操作入口

登录 APP → **订单中心** → **运单详情页**。

## 四、操作步骤

请按顺序完成以下操作：

1. **装货地打卡**
   - 到达装货地后，点击 **【到达装货地】** 完成打卡。
   - 若因 GPS 模糊或网络问题无法打卡，可上传水印相机照片作为证明完成打卡。
   - ![image.jpeg](./images/司机端注册操作说明_1_ad777e38.png)![image.jpeg](./images/司机端注册操作说明_2_5d7f12d2.png)

2. **装货完成**
   - 货主装货完成后，司机需上传 **2 张照片**：
     - 装货完成水印照片
     - 人车合照（包含车牌）水印照片
   - 点击 **【确定】** 即可完成装货。
   - ![image.png](./images/司机端注册操作说明_3_3281b260.png)![image.png](./images/省公司三方车下单操作说明_1_b723c91c.png)

3. **卸货地打卡**
   - 到达卸货地后，点击 **【到达卸货地】** 完成打卡。
   - 若因 GPS 模糊或网络问题无法打卡，可上传水印相机照片作为证明完成打卡。
   - ![image.png](./images/省公司三方车下单操作说明_2_ba076f40.png)![image.png](./images/省公司三方车下单操作说明_3_c3d4a014.png)

4. **卸货完成**
   - 货主卸货完成后，司机需上传 **3 张照片**：
     - 卸货完成水印照片
     - 人车合照（包含车牌）水印照片
     - 货主收货证明单据照片
   - 点击 **【确定】** 即可完成卸货。
   - ![image.png](./images/省公司三方车下单操作说明_4_23bd7c3b.png)

5. **等待结算**
   - 卸货完成后，在 **【我的】** → **【账户信息】** 页面维护银行卡信息，否则无法结算运费。
   - 提醒货主在平台确认完成，平台审核通过后，运费将自动支付到司机维护的银行卡。
   - ![我的.jpeg](./images/省公司三方车下单操作说明_5_c2de3071.png)

## 五、操作结果

完成所有步骤后，运输任务状态更新为“等待结算”，运费将在货主确认且平台审核通过后自动支付至司机银行卡。

## 六、注意事项

::: danger 重点提醒
- 装货完成和卸货完成上传的照片必须包含水印，且人车合照需清晰显示车牌。
- 若无法正常打卡，必须上传水印照片作为证明，否则可能导致运输任务无法正常流转。
- 卸货完成后需及时在 **【我的】→【账户信息】** 维护银行卡信息，否则无法结算运费。
:::

::: warning 注意事项
- 打卡范围受 GPS 定位影响，请确保到达指定区域后再操作。
- 运输过程中需保持网络稳定（4G/5G/Wi-Fi），避免因信号问题影响操作。
:::

## 七、常见问题

**Q1：什么时候结算运费？**

A：司机运输完成后，货主在平台确认完成，同时平台核实运输过程中没有失温、破损、轨迹偏离等异常情况，审核通过后直接打款至司机银行卡。

**Q2：无法打卡怎么办？**

A：请确认是否已到达打卡围栏范围。若已到达但仍无法打卡，尝试关闭再开启蓝牙或网络，或上传水印相机照片作为证明。
