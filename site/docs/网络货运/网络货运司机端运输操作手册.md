# 一. 网络货运司机端运输操作手册

## 1.1 业务场景与名词解释

### 1.1.1 业务场景（为什么用？）

- 在网络货运平台运输中，司机需要操作司机端进行对应运输操作，从而确保司机的运输工作真实有效，能够帮助司机更快的结算运费

### 1.1.2 核心名词解释（不迷路）

- <strong>司机</strong>：指在中通冷运网络货运平台上的承运方
- <strong>网络货运平台</strong>：即中通冷运网络货运平台，是专为网点、省公司、生态圈以及KA客户等解决整车业务场景的网络货运承运平台

## 1.2 前置准备与环境配置

- <strong>账号与权限要求</strong>：司机个人手机号码注册
- <strong>物理/环境准备</strong>：准备一部安卓或 iOS 系统的智能手机，确保网络连接正常（4G/5G/Wi-Fi）；
- <strong>配套工具/链接</strong>：🛠️ <strong>核心组件下载</strong>：

## 1.3 场景化标准操作步骤

### 1.3.1 场景一：[司机运输操作]

- <strong>系统功能路径</strong>：登录APP ->订单中心->运单详情页

### 1.3.2 核心操作步骤

1. <strong>[装货地打卡]</strong> 到达装货地时，司机点击【到达装货地】，即可完成打卡；若由于GPS模糊或者网络问题导致无法打卡，则可以通过上传水印相机图片证明，完成打卡

![image.jpeg](./images/司机端注册操作说明_1_ad777e38.png)![image.jpeg](./images/司机端注册操作说明_2_5d7f12d2.png)

<strong>2.[装货完成]</strong>在货主装货完成后，司机需要上传2张照片：装货完成水印照片，人车合照（包含车牌）水印照片；点击【确定】即可装货完成；

![image.png](./images/司机端注册操作说明_3_3281b260.png)![image.png](./images/省公司三方车下单操作说明_1_b723c91c.png)

<strong>3.[卸货货地打卡]</strong>到达卸货地时，司机点击【到达卸货地】，即可完成打卡；若由于GPS模糊或者网络问题导致无法打卡，则可以通过上传水印相机图片证明，完成打卡

![image.png](./images/省公司三方车下单操作说明_2_ba076f40.png)![image.png](./images/省公司三方车下单操作说明_3_c3d4a014.png)

<strong>4.[卸货完成]</strong>在货主卸货完成后，司机需要上传3张照片：卸货完成水印照片、人车合照（包含车牌）水印照片、货主收货证明单据照片；点击【确定】即可卸货完成；

![image.png](./images/省公司三方车下单操作说明_4_23bd7c3b.png)

<strong>5.【等待结算】</strong>

司机卸货完成后，在【我的】-【账户信息】页面，维护银行卡信息，否则无法结算运费；司机可以提醒货主在系统确认完成，货主确认后，平台审核通过，则会自动支付运费到司机维护的银行卡；

![我的.jpeg](./images/省公司三方车下单操作说明_5_c2de3071.png)

## 1.5 高频常见问题（FAQ）

- <strong>Q1：[什么时候结运费？]</strong>
- <strong>A</strong>：在司机运输完成后，货主在平台确认完成，同时平台核实运输过程中没有失温、破损、轨迹偏离等异常情况，会审核通过，直接打款；
