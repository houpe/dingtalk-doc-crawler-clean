# 网络货运货主注册

## 一、适用场景

网点的整车业务需要通过网络货运平台找车时，需在 **中通冷运网络货运平台** 完成货主注册。

::: tip 核心名词
- **货主**：指在中通冷运网络货运平台上的发货方。
- **网络货运平台**：即中通冷运网络货运平台，专为网点、省公司、生态圈及 KA 客户等解决整车业务场景的网络货运承运平台。
:::

## 二、前置条件

- **账号与权限**：当前账号需包含 **中通冷运租户权限**（例如网点/省公司角色）。若无权限，请联系系统管理员。
- **物理/环境准备**：一台可联网的电脑，推荐使用 Chrome 浏览器。
- **系统入口**：[官方登录地址](https://zc.ztocc.com/dashboard)

## 三、操作入口

登录系统后，按以下路径进入货主注册页面：

1. 点击顶部 Tab 切换至 **【中通冷运】**。
2. 在左侧侧边栏依次点击 **【基础资料】** → **【货主管理】** → **【企业货主】**。

## 四、操作步骤

1. **新增货主**
   进入 【企业货主】 页面后，点击列表右上角的 **【新增】** 按钮，跳转至新增货主资料页面，填写相关信息。

   ![image.png](./images/司机端注册操作说明_1_ad777e38.png)

2. **等待平台审核**
   提交资料后，系统会自动同步给平台进行审核。请关注审核状态，平台会在 **3 个工作日内** 完成审核。审核期间可编辑修改资料，修改后会重新提交审核。

   ![image.png](./images/司机端注册操作说明_2_5d7f12d2.png)

3. **审核通过后，维护税务信息**
   审核通过后，为方便开票，请及时维护税务信息。点击 **【编辑】**，切换 Tab 至 **【开票信息管理】**，然后点击 **【新增】** 填写税务信息资料。

   ![image.png](./images/司机端注册操作说明_3_3281b260.png)

## 五、操作结果

货主注册审核通过后，即可正常使用网络货运平台进行找车、发单等操作。税务信息维护完成后可正常开具发票。

## 六、注意事项

::: warning 注意事项
- 平台审核时间为 **3 个工作日内**，请耐心等待。
- 审核期间允许编辑修改资料，每次修改后需重新提交审核。
- 审核通过后务必及时维护 **开票信息（税务信息）**，以免影响后续开票。
:::

::: danger 重点提醒
- 新增货主时填写的资料必须真实、完整，否则可能导致审核不通过。
- 只有审核通过的货主才能使用平台进行后续操作。
:::

## 七、常见问题

| 异常现象 | 常见原因 | 解决方案 |
| -------- | -------- | -------- |
| 提示“无访问权限” | 当前账号未被授权中通冷运租户权限 | 联系技术支持或通过小吉工单申请中通冷运权限 |

**Q：我作为货主，如何快速了解需要做哪些事情？**
A：点击侧边栏 **【货主首页】**，可在首页查看货主主要功能及操作流程，根据流程引导即可完成货主相关操作。
