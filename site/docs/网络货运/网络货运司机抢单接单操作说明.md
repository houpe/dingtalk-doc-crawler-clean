# 网络货运司机抢单接单

## 一、适用场景

本文适用于司机在**中通冷运网络货运平台**进行接单操作，主要包含以下两种场景：

1. **司机抢单**：司机在**货源大厅**查找匹配的订单并抢单。
2. **指派接单**：货主下单时已选择司机，司机在**待接单/待确认**列表中确认是否接单。

## 二、前置条件

- **账号与权限要求**：司机需使用个人手机号码注册。
- **设备与网络要求**：准备一部安卓或 iOS 系统的智能手机，并确保网络连接正常（**4G/5G/Wi-Fi**）。
- **配套工具/链接**：
  - 🛠️ **核心组件下载**
  - 👉 [点击下载中通冷运司机端 APP （安卓）](https://zmas.zto.com/download/NTMS)
  - 下载鲸小宝APP（IOS）：敬请期待

## 三、操作入口

### 3.1 司机抢单

**系统功能路径**：登录APP ->货源大厅

### 3.2 司机接单

**系统功能路径**：登录APP ->订单中心->待确认

## 四、操作步骤

### 4.1 场景一：司机抢单

1. 登录 APP，进入**货源大厅**。
2. 输入**出发地**和**目的地**，查找匹配的货源；也可以添加筛选条件进行查找。

![货源大厅.jpeg](./images/司机端注册操作说明_1_ad777e38.png)

3. 找到合适货源后，点击**立即接单**。
4. 按页面提示缴纳**保证金**，完成抢单。
5. 若有车载温控设备，请在接单后进行授权：[如何授权温控设备？](https://docs.qq.com/doc/DZW5IbnJETXhqdlpP)

### 4.2 场景二：司机接单

1. 登录 APP，进入**订单中心->待确认**。
2. 在**待确认**订单页面，确认订单信息。
3. 确认无误后，点击**接单**。
4. 按页面提示支付**保证金**，完成接单。
5. 若有车载温控设备，请在接单后进行授权：[如何授权温控设备？](https://docs.qq.com/doc/DZW5IbnJETXhqdlpP)

![image.png](./images/司机端注册操作说明_2_5d7f12d2.png)

## 五、操作结果

- **司机抢单**：缴纳**保证金**后，抢单完成。
- **司机接单**：在**待确认**订单中点击**接单**并支付**保证金**后，接单完成。
- 如有车载温控设备，接单后需按要求完成温控设备授权。

## 六、注意事项

::: warning 注意事项
- 待确认订单如果超过**15分钟**未缴纳保证金，视为抢单失败，订单会回到**货源大厅**。
- 如果当前时间已到达**装货时间**，系统默认取消订单。
- 请及时完成**保证金**支付，并在订单**装货时间**之前完成接单。
:::

## 八、常见问题

**Q1：为什么要缴纳保证金？**

**A**：为了保障平台双方的合法权益，司机在接单时需要缴纳保证金，避免爽约；在司机运输完成后会如数退还给司机。
