# 省公司三方车下单

## 一、适用场景
省公司临时需要使用三方车（非自有车辆）时，通过网络货运平台在线下单，替代线下找车调度模式，操作更便利。

## 二、前置条件
- **账号权限**：需拥有中通冷运租户权限（例如网点/省公司），若无权限请联系系统管理员。
- **环境准备**：一台可联网的电脑，推荐使用Chrome浏览器。
- **系统链接**：[https://zc.ztocc.com/dashboard](https://zc.ztocc.com/dashboard)
- **司机准备**：三方车司机需提前在中通冷运网络货运平台完成注册（详见司机端注册操作说明）。

## 三、操作入口
请根据具体场景对应的系统路径操作：
- 注册货主：登录系统 → 顶部tab切换至**【中通冷运】** → 左侧边栏**【基础资料】** → **【货主管理】** → **【企业货主】**
- 货主充值：登录系统 → 顶部tab切换至**【中通冷运】** → 左侧边栏**【财务管理】** → **【充值提现】** → **【充值】**
- 省公司下单：登录系统 → 顶部tab切换至**【冷链快运】** → 左侧边栏**【运营运输管理】** → **【运输计划】** → **【调度计划】**

## 四、操作步骤

### 4.1 场景一：注册网络货运平台货主
1. **新增货主**
   进入**【企业货主】**页面后，点击右上角**【新增】**按钮，填写货主资料并提交。
   ![image.png](./images/司机端注册操作说明_1_ad777e38.png)

2. **等待平台审核**
   系统自动提交至平台审核，平台会在 **3个工作日** 内完成。审核期间可编辑修改资料，修改后重新提交。
   ![image.png](./images/司机端注册操作说明_2_5d7f12d2.png)

3. **审核通过后维护税务信息**
   点击**【编辑】**，切换至**【开票信息管理】**标签页，点击**【新增】**填写税务资料。
   ![image.png](./images/司机端注册操作说明_3_3281b260.png)

4. **联系总部运营关联云冷分拨账号**（重要！）
   联系总部运营（@纪宗君、@曾苏展、@杨震东），将网络货运平台货主账号与省公司下云冷的分拨中心账号关联，才能实现云冷下单同步到网络货运平台。

### 4.2 场景二：省公司货主充值
1. **点击充值**
   进入**【充值】**页面，点击右上角**【充值】**按钮。
   ![image.png](./images/省公司三方车下单操作说明_1_b723c91c.png)

2. **输入充值金额**
   在弹出框中输入需要充值的额度，点击**【确认】**。
   ![image.png](./images/省公司三方车下单操作说明_2_ba076f40.png)

3. **根据引导完成充值**
   页面会引导完成后续支付操作。

### 4.3 场景三：省公司下单
1. **新增调度计划**
   - 若为临时班线：点击**【新增临时调度计划】**。
   - 若为固定班线：点击对应班线上的**【调度】**按钮。
   ![image.png](./images/省公司三方车下单操作说明_3_c3d4a014.png)

2. **选择运输公司和司机**
   运输公司必须选择 **【中通冷运（天津）科技有限公司】**；司机选择已注册的三方车司机（司机需提前在司机端APP完成注册）。
   ![image.png](./images/省公司三方车下单操作说明_4_23bd7c3b.png)

3. **填写必填信息，输入运费**
   计费方式选择**【自定义价格】**，在计费单价中输入需要支付司机的运费（**不含税**）。
   ![image.png](./images/省公司三方车下单操作说明_5_c2de3071.png)

4. **核实下单是否成功**
   查看列表中对应调度单号后面的**【订单编号】**字段，若存在网络货运订单号，则下单成功；若无，请联系技术支持处理，此单暂不要继续操作。
   ![image.png](./images/省公司三方车下单操作说明_6_290deea8.png)

## 五、操作结果
- **货主注册**：状态变为“审核通过”后，可正常下单。
- **货主充值**：账户余额增加。
- **省公司下单**：调度计划对应生成网络货运订单号，后续司机可通过小程序操作，轨迹、结算等数据自动同步。

## 六、注意事项
::: danger 重点提醒
- 运输公司必须选择 **【中通冷运（天津）科技有限公司】**，否则下单后不会返回网络货运订单号。
- 司机必须已在中通冷运网络货运平台注册并完成认证，否则无法选择。
- 货主注册后需联系总部运营关联云冷分拨账号，否则云冷下单无法同步至网络货运平台。
- 运费输入为**不含税**金额。
:::

::: warning 注意事项
- 货主注册审核需 **3个工作日**，建议提前操作。
- 一个省公司为一个货主，多个分拨中心可共用此货主账号。
- 若充值或下单遇到异常，参考常见问题或联系技术支持。
:::

## 七、常见问题
- **Q1：一个省公司有多个分拨中心，是否需要建多个货主？**
  A：不用。一个省公司对应一个货主，多个分拨中心可共用。

- **Q2：下单之后后续怎么操作？**
  A：后续操作与云冷调度计划完全一致。司机会操作司机小程序，轨迹、结算等数据通过系统自动同步。

- **Q3：提示“无访问权限”**
  A：联系技术支持或通过小吉工单申请中通冷运权限。

- **Q4：找不到司机**
  A：司机未在中通冷运司机端APP完成注册，请指导司机下载并完成注册认证。

- **Q5：下单后没有返回订单号**
  A：首先确认运输公司是否选择为 **【中通冷运（天津）科技有限公司】**，若是，则联系技术支持排查。
