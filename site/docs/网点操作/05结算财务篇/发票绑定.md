# 发票绑定

## 一、适用场景
网点日常经营产生的派件费、二级进港服务费等收入，按账期归集并映射对应开票税目，实现费用账单、发票单据、财务数据一一对应，满足网点财务对账、进项/销项发票归集，解决票据分散、账票不符、开票主体错误、税目错误等问题。

## 二、前置条件
- **账号与权限**：角色需包含 `[网点财务]`，若无权限请联系系统管理员。
- **物理/环境**：无特殊要求。
- **配套工具/链接**：
  - 官方系统登录入口：👉 [[点击进入系统](https://cwzx.ztocc.com/dashboard)]
  - 发票板块快捷链接：👉 [[点击一键直达销项账单页面](https://cwzx.ztocc.com/app/#/middlePlatform/salesInvoice)]
  - 发票管理快捷链接：👉 [[点击一键直达发票管理页面](https://cwzx.ztocc.com/app/#/middlePlatform/manage)]

## 三、操作入口
- **场景一（线下开票绑定账单）**：`登录系统` → `左侧菜单栏` → `[发票板块]` → `[销项账单]` 或 `[发票管理]`
- **场景二（发票红冲）**：`财务中心` → `发票板块` → `发票管理`

## 四、操作步骤

### 4.1 场景一：线下开票绑定账单

1. **查询应开发票信息**
   - 进入页面后，在右上角查询条件中选择 **【未生成】**，点击 **【查询】** 按钮。
   - 查看应开发票的税目和金额，核对无误后登录税局开票。
   ![image.png](./images/包仓仓位用的好快，哪里能看到仓位的使用明细呢？_1_4b707d03.png)

2. **上传发票验真**
   - 点击右上角 **【发票上传】** 按钮。
   - 在弹框中点击 **【选择文件】**，选择 **xml格式** 的电子发票。
   - 点击 **【开始上传】**，系统校验文件格式及内容。
   - 等待状态更新为 **“上传成功”** 后，点击 **【确定】** 保存。
   - 检查发票验真状态，待显示 **“校验成功”** 后继续下一步。
   ![image.png](./images/发票绑定_1_5db04e7c.png)

3. **发票绑定账单**
   - 在账单列表中勾选 **待绑定发票的账单**。
   - 点击右上角 **【线下发票绑定】** 按钮。
   - 在弹框中点击 **【选择发票】**，在发票选择弹框中勾选对应的发票后保存。
   - 点击 **【解析发票】**，系统自动将发票金额明细分摊到所选账单中。
   - 分摊成功后点击 **【保存】**，发票绑定成功。
   ![image.png](./images/发票绑定_2_8ab09d09.png)
   ![image.png](./images/发票绑定_3_3e26bdf1.png)

### 4.2 场景二：发票红冲

1. **上传红票并验真**
   - 进入 **发票管理** 页面。
   - 点击右上角 **【发票上传】**，选择 xml 格式的红字电子发票进行上传（操作与蓝票上传一致）。
   - 等待验真状态变为 **“校验成功”**。

2. **自动红冲蓝票**
   - 红票验真成功后，系统自动红冲对应的蓝票。
   - 查看红票及对应蓝票的单据状态，状态从 **“已开票”** 变为 **“已红冲”**。

## 五、操作结果
- 线下发票绑定成功后，账单与发票关联，开票状态更新。
- 发票红冲后，对应蓝票状态变为“已红冲”，账单开票状态恢复为“未生成”。

## 六、注意事项
- 上传的发票文件必须为 **xml 格式**（税局下载的 zip 压缩包需解压后选择 xml 文件）。
- 发票的购销方信息需与账单对应公司一致，否则绑定时会找不到发票。
- 跨月红冲需要上传红票，系统自动处理，不可直接解绑。

## 七、常见问题

### 7.1 异常处理表

| 序号 | 异常现象 | 常见原因 | 解决方案 |
|------|----------|----------|----------|
| 1 | 查询不到待开发票的账单 | 1. 账单未关账
2. 角色权限缺少关联公司数据权限 | 1. 联系总部财务确认关账状态
2. 联系管理员开通权限 |
| 2 | 上传发票文件失败、格式不支持 | 上传的发票文件为 zip 压缩包 | 解压后选择 xml 文件上传 |
| 3 | 绑定账单选择发票时，发票列表为空 | 1. 发票校验中或校验失败
2. 勾选账单的公司与发票购销方信息不一致 | 1. 等待校验成功，或根据失败原因处理发票
2. 确认购销方一致，若公司名称变更联系管理员 |

### 7.2 高频 FAQ

- **Q1：发票税率开错了，已经绑定到账单，可以重开发票重新绑定吗？**
  **A**：可以。若未跨月，勾选账单点击 **【线下发票解绑】** 即可；若已跨月，需上传对应的红票，系统自动红冲蓝票，账单开票状态更新为“未生成”，再上传正确发票重新绑定。

- **Q2：未及时开票，被总部扣了税点，还可以补开发票吗？**
  **A**：可以。当年账单补开发票后可正常绑定到账单，已扣税点联系总部财务申请税点返还。
