# 包仓仓位用的好快，哪里能看到仓位的使用明细呢？

## 一、适用场景

说明在什么情况下需要查看包仓仓位的使用明细。

## 二、前置条件

- 账号已开通 **财务管理-账单管理-包仓账单** 菜单权限。
- 已准备好相关的包仓业务数据。

## 三、操作入口

**财务管理** → **账单管理** → **包仓账单**

## 四、操作步骤

1. 登录鲸天系统，进入 **财务管理** → **账单管理** → **包仓账单** 页面。
2. 如果该包仓存在运单，则 **操作** 列会显示 **查看详情** 按钮。点击该按钮，弹出运单的使用明细窗口。

   ![image.png](./images/包仓仓位用的好快，哪里能看到仓位的使用明细呢？_1_4b707d03.png)

::: warning 注意事项
查看运单的占仓情况时，请重点关注 **占仓重量**。当仓位不足时，运单只会部分占仓。
:::

## 五、操作结果

在弹出窗口中查看包仓运单的使用明细，确认数据是否准确。

## 六、注意事项

如页面展示、权限范围或业务规则与本文不一致，请以当前系统配置和最新业务规则为准。
