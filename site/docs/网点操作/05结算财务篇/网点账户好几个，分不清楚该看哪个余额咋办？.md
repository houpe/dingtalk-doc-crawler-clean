# 网点账户好几个，分不清楚该看哪个余额咋办？

## 一、适用场景
本文适用于需要区分网点不同账户余额、了解各账户用途的一线网点、中心、云仓或网络货运人员。当您发现网点有多个账户，不确定该看哪一个余额时，请参考本文。

## 二、前置条件
- 您的账号已开通 **财务管理-账户管理-网点账户** 菜单权限。
- 已准备好需要查询的网点信息（如开户机构、账户类型等）。

## 三、操作入口
**财务管理 → 账户管理 → 网点账户**

## 四、操作步骤

1. 登录鲸天系统，进入 **财务管理 → 账户管理 → 网点账户**。
2. 在查询条件中填写 **开户机构** 和 **账户类型**（选择 **业务账户**），然后点击查询。
3. 根据网点层级查看对应的账户余额：
   - **一级网点**（与总部签约）：查看 **总部平台账户** 下的账户余额（即下图中的账户①）。
   - **二级网点**（与一级网点签约）：查看 **一级网点平台账户** 下的账户余额（即下图中的账户②和账户③）。
4. 如果一级网点未开设二级网点，只需关注账户①即可。
5. 如果一级网点已开设二级网点，请参考下方补充说明正确操作。

## 五、附加说明：账户设计原理与正确使用方式

> 通常每个网点会存在三个账户，理解其设计原理有助于快速判断该看哪个余额。

![image.png](./images/发票绑定_2_8ab09d09.png)

### 5.1 账户①：发货账户（一级网点与总部签约）
- 一级网点预充费用后，用于发货时支出。
- 系统仅校验账户①的余额，若余额不足会导致欠费。

### 5.2 账户②：二级网点充值/提现账户
- 二级网点与一级网点签约，其余额由一级网点线下收款后线上调整（手工充值/手工提现）。
- 账户②显示的是 **线上手工操作** 的累计金额。

### 5.3 账户③：一级网点对二级网点的收入账户
- 二级网点开单或发生其他业务时，一级网点向二级网点收取的费用计入账户③。
- 账户③的余额不影响一级网点的发货支出（发货支出只扣账户①）。

::: danger 重点提醒
**二级网点开单时，一级网点收取的费用在账户③中，而一级网点的发货支出是从账户①中扣除。** 账户①和账户②/③之间是独立的，没有任何关联。因此，一级网点的余额减少（账户①）导致欠费，是因为没有将线下收到的二级网点充值款及时转入账户①。
:::

### 5.4 正确的处理方式

当一级网点线下收取二级网点充值费用后，需要同时做两件事：
1. 通过 **手工充值** 功能，调整二级网点的 **账户②** 余额。
2. 将线下收到的费用，通过 **线上充值** 功能，充入一级网点的 **账户①** 余额中。

这样既能保证二级网点的开单权限，又能维持一级网点自身的发货余额充足。

## 六、操作结果
查询完成后，页面会显示对应平台下的账户明细余额。请根据网点层级和业务需求核对余额是否与预期一致。

## 七、注意事项
- 如页面展示、权限范围或业务规则与本文不一致，请以当前系统配置和最新业务规则为准。
- 账户①、②、③均为 **业务账户**，查询时务必选择 **账户类型 = 业务账户**。
- 手工充值与线上充值属于不同的操作入口，请勿混淆。
