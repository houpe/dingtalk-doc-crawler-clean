# 网点账户好几个，分不清楚该看哪个余额咋办？

## 一、适用场景

本文适用于网点在 **网点账户** 页面查看账户余额时，因存在多个账户，无法判断应查看哪个余额的场景。

## 二、前置条件

1. 已登录鲸天系统。
2. 账号已开通 **财务管理-账户管理-网点账户** 菜单权限。
3. 已确认需要查询的 **开户机构**、**账户类型** 等信息。

## 三、操作入口

**财务管理-账户管理-网点账户**

## 四、操作步骤

1. 登录鲸天系统，进入 **财务管理-账户管理-网点账户**。

2. 在查询条件中填写 **开户机构**，并选择 **账户类型（业务账户）**。

3. 查看 **开户平台** 为对应上级的明细余额：

   - **一级网点与总部签约**：查看 **总部平台账户** 下的账户余额。
   - **二级网点与一级网点签约**：查看 **一级网点平台账户** 下的账户余额。

## 五、操作结果

完成查询后，可在 **网点账户** 页面看到对应开户平台下的账户明细余额。

## 七、账户理解

通常每个网点会存在三个账户。判断要看哪个账户时，需要结合账户的设计原理和应用场景。

![image.png](./images/发票绑定_1_5db04e7c.png)![image.png](./images/发票绑定_2_8ab09d09.png)

- **账户①**：一级网点与总部签约后，预充费用用于发货，可查看总部平台账户下开设的账户①。
  - 若一级网点未开发二级网点，只需关注 **账户①**。
- **账户②**：二级网点与一级网点签约后，二级网点的余额由一级网点线下收款后线上调整；线上为二级网点 **手工充值/手工提现** 的总金额可查看账户②。
- **账户③**：二级网点开单或发生其他业务时，一级网点对二级网点的收入可查看 **账户③**。

## 八、常见问题

### 8.1 二级网点开单，为什么一级网点的余额一直在减少，最终导致欠费？

一级网点的 **账户①** 与 **账户②/③** 之间没有联系。

二级网点开单时：

- 一级网点收取的费用在 **账户③** 中。
- 一级网点支出的金额从 **账户①** 中扣除。
- 系统只校验 **账户①** 的余额。

简易处理方式：一级网点线下收取二级网点充值费用后，需要做两件事：

1. 通过 **手工充值** 调整二级网点的账户余额。
2. 将线下收到的费用，通过线上 **充值**，充入一级网点的账户中。
