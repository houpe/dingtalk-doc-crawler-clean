# 敷衍问题件如何有效举证？

## 一、适用场景

本文适用于需要对不规范、虚假或敷衍处理的问题件进行举证的业务场景。

## 二、前置条件

请确认账号已开通对应菜单权限，并已准备好需要举证的问题件相关业务数据。

## 三、操作入口

请以系统实际菜单路径为准。

## 四、操作步骤

1. 进入问题件相关页面，找到需要举证的问题件。

2. 根据举证类型确认是否满足举证条件。

::: danger 重点提醒
- **举证登记不规范、虚假问题件**：必须由**问题件被通知方**举证，且必须在**问题件发布之日起的 72 小时内**举证，超时不允许。
- **举证回复不规范**：必须由**问题件发布方**举证，且必须在**回复后的 72 小时内**举证，超时不允许。
:::

3. 在页面中按实际情况发起举证，并提交相关信息。

![image.png](./images/仲裁如何申报、申诉？_10_2674e296.png)

## 五、操作结果

操作完成后，请在系统页面确认举证是否提交成功，以及问题件状态或处理结果是否符合预期。

## 六、注意事项

::: warning 注意事项
如页面展示、权限范围或业务规则与本文不一致，请以当前系统配置和最新业务规则为准。
:::

::: danger 重点提醒
举证有明确的角色和时效限制，请在**72 小时内**完成对应举证，超时不允许。
:::
