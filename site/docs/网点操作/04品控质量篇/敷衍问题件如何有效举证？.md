# 敷衍问题件如何有效举证？

## 一、适用场景

本文适用于需要了解“敷衍问题件如何有效举证？”相关操作的业务场景。

## 二、前置条件

- 请确认账号已开通对应菜单权限。
- 已准备好操作所需的业务数据（如问题件编号、相关信息等）。

## 三、操作入口

请以系统实际菜单路径为准。

## 四、操作步骤

### 4.1 问题件举证

根据举证类型不同，操作方式分为以下两种场景。

#### (1) 场景一：举证登记不规范或虚假问题件

1. 确认当前操作角色为**问题件被通知方**。
2. 在**问题件发布之日起的72小时内**完成举证操作，超时则不允许举证。
3. 进入举证页面，按系统提示上传或填写相关证明材料。
4. 提交举证后，等待系统反馈结果。

![image.png](./images/仲裁如何申报、申诉？_10_2674e296.png)

#### (2) 场景二：举证回复不规范

1. 确认当前操作角色为**问题件发布方**。
2. 在**对方回复后的72小时之内**完成举证，超时则不允许举证。
3. 根据系统提示上传或补充相关证明资料。
4. 提交举证并确认状态。

::: danger 重点提醒
- 场景一：必须由 **问题件被通知方** 举证，且在 **问题件发布之日起72小时内** 完成。
- 场景二：必须由 **问题件发布方** 举证，且在 **回复后的72小时内** 完成。
- 超过上述时限，系统将不允许举证操作。
:::

## 五、操作结果

操作完成后，请在系统页面确认数据状态或举证处理结果是否符合预期。

- 举证时请确保上传的证明材料清晰、完整，避免因材料问题导致举证失败。
