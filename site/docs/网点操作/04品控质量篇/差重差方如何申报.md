# 一. 差重差方如何申报

## 1.1 菜单入口及预览

<strong>操作入口</strong>：服务质量，流程节点人员在操作查看数据。非流程节点查询查看全部数据

发布：差重差方申报

差重差方查询：无影响申诉单

差重差方申诉：有影响申诉单查询、申诉处理

![image.png](./images/仲裁如何申报、申诉？_10_2674e296.png)

## 1.2 发布/申诉差重差方

1. 点击查询申诉页发布

![image.png](./images/仲裁如何申报、申诉？_11_14378a46.png)

2. 跳转发布页，输入内容，发布

- 有影响需申诉：登记方登记后，判断是否影响，有影响的判断条件：（复核重量-录单重量）/录单重量>5%或（录单重量-复核重量）/录单重量>5%或（复核体积-录单体积）/录单体积＞5%或（录单体积-复核体积）/录单体积＞5%或（复核结算重量-录单结算重量）/录单结算重量>5%或（录单结算重量-复核结算重量）/录单结算重量>5%，有影响后，走寄件网点72小时申诉流程，72小时内申诉，单据立即到总部处理，如果72小时未申诉，则到72小时后单据到总部处理；如无影响，则仅记录，流转至差重差方查询；
- 被投诉对象可以是任务机构；

![image.png](./images/仲裁如何申报、申诉？_1_575eadf9.png)

- 差重差方申诉为有影响数据，流转至差重差方申诉，责任方可以进行申诉至总部，进行审核处理
- 差重差方查询为无影响数据的记录，无任何处理流程

![image.png](./images/仲裁如何申报、申诉？_2_1b53f941.png)
