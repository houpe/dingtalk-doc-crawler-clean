# 问题件预处罚有歧义，想要申诉？

## 一、适用场景

当您对超时单的预处罚结果存在异议时，需要在规定时间内发起申诉。

## 二、前置条件

- 账号拥有**问题件申诉**相关权限（如不确定请联系管理员）。
- 有歧义的超时单仍处于**72小时内**（当前时间 - 创建时间 ≤ 72小时）。

## 三、操作入口

请以系统实际菜单路径为准，通常在 **处罚中心** 或 **问题件管理** 模块下。

## 四、操作步骤

### 4.1 举证申诉（主动提供证据）

适用于需要针对超时单补充举证材料的情况。

1.1 进入申诉入口，找到目标有歧义的超时单。

1.2 点击 **申诉** 按钮，进入举证界面。

1.3 上传相关证据（截图、文件等），填写申诉理由。

1.4 确认提交。

::: tip 提示
举证申诉需在72小时内完成提交，超时系统将自动完结并执行处罚。
:::

![image.png](./images/仲裁如何申报、申诉？_10_2674e296.png)

### 4.2 超时未回复申诉（针对超时未处理的情况）

适用于因系统超时未自动回复、但实际已处理或存在其他原因的申诉。

2.1 在申诉列表中找到标记为“超时未回复”的单据。

2.2 点击 **申诉** 按钮，填写说明（如“实际已回复但系统未识别”或其他解释）。

2.3 确认提交。

::: danger 重点提醒
- 每张有歧义的超时单仅能发起一次申诉。
- 若超过72小时未申诉，系统将自动完结该申诉单并执行处罚。
:::

![image.png](./images/仲裁如何申报、申诉？_11_14378a46.png)

## 五、操作结果

- 申诉提交后，系统会记录并等待审核。
- 在规定时间内（72小时内）完成申诉的，处罚将被暂停；若审核通过则撤销处罚。
- 超过72小时未申诉的，申诉单自动完结并执行处罚。

## 六、注意事项

- ⚠️ 申诉时效严格为**72小时**，从超时单创建时间开始计算，超时不可补交。
- 📌 请务必提供真实有效的证据或说明，虚假申诉可能被驳回并导致额外处罚。
- ❗ 两种申诉入口（举证申诉与超时未回复申诉）适用场景不同，请按实际情况选择。

## 七、常见问题

- **问：申诉提交后可以修改吗？**
  答：一般情况下提交后不可修改，建议提交前仔细核对。

- **问：如果超过72小时发现误处罚怎么办？**
  答：自动处罚后不再支持申诉，请联系网点管理员或客服走线下处理。
