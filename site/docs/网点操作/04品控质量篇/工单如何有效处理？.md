# 工单如何有效处理？

## 一、适用场景
本文适用于一线网点、中心、云仓或网络货运人员处理各类工单（普通工单、申诉工单、咨询工单）的场景。总部人员可参考录单、发短信等高级操作。

## 二、前置条件
- 已登录系统，拥有相应权限：**品控工作台（操作）** 仅当前节点责任人可处理（一级可帮二级处理）；**品控工作台（查询）** 所有人员可查询。
- 工单数据已生成（如运单轨迹、历史工单记录等）。
- 申诉工单须在 **72小时** 内发起。

## 三、操作入口
系统菜单路径（请以实际系统为准）：**品控工作台** → **操作** / **查询**。
具体入口包括：
- **普通工单查询**：处理（暂存/完结）。
- **工单申诉**：72小时未申诉单申诉。
- **咨询工单**：所有咨询工单查询（无需网点处理的工单）。
- **品控工作台（查询）**：所有人员可查询。
- **录单/发短信**（仅总部权限）：录入工单、发送短信、催单/转单。

## 四、操作步骤

### 4.1 网点处理普通工单
1. 进入 **品控工作台（操作）**，查看“待处理”数量（普通工单待处理 + 申诉单待处理总和）。
2. 点击工单行后的 **【处理】** 按钮，进入处理详情页。
3. 在详情页可执行以下操作：
   - **暂存**：保存当前处理进度，后续可继续处理。
   - **完结**：工单处理完成，提交结果。
4. 处理完成后，工单状态更新为“完结”或“暂存”。

::: warning 注意事项
工单升级至上级后，当前责任网点无法处理，仅上级可处理。
:::

### 4.2 网点申诉工单
1. 在 **普通工单申诉单** 模块中，仅限 **72小时内** 的工单可进行申诉。
2. 点击操作列的 **【处理】** 按钮，进入详情页。
3. 填写 **申诉内容** 并上传附件，点击 **确认** 提交申诉。
4. 提交后，申诉单将流转至总部进行审核裁定。

![image.png](./images/仲裁如何申报、申诉？_3_c164c666.png)

### 4.3 总部录单/发短信（仅总部权限）
1. 在 **录单页面** 录入工单，选择工单类型：
   - **普通工单**：可流转至处理网点进行后续处理。
   - **咨询工单**：仅记录，不做网点流转处理。
2. **处理网点**：系统自动带出运单轨迹机构，默认为工单类型配置的对应处理网点（寄件网点 / 目的网点），也可手动选择其他网点推送处理。
3. **工单处理记录**：输入运单号后自动显示该运单的历史工单记录及物流轨迹，便于快速了解问题。
4. 点击右下角 **【短信图标】**，可自定义发送短信至运单对应的目的网点或寄件网点。

![image.png](./images/仲裁如何申报、申诉？_4_775d1472.png)

![image.png](./images/仲裁如何申报、申诉？_5_cf89d5f4.png)

5. 对于 **处理中** 的工单，可进行以下操作：
   - **追加说明**：在详情页追加内容，处理网点可在处理记录中查看。
   - **催单**：点击催单按钮，处理网点可在工单列表或处理记录中看到催单次数，以便及时处理。

![image.png](./images/仲裁如何申报、申诉？_6_021a0df3.png)

## 五、操作结果
- 普通工单处理完成后，状态变为 **完结** 或 **暂存**（可继续处理）。
- 申诉单提交后，流转至总部审核，等待裁定结果。
- 处理中工单可被催单和追加说明，处理网点可实时查看。
- 待办消息通知会提醒处理网点以下场景：
  - 新普通工单（新建/升级/转单）
  - 首响超时新申诉单
  - 完结超时新申诉单
  - 二次工单新申诉单
  - 超时升级新申诉单

## 六、注意事项
- **权限限制**：**品控工作台（操作）** 仅当前节点责任人可处理（一级可帮二级处理）；**品控工作台（查询）** 所有人员可查询。
- **工单升级**：工单升级至上级后，当前责任网点无法处理，仅上级可处理。
- **申诉时效**：普通工单申诉需在 **72小时** 内发起，超时将无法申诉。
- **咨询工单**：仅记录，无需网点处理。
- **总部录单**：仅总部权限可进行录单、发短信、催单/转单操作。
- **催单提醒**：处理网点需关注催单次数，及时响应以提高工单处理效率。

## 七、常见问题
**Q：工单升级后，原责任网点还能处理吗？**
A：不能。工单升级至上级后，仅上级可处理，原网点无法操作。

**Q：申诉工单需要在多长时间内提交？**
A：普通工单申诉需在 **72小时** 内提交，超时后无法申诉。

**Q：咨询工单需要网点处理吗？**
A：不需要。咨询工单仅用于记录，不流转至网点处理。
