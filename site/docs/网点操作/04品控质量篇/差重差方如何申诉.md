# 差重差方如何申诉

## 一、适用场景

本文适用于需要了解“差重差方如何申诉”相关操作的业务场景。

## 二、前置条件

请确认账号已开通对应菜单权限，并已准备好操作所需的业务数据。

## 三、操作入口

请以系统实际菜单路径为准。

## 四、操作步骤

1. 责任网点在收到差重差方通知后 **72小时内** 可以进行申诉。
   ![image.png](./images/仲裁如何申报、申诉？_10_2674e296.png)

2. 点击 **【处理】** 按钮，跳转至详情页。
   输入 **申诉内容** 并上传 **附件**，提交后流转至总部审核。
   ![image.png](./images/仲裁如何申报、申诉？_11_14378a46.png)

::: danger 重点提醒
累计责任：同个责任网点当月被成功投诉次数大于 **5次** 后（不含第5次），每票累计处罚 **1000元**。
:::

## 五、操作结果

操作完成后，请在系统页面确认数据状态或处理结果是否符合预期。

## 六、注意事项

如页面展示、权限范围或业务规则与本文不一致，请以当前系统配置和最新业务规则为准。
