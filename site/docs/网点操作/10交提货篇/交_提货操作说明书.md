# 交提货

## 一、适用场景

本功能依托 **中通冷链司机版** 小程序，用于司机线上完成 **网点交货** 和 **网点提货** 全流程操作。通过该功能可规范冷链货物交接标准、实时查看任务进度，提升网点货物交接效率。

- **网点交货**：司机将货品运送至指定分拨/集配，线上提交交货申请、完成交接确认。
- **网点提货**：司机前往指定分拨/集配提取货品，线上提交提货申请、完成交接确认。
- **任务管理**：汇总展示交货、提货、自提等全部交接任务，可查看任务编号、网点、状态、时间等信息。

## 二、前置条件

1. **账号与权限**：使用专属司机账号登录 **中通冷链司机版** 小程序。账号由网点统一分配，无权限请联系网点管理员。
2. **设备与权限要求**：使用智能手机，**必须开启小程序定位权限**，否则功能入口不显示。
3. **配套工具**：微信搜索并打开【中通冷链司机版】小程序。

## 三、操作入口

- **网点交货**：小程序首页 → 常用服务 → **交提货** → **网点交货**
- **网点提货**：小程序首页 → 常用服务 → **交提货** → **网点提货**

## 四、操作步骤

### 4.1 场景一：网点交货

1. 进入小程序首页，点击 **【交提货】** 入口。
2. 在交提货页面，选择 **【网点交货】** 选项。
3. 阅读完整交货须知，确认内容后点击 **【我同意】**。
4. 在任务列表中选中对应交货任务，点击 **【提交】**。
5. 提交成功后自动跳转至任务管理页面，查看任务进度与状态。

![image.png](./images/交_提货操作说明书_1_e985df20.png)

### 4.2 场景二：网点提货

1. 小程序首页点击 **【交提货】** 入口。
2. 选择 **【网点提货】** 选项进入对应页面。
3. 阅读提货须知，确认无误后点击 **【我同意】**。
4. 勾选需要执行的提货任务，点击 **【提货】** 提交申请。
5. 提交完成后跳转至任务管理页面，查看提货任务状态。

![image.png](./images/交_提货操作说明书_2_66c17ef0.png)

## 五、操作结果

- 提交任务后自动跳转至 **任务管理** 页面，可切换 **交货、提货、自提** 分类，查看 **待处理、已完成、关闭** 等各类任务状态。
- 任务管理页面展示任务编号、网点、状态、时间等详细信息。

## 六、注意事项

::: danger 重点提醒
- **必须开启小程序定位权限**，否则交提货入口不可见。
- 冷藏货品表面温度要求 **0-10℃**，冷冻货品表面温度要求 **低于-15℃**。温度不达标、包装破损、无法核验内件等问题，承运方不承担相关责任。
:::

::: warning 注意事项
- 提交任务前请确认已选中正确的任务，避免误操作。
- 若任务列表为空，请联系网点工作人员创建调度单，刷新页面后即可显示。
- 网络不稳定时，请切换至稳定网络或彻底关闭小程序后台重试。
:::

## 七、常见问题

### 7.1 Q1：交提货任务提交后，在哪里查看进度？

A：任务提交后自动跳转至 **任务管理** 页面，可切换分类查看各状态任务。

### 7.2 Q2：冷链货物收货温度标准是什么？

A：冷藏货品表面温度要求 **0-10℃**，冷冻货品表面温度要求 **低于-15℃**。温度不达标、包装破损、无法核验内件等情况引发的货物问题，承运方不承担相关责任。
