# 一. 交提货

## 1.1 业务场景与名词解释

### 1.1.1 业务场景

本功能依托中通冷链司机版小程序，用于司机线上完成<strong>网点交货、网点提货</strong>全流程操作，实现货物交接线上化管控。该功能规范冷链货物交接标准，支持实时查看任务进度，提升网点货物交接整体效率。

### 1.1.2 核心名词解释

- <strong>网点交货</strong>：司机将货品运送至指定分拨/集配，线上提交交货申请、完成交接确认的流程。
- <strong>网点提货</strong>：司机前往指定分拨/集配提取货品，线上提交提货申请、完成交接确认的流程。
- <strong>任务管理</strong>：汇总展示交货、提货、自提等全部交接任务，可查看任务编号、网点、状态、时间等信息的管理模块。

## 1.2 前置准备与环境配置

1. <strong>账号与权限</strong>：使用专属司机账号登录<strong>中通冷链司机版</strong>小程序，账号由网点统一分配，无权限请联系网点管理员。
2. <strong>设备与权限要求</strong>：使用智能手机，<strong>必须开启小程序定位权限</strong>，否则功能入口无法展示。
3. <strong>配套工具/入口</strong>

- 官方入口：微信搜索并打开【中通冷链司机版】小程序

## 1.3 场景化标准操作步骤

### 1.3.1 场景一：网点交货

- 系统功能路径：中通冷链司机版小程序首页 → 常用服务 → 交提货 → 网点交货

1. 进入小程序首页，找到并点击【交提货】入口
2. 在交提货页面，选择【网点交货】选项
3. 阅读完整交货须知，确认内容后点击【我同意】
4. 在任务列表选中对应交货任务，点击【提交】
5. 提交成功后自动跳转至任务管理列表，查看任务进度与状态

### 1.3.2 场景二：网点提货

- 系统功能路径：中通冷链司机版小程序首页 → 常用服务 → 交提货 → 网点提货

1. 小程序首页点击【交提货】入口
2. 选择【网点提货】选项进入对应页面
3. 阅读提货须知，确认无误后点击【我同意】
4. 勾选需要执行的提货任务，点击【提货】提交申请
5. 提交完成后跳转至任务管理页面，查看提货任务状态

## 1.5 高频常见问题（FAQ）

### 1.5.1 Q1：交提货任务提交后，在哪里查看进度？

A：任务提交后会自动跳转至<strong>任务管理</strong>页面，可切换交货、提货、自提分类，查看待处理、已完成、关闭等各类任务状态。

### 1.5.2 Q：冷链货物收货温度标准是什么？

A：冷藏货品表面温度要求 <strong>0-10℃</strong>，冷冻货品表面温度要求 <strong>低于-15℃</strong>。货品温度不达标、包装破损、无法核验内件等情况引发的货物问题，承运方不承担相关责任。
