# 送货入仓

## 一、适用场景

本手册适用于司机使用 **中通冷链司机版** 小程序完成货物 **送货入仓** 线上提报的全流程操作。通过本功能，司机可在线阅读入仓须知、选择对应运单并提交，实现入仓交接线上化流转，配合网点交提货体系统一管理任务，实时查看任务状态。

## 二、前置条件

1. **账号与权限**
   - 使用微信登录 **中通冷链司机版** 小程序。
   - 新司机需完成手机号授权、车辆信息注册绑定；已有账号可直接登录。
   - 若无使用权限，请联系网点管理员。

2. **设备与权限**
   - 使用智能手机，开启微信及小程序的 **定位权限**。
   - 允许小程序获取手机号信息。
   - 新注册需使用相机拍摄并上传车辆照片。

3. **配套工具/入口**
   - 官方入口：微信搜索并进入 **中通冷链司机版** 小程序。

## 三、操作入口

- **新用户注册**：微信 → 中通冷链司机版小程序 → 登录授权 → 车辆注册
- **日常送货入仓**：中通冷链司机版小程序首页 → 常用服务 → 交提货 → 送货入仓

## 四、操作步骤

### 4.1 场景一：新用户账号与车辆注册（首次使用必做）

1. 打开小程序，点击 **手机号快捷登录**，勾选同意《个人信息授权申明》。
2. 在弹出的手机号授权窗口中点击 **允许**，完成微信手机号授权登录。
3. 进入小程序首页，点击常用服务内的 **交提货**。
4. 若未绑定车辆，系统弹出“请注册车辆”提示，点击进入注册页面。
5. 依次填写车辆信息、选择是否为新能源车、是否为冷藏车，上传车辆照片，最后点击 **提交**，完成车辆注册绑定。

![image.png](./images/交_提货操作说明书_1_e985df20.png)

### 4.2 场景二：日常送货入仓操作

1. 进入小程序首页，在常用服务区域点击 **交提货** 入口。
   ::: tip 界面说明
   首页顶部展示欢迎语、车辆编号，中部罗列车线任务、车辆交接、交提货、打卡等功能。
   :::
2. 在交提货选项栏中，选择 **送货入仓**。
   ::: tip 界面说明
   页面包含网点交货、网点提货、客户自提、送货入仓四个选项。
   :::
3. 阅读完整《送货入仓须知》，确认条款内容后点击 **同意**。
   ::: tip 界面说明
   须知明确冷链收货温度标准、货物损耗责任划分、包装及运输相关约定，底部为【同意】按钮。
   :::
4. 进入任务列表，选中需要执行的 **送货入仓** 订单。
   ::: tip 界面说明
   列表展示任务编号、货主名称、票数/件数、操作时间、车辆信息。
   :::
5. 核对运单信息无误后，点击 **提交** 按钮。
6. 提交成功后自动跳转至 **任务管理** 列表，可查看该任务状态与进度。
   ::: tip 界面说明
   任务管理页可切换全部、交货、提货、自提、送货入仓分类，任务状态分为待处理、已完成、关闭等。
   :::

![image.png](./images/交_提货操作说明书_2_66c17ef0.png)

## 五、操作结果

- 提交成功后，系统自动跳转至 **任务管理** 列表。
- 在“送货入仓”分类下可查看该任务的状态为“待处理”或“已完成”。
- 任务状态为“关闭”表示该笔送货入仓任务已取消或终止。

## 六、注意事项

::: tip 权限与设备要求
- 必须开启微信及小程序的 **定位权限**，否则无法使用送货入仓功能。
- 新注册用户必须完成车辆注册绑定，否则无法进行任何交提货操作。
:::

::: warning 货物温度标准与责任
- 冷藏托运物表面温度要求：**0-10℃**
- 冷冻托运物表面温度要求：**低于 -15℃**
- 若货物温度不达标、包装破损、无法核验内件或常温转运，由此产生的失温、变质、破损等问题，承运方不承担赔偿责任。
- 具体细则以《送货入仓须知》及中通冷链《电子运单契约条款》为准。
:::

::: danger 关键规则
- 车辆注册提交时，需确保小程序拥有 **相机权限**，照片大小适中。
- 登录时若无法获取手机号，需在微信设置中开启 **手机号权限**。
- 任务列表为空时，请先联系货主或网点确认是否已创建入仓运单。
:::

## 七、常见问题

### 7.1 Q1：首次使用小程序，为什么一定要注册绑定车辆？

A：车辆信息绑定是使用交提货、送货入仓等功能的必要前提。系统需关联司机与运输车辆信息，完成任务溯源与管理。未绑定车辆无法开展后续操作。

### 7.2 Q2：送货入仓任务提交后，如何查看任务状态？

A：提交成功后会自动跳转至任务管理页面，点击顶部 **送货入仓** 分类标签，即可筛选查看对应任务，区分待处理、已完成、关闭等状态。

### 7.3 Q3：任务显示“关闭”代表什么？

A：任务状态为“关闭”，说明该笔送货入仓任务已取消或终止，无需再进行操作。

### 7.4 Q4：登录时无法获取手机号怎么办？

A：1. 退出小程序，进入小程序设置，开启手机号权限；2. 切换稳定网络后重新登录授权。

### 7.5 Q5：点击交提货后没有送货入仓选项？

A：1. 点击小程序右上角【···】，进入设置；2. 开启位置信息（使用小程序时允许）；3. 重新进入小程序。

### 7.6 Q6：进入送货入仓页面，任务列表为空？

A：联系对应货主或网点工作人员，完成运单创建，刷新页面即可加载任务。

### 7.7 Q7：车辆注册提交失败，无法上传照片？

A：1. 开启小程序相机权限；2. 重新拍摄清晰车辆照片，控制图片大小后再次提交。

### 7.8 Q8：提交送货入仓任务失败？

A：1. 切换WiFi/移动网络重试；2. 关闭小程序后台，重启小程序再操作。
