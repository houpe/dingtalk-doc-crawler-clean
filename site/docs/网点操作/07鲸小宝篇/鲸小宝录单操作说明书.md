# 鲸小宝录单

## 一、业务场景与名词解释

### 1.1 业务场景

网点日常承接客户寄件需求时，通过<strong>鲸小宝APP</strong>完成手工录单操作，完整录入寄收件信息、货物信息、服务类型等内容，生成正式运单。本次同步完成全线产品体系升级，在原有产品基础上新增多款冷链产品，统一录单流程、产品选择规则、改单限制及面单展示样式，规范全网点录单标准，保障运单信息准确、流转顺畅。

### 1.2 核心名词解释

- <strong>产品分类</strong>：录单一级选择类目，分为<strong>冷链快运（原零担/小件）</strong>、<strong>冷链快递</strong>，用于快速区分业务类型。
- <strong>冷链快运</strong>：主流冷链寄件类型，包含多款细分产品，适配不同时效、重量、价位的货物运输需求。
- <strong>冷链快递</strong>：独立产品类型，按专属规则下单、流转与结算。
- <strong>细分产品</strong>：冷链快运下包含多款差异化产品（含鲸确达、鲸速达、鲸惠达等），各产品时效、载重、定价、服务标准不同。
- <strong>面单</strong>：货物运输凭证，分为子单、签收联、留存联，是货物交接、派送、签收的依据。

## 五、高频常见问题（FAQ）

### 5.1 Q1：录单时为什么要先选择产品分类，再选细分产品？

○ A：产品分类为一级大类，区分冷链快运、冷链快递两大业务板块；细分产品根据时效、载重、价格划分，系统会自动匹配可用品类，简化操作同时符合业务规则。

### 5.2 Q2：手机APP录单规则和PC端、产品选择逻辑一样吗？

○ A：完全一致，两端流程、筛选规则、改单限制、面单标准全部统一。

### 5.3 Q3：订单什么情况下不能修改产品类型？

○ A：订单流转至中心到件、卸车环节后，禁止修改产品类型，仅可调整收货网点。

### 5.4 Q5：面单上的简写标识代表什么作用？

○ A：用于网点、派送人员快速识别订单所属产品类型，提升货物分拣、交接效率。
