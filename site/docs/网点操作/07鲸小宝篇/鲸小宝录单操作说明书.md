# 鲸小宝录单

## 一、适用场景

网点日常承接客户寄件需求时，通过**鲸小宝APP**完成手工录单操作，包括录入寄收件信息、货物信息、服务类型等，生成正式运单。本次产品体系升级后，新增多款冷链产品，统一录单流程、产品选择规则、改单限制及面单样式。

## 二、前置条件

- **账号与权限**：操作人员需持有鲸小宝**网点录单菜单**权限。如无权限，请联系系统管理员开通。
- **物理/环境准备**：
  - 手机保持正常网络连接，APP已更新至最新版本。
  - 蓝牙打印机调试正常，支持新版电子面单打印。
- **配套工具**：
  - 👉 [点击下载鲸小宝APP（安卓）](https://zmas.zto.com/download/com.app.ztocc.andriod)
  - iOS用户请在App Store搜索“鲸小宝”下载。

## 三、操作入口

**鲸小宝APP → 网点录单**

## 四、操作步骤

1. 打开鲸小宝APP，进入**网点录单**菜单。
2. 选择**产品分类**（冷链快运或冷链快递）。
   - **冷链快运**：主流冷链寄件类型，包含多款细分产品（如鲸确达、鲸速达、鲸惠达等），适配不同时效、重量、价位。
   - **冷链快递**：独立产品类型，按专属规则下单、流转与结算。
3. 根据页面展示的**可用细分产品**选择具体产品。
   ::: tip 提示
   部分细分产品无法选择时，表示货物重量、运输时效或线路成本不满足该产品规则，系统会自动过滤。请按页面可用产品下单，或核对货物/地址信息后重新选择。
   :::
4. 依次填写**寄件人信息**、**收件人信息**、**货物信息**（品名、重量、数量等）、**服务类型**。
5. 确认信息无误后，点击**提交**生成运单。
   - 生成后可在订单列表中查看和管理运单，支持按全品类产品筛选查询。

## 五、操作结果

- 生成正式运单，订单状态为“待揽收”或对应初始状态。
- 可在**订单列表**中查看运单详情，打印面单。

## 六、注意事项

- **产品选择规则**：录单时必须先选产品分类，再选细分产品；系统会自动匹配可用品类，不可选择不合规产品。
- **改单限制**：订单流转至中心到件、卸车环节后，禁止修改产品类型，仅可调整收货网点。
- **面单简写标识**：面单上的简写用于网点、派送人员快速识别订单所属产品类型，提升分拣、交接效率。
- **菜单查询问题**：若菜单内找不到对应产品筛选项，请刷新页面或重启APP后重试。

## 七、常见问题

### 7.1 Q1：录单时为什么要先选择产品分类，再选细分产品？

A：产品分类为一级大类，区分冷链快运、冷链快递两大业务板块；细分产品根据时效、载重、价格划分。系统会自动匹配可用品类，简化操作同时符合业务规则。

### 7.2 Q2：手机APP录单规则和PC端一样吗？

A：完全一致，两端流程、筛选规则、改单限制、面单标准全部统一。

### 7.3 Q3：订单什么情况下不能修改产品类型？

A：订单流转至中心到件、卸车环节后，禁止修改产品类型，仅可调整收货网点。

### 7.4 Q4：面单上的简写标识代表什么作用？

A：用于网点、派送人员快速识别订单所属产品类型，提升货物分拣、交接效率。
