# 鲸准达相关配置说明

## 一、适用场景

本文用于指导省区或总部管理员配置鲸准达相关功能，包括：
- 新营业网点开启鲸准达权限；
- 省区勾画鲸准达围栏；
- 编辑已提交的围栏图层。

鲸准达仅适用于满足以下条件的零担、小件产品（不含自提、派送专车/送货进仓-共配/送货进仓-专车）：
1. 派件网点已开启鲸准达权限；
2. 派件地址在鲸准达围栏内；
3. 预计签收时间 - 开单截单时间 ≤ 96小时；
4. 开单时间在 **[0点：每日网点截单时间]** 内（注：条件4本次未生效，预计最迟下周上线）。

同时满足以上 3 个条件（条件4暂时不生效）的单才会被标记为鲸准达。

## 二、前置条件

- 具备省区或总部管理员权限，能够操作围栏勾画与网点配置；
- 熟悉小件区域维护的操作逻辑；
- 已有鲸准达围栏基础数据或图层。

## 三、操作入口

登录后台管理系统，进入相关功能模块。具体菜单路径请以系统实际为准（可参考“新营业网点开启-鲸准达时效配置”或“省区勾画鲸准达围栏”入口）。

## 四、操作步骤

### 4.1 新营业网点开启鲸准达权限

1. 进入 **鲸准达时效配置** 页面。
2. 找到需开启的营业网点，在对应开关或列表中启用鲸准达权限。
3. 保存配置。

![image.png](./images/业务员码设置_10_2335b472%202.png)

### 4.2 省区勾画鲸准达围栏

1. 进入鲸准达围栏管理页面（与小件区域维护入口类似）。
2. 按省区勾画围栏区域，勾画方式与小件区域一致。
3. 系统会自动与已有网点的区域进行裁剪（仅保留围栏内且有网点的部分）。

![image.png](./images/业务员码设置_10_2335b472.png)

### 4.3 编辑已提交的围栏图层

1. 在编辑页面打开需要修改的鲸准达图层。
2. 编辑时，显示的图层为上次提交审核前的版本（简称 A）。
3. 修改完成后提交审核。审核通过后，图层会更新为最终结果。

::: warning 注意事项
在编辑页面中，鲸准达图层和上次提交审核前的保持一致（按最新展示你的勾画且提交结果，简称A）。之前的逻辑是审核后会显示为最终的结果。

**目的**：若勾画的区域存在新增、删减网点（零担、小件），鲸准达围栏可自动重新裁剪，但仅限于 A 范围内。
:::

![image.png](./images/业务员码设置_11_9ccc3204%202.png)

## 五、操作结果

- 新网点开启后，该网点派件时符合条件的订单会被标记为 **鲸准达**。
- 围栏配置生效后，派件地址在围栏内的订单才能进入鲸准达逻辑。
- 编辑提交的图层审核通过后，自动裁剪生效。

## 六、注意事项

- 鲸准达仅针对零担、小件产品，**不含**自提、派送专车/送货进仓-共配/送货进仓-专车。
- 如果已开单并出现以下情况，会**取消**鲸准达标记：
  - 首分拨中心发件后，修改目的网点或修改地址导致目的网点变化；
  - 修改路由方案，从默认路由变成中心直发（不匹配鲸准达时效）。
- 条件4（开单时间在0点至每日网点截单时间）本次未生效，预计最迟下周上线，届时需同时满足所有条件。
