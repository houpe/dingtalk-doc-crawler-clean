# 业务员码设置

## 一、适用场景

当网点派件较多，需要通过**业务员编码**区分运单所属派送区域或线下约定司机时，可参考本文查看和设置业务员码。

- 派件太多，司机派送如何区分是我派送的运单？**本期内容**
- 货太多，司机怎么合理规划先后排线？**即将上线**[暗中观察]
- 揽件太多，如何自动指派司机揽件？**即将上线**[暗中观察]

## 二、前置条件

1. 已登录系统账号。
2. 账号具备查看或维护**网点业务员围栏**的权限。
3. 已有网点围栏数据，业务员围栏需在对应网点围栏范围内绘制。
4. 建议绘制前先规划网点区域划分方式，例如大概要分为几部分、司机是否固定、人数是否固定等。

## 三、操作入口

系统路径：**经营管理中心 > 地址管理 > 网点业务员围栏**

![image.png](./images/业务员码设置_10_2335b472%202.png)

## 四、操作步骤

### 4.1 查看业务员编码

业务员编码可在以下位置查看。

1. 查看**三段码最后 1 位**。

![image.png](./images/业务员码设置_10_2335b472.png)

![image.png](./images/业务员码设置_11_9ccc3204%202.png)

2. 在**快件跟踪**或**运单详情**中查看。

### 4.2 业务员编码使用方式

::: warning 注意事项
当前系统中，**业务员编码未和司机任务强关联**，需要线下判断使用。

例如：线下口头约定**李四对应 02**，则按约定识别对应司机。

后续会结合司机任务进行关联处理。
:::

### 4.3 进入网点业务员围栏

1. 点击**网点业务员围栏**。
2. 登录后可查看本网点及下属网点的围栏信息。

![image.png](./images/业务员码设置_11_9ccc3204.png)

![image.png](./images/业务员码设置_12_650b65ce%202.png)

交互说明：

![image.png](./images/业务员码设置_12_650b65ce.png)

### 4.4 新增业务员围栏

1. 点击**勾画业务围栏**图标，进入围栏勾画页面。

![image.png](./images/业务员码设置_13_19522dd5%202.png)

2. 点击页面上的**添加**按钮，进入派件业务员围栏初始化界面。

![image.png](./images/业务员码设置_13_19522dd5.png)

3. 选择初始化围栏方式。

::: tip 初始化围栏方式
- **自由绘制**：在网点范围内，自由绘制业务员围栏。
- **复用网点围栏图层**：复用网点围栏图层，适用于之前网点在勾画围栏时，已经将区域用图层区分的场景，例如某些三方合作商。

网点可按需选择初始化方式，也可以结合使用。前期建议使用**自由绘制**。
:::

::: warning 注意事项
绘制前，建议先思考自己的网点大概要分为几部分。

如果司机不固定、人数不固定，需要先把变动区域的最小部分勾画出来。

例如：司机每日可约定配送不同编码的区域。
:::

### 4.5 自由绘制业务员围栏

1. 选择**自由绘制**方式。
2. 按页面提示绘制业务员围栏并填写业务员编码。
3. 确认无误后，点击**应用**。

### 4.6 复用网点围栏图层

1. 选择**复用网点围栏图层**方式。
2. 按页面提示选择需要复用的网点围栏图层。
3. 确认后点击**应用**。

![image.png](./images/业务员码设置_16_f4385eea.png)

![image.png](./images/业务员码设置_17_15961f9e%202.png)

### 4.7 修改已有图层

1. 在业务员围栏列表中，点击对应围栏的**编辑**图标。

![image.png](./images/业务员码设置_17_15961f9e.png)

![image.png](./images/业务员码设置_18_fd6186c0%202.png)

2. 点击**开始绘制**，补充此业务员的围栏。

::: warning 注意事项
编辑已有图层时，**只支持自由绘制**。
:::

3. 修改完成后，点击**应用**生效。

### 4.8 修改编码

1. 进入对应业务员围栏的编辑页面。
2. 修改业务员编码。
3. 点击**应用**生效。

![image.png](./images/业务员码设置_18_fd6186c0.png)

### 4.9 删除业务员围栏

1. 进入对应业务员围栏的编辑页面。
2. 执行删除操作。
3. 点击**应用**生效。

![image.png](./images/业务员码设置_19_4a4ac041%202.png)

### 4.10 画错或不想修改时重置

如果还没有点击**应用**，可点击**重置**，系统会回到上次应用的数据。

![image.png](./images/业务员码设置_19_4a4ac041.png)

## 五、操作结果

1. 业务员围栏设置完成并点击**应用**后，配置立即生效。
2. 派件地址落在对应业务员围栏内的订单，会匹配对应业务员编码，例如业务员编码为**01**。
3. 后续可在**三段码最后 1 位**、**快件跟踪**或**运单详情**中查看业务员编码。

## 六、注意事项

::: danger 重点提醒
- 请在对应的**网点围栏**内绘制业务员围栏。
- 建议绘制区域稍微大于网点围栏，系统会自动裁剪掉超过区域的部分。无需担心画超出，重点是保证网点范围内都覆盖。
- 如果绘制的围栏重叠，系统将删除先绘制的重叠部分，**后画的优先级更高**。
- 编辑后不会立即生效，必须点击**应用**才会生效，请在合理时间设置。
- 业务员编码**不允许重复**。
:::

::: warning 注意事项
- 一级网点和二级网点之间的业务员围栏相互独立，没有关系，也没有关联。网点的业务围栏只和自己的网点围栏有关。
- 当前不勾画业务员围栏不影响使用，非强制配置。
- 后续将关联司机任务，并增加揽件业务员围栏。
- 若网点围栏有新增，请及时修改业务员围栏。
- 若网点围栏有删除，可不用修改，因为运单匹配不到。
:::

## 七、常见问题

### 7.1 哪些情况不会打印业务员编码？

- **指定网点录单**的运单。
- **业务员围栏没有覆盖对应派件地址**的运单。

### 7.2 业务员编码会不会存在错误？

存在错误的可能。

由于地址解析错误或地址不准确，可能导致解析错误，概率不大。建议在业务员编码基础上，核对地址进行二次确认。

对于多次错误的地址，可上报工单进行校正处理。

### 7.3 不勾画业务员围栏，有什么影响？

当前不影响，非强制配置。

后续将关联司机任务，并增加揽件业务员围栏。

### 7.4 网点围栏有变化，需要及时修改业务员围栏吗？

- 若网点围栏有新增，请及时修改业务员围栏。
- 若网点围栏有删除，可不用修改，因为运单匹配不到。
