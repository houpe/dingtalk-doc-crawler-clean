# 中心-网点直发

## 适用场景

本文用于指导省区运营、总部运营、总部网络、网点录单/改单操作人员，在 **鲸天系统 PC 端**完成中心直发相关操作。

适用业务包括：

- 承运商资料录入建档
- 中心直发线路创建与审批
- 中心直发专属报价配置
- 网点录单 / 订单转运单选择路由方案
- 寄件运单改单
- 直发运单面单打印

### 业务说明

为区分干线运输模式，系统提供 **默认路由**、**中心直发** 两种路由方案。

- **默认路由**：网络自有固定干线，发车周期固定、时效稳定，无需额外申请，所有运单默认可选。
- **中心直发**：省区向总部申请的三方分拨直达运力，两点分拨无中转；发车周期不固定，开通后才可在录单界面选择，**不支持包仓政策**。
- **路由方案**：网点录单、订单转运单页面可切换干线运输模式，分为 **默认路由**、**中心直发** 两类。
- **三段码**：面单第一段编码，直发运单会增加 **“直”** 标识，用于区分直达线路。

## 前置条件

### 账号与权限

操作人员需拥有以下权限：

- **承运商新增**
- **班线管理**
- **特定业务报价**
- **网点录单**
- **运单改单**
- **面单打印**

如无权限，请联系系统管理员开通。

### 资料准备

操作前请提前准备以下资料：

1. 承运商全套资质：营业执照、道路运输许可证、法人证件、银行开户信息、税点、结算方式。
2. 线路基础信息：始发/目的分拨、车型、温层、发车周期、时效、合作承运商。
3. 报价资料：各重量段首重、续重、最低收费标准。

## 操作步骤

### 承运商资料录入建档

**系统路径**：**运营运输 → 基础资料 → 运输公司**

::: warning 注意事项
网点直发无需录入承运商信息。
:::

1. 进入 **运输公司** 页面，点击 **新增/编辑**。
2. 完整填写带 **\*** 的必填字段，包括：企业名称、法人、联系人、电话、注册地址、营业期限、纳税人识别号、开户行、对公账户、月结/回单/开票规则、税点、绑定车队、备注。
3. 上传资质附件，包括：法人身份证、营业执照、道路运输许可证、开户证明。
   - 单文件 **≤20M**
   - 最多 **5** 份
   - 支持图片、PDF、压缩包
4. 核对信息无误后，点击保存。
5. 提交总部运营完成资质审核。

### 中心直发线路创建与审批

**系统路径**：**经营管理中心 → 路由管理 → 班线管理**

1. 进入 **班线管理** 页面。
2. 筛选线路类型 **中心直发**。
3. 点击 **新增干线**，进入创建页面。
4. 填写必填信息，包括：始发分拨、目的分拨、运营车型、温层、发车周期、生效/失效时间、计划发车/到达时间、跨越天数、合作承运商、费用说明、附件。
5. 保存时如系统提示 **始发/目的距离为空**，请联系总部当班调度维护数据后再提交。
6. 提交审批。

中心直发审批流程：

1. 省区运营发起。
2. 总部运营审核。
3. 总部网络审核。

网点直发审批流程：

1. 网点发起线路申请。
2. 所属省区网管审核。
3. 总部网络审核。

列表支持 **复制新增**、**查看版本记录**。

### 中心直发专属报价配置

**系统路径**：**财务管理 → 报价管理 → 特定业务报价-中心直发报价**

1. 进入 **特定业务报价-中心直发报价** 页面。
2. 点击 **新增**。
3. 填写报价名称。
4. 选择已审批生效的直发线路、适用产品、报价起止日期。
5. 维护重量区间价格：新增多行，分别设置首重、续重、最低收费。
6. 保存并提交审核。
7. 审核状态分为 **未审核**、**通过**、**驳回**。
8. 如报价被驳回，可修改后重新提交。
9. 可使用 **复制报价** 快速创建报价。

### 网点录单 / 订单转运单选择路由方案

**系统路径**：**鲸天系统 → 订单管理 → 网点录单**

1. 进入 **网点录单** 页面。
2. 填写收发信息。
3. 在路由方案区域查看可选方案：
   - **默认路由**
   - 已开通的 **中心直发线路**
4. 根据业务需要选择目标线路。
5. 系统自动匹配对应运价。
6. 核对费用无误后保存。

::: tip 显示规则
- 未开通匹配的中心直发线路：仅展示 **默认路由**，直发选项隐藏。
- 存在生效直发线路：同时展示 **默认路由** 和 **中心直发**，操作员按需单选。
:::

::: danger 重点提醒
中心直发线路 **不可使用包仓政策**，仅 **默认路由** 支持包仓勾选。
:::

### 寄件运单改单路由相关操作

**系统路径**：**寄件运单管理 → 运单详情 → 编辑**

1. 进入运单详情后，点击 **编辑**。
2. 如信息填写错误，可点击 **重置**，一键恢复原始数据。
3. 未发车运单可重新切换路由方案（**默认路由 / 中心直发**）。
4. 切换后，系统会重新核算运费。

::: danger 重点提醒
货物已从首中心发出后，**禁止修改目的网点、温层、产品类型**，路由方案同步不可变更。
:::

### 直发运单面单打印规则

**系统路径**：**寄件运单管理 → 面单打印**

1. 进入 **面单打印** 页面。
2. 打印直发运单面单。
3. 查看面单三段码第一段，直发运单会增加直发标识。
   - 格式示例：**“1直13”**
   - 数字代表班次
   - **“直”** 标记中心直发线路
   - 后接下一操作分拨代码
4. 面单会完整展示产品、温层、承运商、分拨节点信息，用于区分普通干线与直达干线运单。

### 整体审批链路

1. 省区运营录入承运商、发起中心直发线路新增。
2. 总部运营审核承运资质、线路运力时效。
3. 总部网络维护对应中心直发报价。
4. 线路和报价全部审批生效后，网点录单页面才可选择中心直发路由下单。

## 操作结果

完成以上配置后，应达到以下结果：

1. 承运商资料已保存并提交总部运营审核。
2. 中心直发线路已完成审批并生效。
3. 中心直发报价已维护并审核通过。
4. 网点录单页面可按匹配规则展示 **默认路由** 或 **中心直发**。
5. 选择中心直发后，系统可自动匹配对应运价。
6. 直发运单面单三段码第一段带有 **“直”** 标识。

## 注意事项

::: danger 重点提醒
1. 中心直发需省区主动向总部申请开通，无匹配直达线路时，录单仅能选择 **默认路由**。
2. 中心直发不支持包仓政策，仅 **默认路由** 可勾选包仓。
3. 货物已从始发中心发出后，改单禁止修改 **目的网点、温层、产品及路由方案**。
4. 线路存在生效时效，过期后录单页面会自动隐藏该直发选项，需编辑延期并重新审核。
5. 一条直发线路可配置多套分时段、分产品报价，录单时系统自动匹配当期价格。
:::

## 常见问题

### 常见异常与处理方式

| 序号 | ❌ 异常现象 / 报错提示 | 🔍 常见原因 | 🛠️ 解决方案 |
|------|-------------------------------|-----------------|--------------------|
| 1 | 承运商保存失败，缺少资质附件 | 营业执照/道路运输证等文件未上传 | 补齐全套资质后重新提交 |
| 2 | 新增线路提示始发/目的距离为空 | 系统无两分拨里程基础数据 | 联系总部当班调度维护距离后保存 |
| 3 | 报价页面找不到目标直发线路 | 线路未完成两级审批/已过期 | 等待审批通过或编辑延期重审 |
| 4 | 录单无中心直发选项 | 收发分拨间无生效直达线路 | 使用默认路由，或向总部申请开通线路 |
| 5 | 改单无法切换路由方案 | 货物已从首中心发出 | 不可修改路由，仅可处理其他可编辑字段 |
| 6 | 录单选择中心直发，包仓按钮置灰 | 直发线路不支持包仓政策 | 切换默认路由才可使用包仓 |

### Q1：默认路由和中心直发有什么区别？

A：**默认路由** 是网络自有干线，周期固定、无需申请，支持包仓；**中心直发** 是省区申请三方直达运力，发车周期不固定，不可使用包仓。

### Q2：录单看不到中心直发选项怎么办？

A：当前收发分拨之间无审批生效的直达线路，只能选用 **默认路由**。如需直达，请联系省区运营申请开通。

### Q3：货物发车后还能改路由方案吗？

A：不能。首中心已发车后，**目的网点、温层、产品、路由** 全部锁定，不可修改。

### Q4：直发运单面单如何快速识别？

A：面单三段码第一段带有 **“直”** 字标识，代表中心直达线路。

### Q5：线路、报价分别对接哪位负责人？

A：承运商/线路审批：纪宗君 **18506489666**；直发报价维护：徐思杨 **17317300721**。

### Q6：中心直发可以勾选包仓吗？

A：不可以。包仓仅针对 **默认自营路由** 开放。

### Q7：改单填错信息如何快速复原？

A：改单页面有 **重置** 按钮，点击后可一键恢复原始运单信息。
