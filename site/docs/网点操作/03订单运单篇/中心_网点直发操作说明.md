## 一、业务场景与名词解释

### 1.1 业务场景

为区分干线运输模式，系统提供<strong>默认路由、中心直发</strong>两种路由方案。默认路由为网络自有固定干线；中心直发需省区主动申请三方直达运力。完整流程包含承运商建档、中心直发线路创建审批、直发报价配置、录单路由选择、运单改单、面单打印全链路操作，线路与报价审批生效后方可在录单界面选用直达方案。

### 1.2 核心名词解释

1. <strong>默认路由</strong>：中通网络自营干线，发车周期固定、时效稳定，无需额外申请，所有运单默认可选。
2. <strong>中心直发</strong>：省区向总部申请的三方分拨直达运力，两点分拨无中转；发车周期不固定，开通后方可在录单界面选择，不支持包仓政策。
3. <strong>路由方案</strong>：网点录单、订单转运单页面可切换干线运输模式，分为默认路由、中心直发两类。
4. <strong>三段码</strong>：面单第一段编码，直发运单增加“直”标识区分直达线路。

## 二、前置准备与环境配置

### 2.1 账号与权限要求

操作人员需拥有<strong>承运商新增、班线管理、特定业务报价、网点录单、运单改单、面单打印</strong>权限；无权限联系系统管理员开通。

### 2.2 资料前置收集

1. 承运商全套资质：营业执照、道路运输许可证、法人证件、银行开户信息、税点、结算方式；
2. 线路基础信息：始发/目的分拨、车型、温层、发车周期、时效、合作承运商；
3. 报价资料：各重量段首重、续重、最低收费标准。

### 2.3 系统入口

1. 承运商录入：运营运输→基础资料→运输公司
2. 线路申请：经营管理中心→路由管理→班线管理（筛选中心直发）
3. 报价维护：财务管理→报价管理→特定业务报价-中心直发报价
4. 网点录单：订单管理→网点录单（路由方案选择区）
5. 运单改单：寄件运单管理→运单编辑
6. 面单打印：寄件运单管理→面单打印

## 三、场景化标准操作步骤

### 3.1 场景一：承运商资料录入建档

<strong>系统路径</strong>：运营运输 → 基础资料 → 运输公司

::: warning 注意事项
网点直发无需录入承运商信息

1. 点击【新增/编辑】，完整填写带\*必填字段：企业名称、法人、联系人、电话、注册地址、营业期限、纳税人识别号、开户行、对公账户、月结/回单/开票规则、税点、绑定车队、备注。
2. 上传资质附件：法人身份证、营业执照、道路运输许可证、开户证明；单文件≤20M，最多5份，支持图片/PDF/压缩包。
3. 信息核对无误后保存，提交总部运营完成资质审核。

![image.png](./images/中心_网点直发操作说明_1_a9158e4e.png)

:::

### 3.2 场景二：中心直发线路创建与审批

<strong>系统路径</strong>：经营管理中心 → 路由管理 → 班线管理

1. 筛选线路类型【中心直发】，点击【新增干线】进入创建页面。
2. 填写必填信息：始发分拨、目的分拨、运营车型、温层、发车周期、生效/失效时间、计划发车/到达时间、跨越天数、合作承运商、费用说明、附件。
3. 保存校验：系统提示始发/目的距离为空时，联系总部当班调度维护数据后再提交。
4. 审批流程：省区运营发起→总部运营审核→总部网络审核；列表支持复制新增、查看版本记录，

网点直发审批流程：网点发起线路申请→所属省区网管审核→总部网络审核。

![image.png](./images/中心_网点直发操作说明_2_cfb6c218.png)

### 3.3 场景三：中心直发专属报价配置

<strong>系统路径</strong>：财务管理 → 报价管理 → 特定业务报价-中心直发报价

1. 点击【新增】，填写报价名称，选择已审批生效的直发线路、适用产品、报价起止日期。
2. 维护重量区间价格：新增多行，分别设置首重、续重、最低收费。
3. 保存提交审核，状态分为未审核、通过、驳回；驳回可修改后重提，支持复制报价快速创建。

![image.png](./images/中心_网点直发操作说明_3_428d0ee7.png)

### 3.4 场景四：网点录单/订单转运单 路由方案选择

<strong>系统路径</strong>：鲸天系统 → 订单管理 → 网点录单

1. 页面路由方案区域展示两类选项：默认路由、已开通的中心直发线路。
2. 显示规则：

- 未开通匹配的中心直发线路：仅展示默认路由，直发选项隐藏；
- 存在生效直发线路：同时展示两类方案，操作员按需单选。

3. 限制规则：中心直发线路<strong>不可使用包仓政策</strong>，仅默认路由支持包仓勾选。
4. 操作流程：填写收发信息后，在路由方案区选中目标线路，系统自动匹配对应运价，核对费用后保存。

![image.png](./images/中心_网点直发操作说明_4_d3c46b22.png)

### 3.5 场景五：寄件运单改单路由相关操作

<strong>系统路径</strong>：寄件运单管理 → 运单详情 → 编辑

1. 页面新增【重置】按钮，信息填写错误可一键恢复原始数据。
2. 核心限制：货物已从首中心发出后，<strong>禁止修改目的网点、温层、产品类型</strong>，路由方案同步不可变更。
3. 未发车运单：可重新切换路由方案（默认/中心直发），系统重新核算运费。

### 3.6 场景六：直发运单面单打印规则

<strong>系统路径</strong>：寄件运单管理 → 面单打印

1. 面单三段码第一段增加直发标识，格式示例：“1直13”；数字代表班次，“直”标记中心直发线路，后接下一操作分拨代码。
2. 完整展示产品、温层、承运商、分拨节点信息，区分普通干线与直达干线运单。

![image.png](./images/中心_网点直发操作说明_5_96a90e56.png)

### 3.7 场景七：整体全流程审批链路

1. 省区运营录入承运商、发起中心直发线路新增；
2. 总部运营审核承运资质、线路运力时效；
3. 总部网络维护对应中心直发报价；
4. 线路+报价全部审批生效，网点录单页面才可选择中心直发路由下单。

## 四、核心路由操作约束

1. 中心直发需省区主动向总部申请开通，无匹配直达线路时录单仅能选择默认自营干线。
2. 中心直发不支持包仓政策，仅默认路由可勾选包仓。
3. 货物已从始发中心发出，改单禁止修改目的网点、温层、产品及路由方案。
4. 线路存在生效时效，过期后录单页面自动隐藏该直发选项，需编辑延期重审。
5. 一条直发线路可配置多套分时段、分产品报价，录单自动匹配当期价格。

## 五、常见异常与兜底方案

| 序号 | ❌ 异常现象 / 报错提示 | 🔍 常见原因 | 🛠️ 解决方案 |
|------|-------------------------------|-----------------|--------------------|
| 1 | 承运商保存失败，缺少资质附件 | 营业执照/道路运输证等文件未上传 | 补齐全套资质后重新提交 |
| 2 | 新增线路提示始发/目的距离为空 | 系统无两分拨里程基础数据 | 联系总部当班调度维护距离后保存 |
| 3 | 报价页面找不到目标直发线路 | 线路未完成两级审批/已过期 | 等待审批通过或编辑延期重审 |
| 4 | 录单无中心直发选项 | 收发分拨间无生效直达线路 | 使用默认路由，或向总部申请开通线路 |
| 5 | 改单无法切换路由方案 | 货物已从首中心发出 | 不可修改路由，仅可处理其他可编辑字段 |
| 6 | 录单选择中心直发，包仓按钮置灰 | 直发线路不支持包仓政策 | 切换默认路由才可使用包仓 |

## 六、高频常见问题（FAQ）

### 6.1 Q1：默认路由和中心直发有什么区别？

○ A：默认路由是网络自有干线，周期固定、无需申请，支持包仓；中心直发是省区申请三方直达运力，发车周期不固定，不可使用包仓。

### 6.2 Q2：录单看不到中心直发选项怎么办？

○ A：当前收发分拨之间无审批生效的直达线路，只能选用默认路由，如需直达联系省区运营申请开通。

### 6.3 Q3：货物发车后还能改路由方案吗？

○ A：不能，首中心已发车后，目的网点、温层、产品、路由全部锁定不可修改。

### 6.4 Q4：直发运单面单如何快速识别？

○ A：面单三段码第一段带有“直”字标识，代表中心直达线路。

### 6.5 Q5：线路、报价分别对接哪位负责人？

○ A：承运商/线路审批：纪宗君 18506489666；直发报价维护：徐思杨 17317300721。

### 6.6 Q6：中心直发可以勾选包仓吗？

○ A：不可以，包仓仅针对默认自营路由开放。

### 6.7 Q7：改单填错信息如何快速复原？

○ A：改单页面有【重置】按钮，点击一键恢复原始运单信息。
