# 回单

## 一、适用场景

客户寄件后需要签收凭证（回单）时，网点可根据客户需求办理**纸质回单**或**电子回单**，并支持回单状态查询、上传与编辑。

## 二、前置条件

- 账号需拥有**回单录单、回单查询、回单上传、回单编辑**权限。权限不足请联系系统管理员开通。
- 电脑正常联网，可稳定访问鲸天系统。
- 办理纸质回单需准备实体回单单据；办理电子回单需准备签收凭证图片（用于系统上传）。

## 三、操作入口

鲸天系统 → 经营管理中心 → 运单管理

- 纸质回单：**网点录单 → 录回单**
- 电子回单：**回单管理**

## 四、操作步骤

### 4.1 场景一：纸质回单录单

1. 进入**网点录单**页面，选择**录回单**类型。
2. 输入原运单号或回单号，系统自动带出原运单全部信息（该类信息**不可手动编辑**）。
3. 核对信息无误后，点击**【保存】**完成纸质回单登记。

   ![image.png](./images/中心_网点直发操作说明_1_a9158e4e.png)

::: tip 补充说明
办理纸质回单仅扣除1个主单额度。
:::

### 4.2 场景二：电子回单登记与编辑

#### (1) 回单查询

在**回单管理**页面，可通过运单号、回单类型、回单状态、寄/派件网点、寄件时间等条件筛选，查询对应回单记录。

#### (2) 电子回单登记

::: danger 重点提醒
- 仅**派方**（派件网点、派件所属网点及下属二级网点承包区）可执行登记操作。
- 原运单状态必须为**已签收**。
:::

1. 选中目标运单，点击**【详情】**进入回单登记页面。
2. 点击上传区域，选择签收凭证图片完成上传，提交后即完成电子回单登记。
3. 同一运单仅允许留存一条电子回单数据。

#### (3) 电子回单编辑

若回单上传信息有误，可在对应记录处点击**编辑**，重新上传图片或修改内容后保存。

## 五、操作结果

- 纸质回单：保存后系统生成对应回单记录，扣除1个主单额度。
- 电子回单：上传完成后，回单状态更新为已登记，且仅保留一条记录。

## 六、注意事项

- 纸质回单输入运单号后，系统自动同步原运单信息，所有字段**禁止编辑**。
- 电子回单只能由派方操作，且运单必须已签收。
- 同一运单电子回单只能登记一次，若需修改请使用编辑功能。
- 图片上传失败时，请检查图片格式、文件大小及网络状态。

## 七、常见问题

| 异常现象 | 常见原因 | 解决方案 |
| --- | --- | --- |
| 录入运单号后无法带出原单信息 | 运单号/回单号输入错误，或单号不存在 | 仔细核对单号，确认无误后重新输入 |
| 无法发起电子回单登记 | 当前账号不属于派方，或运单未完成签收 | 1. 使用派方网点账号操作；2. 等待运单完成签收后再登记 |
| 电子回单无法重复新增 | 系统限制同一运单仅保留一条电子回单记录 | 直接对已有记录进行编辑修改 |
| 纸质回单保存失败 | 主单余额不足 | 前往账户充值主单额度后重新保存 |
| 图片上传失败 | 图片格式不符、文件过大、网络异常 | 检查图片格式，压缩文件大小，刷新网络后重试 |

## 八、高频问题

::: tip Q1
**纸质回单录入后可以修改运单信息吗？**
A：不可以，输入单号后系统自动同步原运单信息，所有字段均禁止编辑。
:::

::: tip Q2
**纸质回单会扣除面单额度吗？**
A：会，每办理一笔纸质回单，系统扣除1个主单额度。
:::

::: tip Q3
**哪些账号可以操作电子回单？**
A：仅派件网点、派件所属网点、二级网点承包区等派方账号可操作。
:::

::: tip Q4
**运单签收前能上传电子回单吗？**
A：不可以，系统要求运单状态为已签收时，才能进行电子回单登记。
:::

::: tip Q5
**电子回单登记错误怎么处理？**
A：无需重新新增，直接在原有回单记录上点击编辑，修改后保存即可。
:::
