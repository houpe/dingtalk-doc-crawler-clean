# 删单

## 一、适用场景

网点客户下单后临时弃件、客户取消发货，网点可自主操作删单；超出网点操作时效或权限不足时，可申请总部后台协助取消；长期无任何物流轨迹的运单，系统会自动定时清理作废运单，同时统一管控面单库存返还、手续费扣费、回单登记撤销等财务与物料规则。

## 二、前置条件

### 2.1 账号与权限

- **网点自主删单**：一级网点、二级网点、代理点、承包区账号拥有操作权限；其他机构无权限。
- **总部删单**：仅总部后台管理员账号具备取消权限。

### 2.2 前置校验约束

已作废、已签收、满帮来源运单，任何渠道均无法执行取消操作。

### 2.3 配套工具/链接

- 运营系统登录入口：系统PC后台
- 操作路径：**订单运单管理** → **运单列表**

## 三、操作入口

- 网点自主：登录系统 → 左侧菜单栏【**订单运单篇**】→ **运单列表**
- 总部后台：总部后台 → **订单运单管理** → **运单列表**

## 四、操作步骤

### 4.1 场景一：网点自主取消运单（优先级1）

#### (1) 操作前置校验（全部满足才可操作）

1. **机构权限**：当前账号所属机构为一级网点/二级网点/代理点/承包区；
2. **时间限制**：下单当日23:59:59前操作，跨日无法取消；
3. **轨迹限制**：仅存在寄件网点操作、首分拨/集配站到车轨迹，无其他物流操作轨迹（含问题件）；
4. **运单排除**：已作废、已签收、满帮运单不可删。

#### (2) 核心操作步骤

1. 在运单列表筛选目标运单，点击运单右侧【**取消运单**】按钮；
2. 弹窗确认取消原因，提交操作；
3. 系统自动校验规则，校验通过后运单状态变更为【**已取消**】。

#### (3) 取消后物料&财务规则

1. **面单库存**：运单未到达分拨中心卸车/到件，主单、子单退回开单业务员/网点；已到分拨中心则不返还；
2. **回单物料**：无论何种情况，回单库存不予返还；
3. **保价费**：已参保订单，保价费不予退还；
4. **手续费**：网点自主取消，不收取3元/单手续费。

### 4.2 场景二：总部后台取消运单（优先级2）

#### (1) 操作前置校验

1. **轨迹限制**：除寄件网点操作、首分拨/集配站到车外，存在其他物流轨迹（含问题件）则无法取消；
2. **无时间限制**：不受下单当日23:59:59时效约束；
3. **运单排除**：已作废、已签收、满帮运单不可删。

#### (2) 核心操作步骤

1. 网点提交取消申请，提供运单号；
2. 总部管理员在运单列表检索运单，点击【**总部取消**】；
3. 填写取消事由，确认提交。

#### (3) 取消后物料&财务规则

1. **面单库存**：仅轨迹为开单/改单/已受理/取消运单/撤销扫描/重算费用/收件/发件/封签发车/留仓时，返还主单、子单至业务员/网点；其余轨迹不返还；
2. **回单处理**：若为回单运单，同步撤销回单登记；
3. **财务扣费**：自动扣取**3元/单**手续费，推送扣费通知；
4. **运单状态**：更新为【**已取消**】。

### 4.3 场景三：系统自动取消运单（优先级3，定时任务）

无需人工操作，系统每5分钟自动执行JOB扫描运单，满足条件自动作废。

#### (1) 自动取消前置条件

1. **时效条件**：运单开单满72小时；
2. **轨迹条件**：无任何物流操作轨迹（包含问题件轨迹，有则不自动取消）；
3. **运单排除**：已作废、已签收、派送中、满帮运单、回单产品运单不参与自动取消。

#### (2) 自动取消后规则

1. **取消主体**：取消网点标记为总部；
2. **面单库存**：同总部取消返还逻辑；
3. **回单处理**：回单运单同步撤销回单登记；
4. **财务扣费**：自动收取**3元/单**手续费，推送通知；
5. **订单联动**：若运单来源于电商订单，同步更新上游订单状态为「运单已删除」；
6. **运单状态**：更新为【**已取消**】。

## 五、操作结果

- 网点自主取消：运单状态变为【**已取消**】，面单库存按规定返还或扣除，手续费不扣。
- 总部取消：运单状态变为【**已取消**】，自动扣3元手续费，回单登记撤销。
- 自动取消：运单状态变为【**已取消**】，扣3元手续费，如有上游订单同步更新。

## 六、注意事项

::: danger 重点提醒
- **不可取消的运单**：已作废、已签收、满帮来源运单，任何渠道均无法取消。
- **取消运单后保价费不退**：保价费一经参保不予退还。
- **回单物料不返还**：无论哪种取消方式，回单库存不予返还。
- **网点自主取消无手续费**，总部取消或自动取消会收取**3元/单**手续费。
:::

::: warning 注意事项
- **网点自主取消时间限制**：仅支持下单当日23:59:59前操作。
- **轨迹拦截规则**：存在问题件、收件、派送等非首中心到车轨迹时，网点、总部、系统均无法取消。
- **自动取消72小时无轨迹**：开单满72小时且无任何操作轨迹（含问题件）才会触发。
- **面单返还条件**：是否返还取决于运单是否已到达分拨中心卸车/到件。
:::

## 七、常见异常与兜底方案

| 序号 | ❌ 异常现象 / 报错提示 | 🔍 常见原因 | 🛠️ 解决方案 |
|------|-------------------------------|-----------------|--------------------|
| 1 | 网点点击取消运单按钮置灰不可操作 | 1.账号机构不是一二级网点/代理点/承包区；
2. 运单已跨当日23:59:59；
3. 存在首中心到车以外物流轨迹/问题件；
4. 运单为已作废/已签收/满帮运单 | 1.核对账号所属机构，无权限联系总部；
2. 超出时效走总部取消流程；
3. 查询运单轨迹，存在后续扫描无法删单；
4. 此类运单永久无法取消 |
| 2 | 总部后台取消提示“轨迹不允许删除” | 运单产生收件、派送、问题件等首中心到车以外操作记录 | 该运单已流转，无法取消，告知客户无法作废 |
| 3 | 运单开单超72小时未自动取消 | 运单存在任意物流轨迹（含问题件） | 人工核实轨迹，无法触发自动取消，如需作废走总部流程 |
| 4 | 取消后未返还面单库存 | 运单已到达分拨中心卸车/到件轨迹 | 系统规则不支持返还，无法补回面单 |
| 5 | 网点自主取消后仍被扣3元手续费 | 操作实际为总部后台取消/系统自动取消，并非网点自主操作 | 核对取消操作人，确认操作渠道 |

## 八、常见问题

### 8.1 Q1：网点当日超时，还能删单吗？

A：网点自主取消仅支持下单当日23:59:59前操作；超时可联系总部后台协助取消，但会收取3元/单手续费。

### 8.2 Q2：运单产生问题件，还能取消吗？

A：不能。问题件属于有效操作轨迹，网点、总部、系统自动取消均会拦截该运单。

### 8.3 Q3：取消运单后保价费、回单会退款/返还吗？

A：保价费一经参保不予退还；回单物料无论哪种取消方式都不返还，回单运单仅会撤销登记。

### 8.4 Q4：系统多久执行一次自动取消任务？

A：自动取消定时任务每5分钟运行一次，扫描全量符合72小时无轨迹的运单。

### 8.5 Q5：上游电商订单绑定运单删除后，订单状态会变化吗？

A：会。系统自动同步更新原订单状态为「运单已删除」。

### 8.6 Q6：哪些运单永远无法取消？

A：已作废、已签收、满帮渠道运单，无论网点、总部、系统自动任务都不能删除。
