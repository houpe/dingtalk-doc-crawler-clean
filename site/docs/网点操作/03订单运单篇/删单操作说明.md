# 删单

## 一、适用场景

本文适用于网点客户下单后临时弃件、客户取消发货等需要取消运单的场景。

可处理的删单方式包括：

1. **网点自主取消运单**：网点在权限和时效内自行取消。
2. **总部后台取消运单**：网点超出操作时效或权限不足时，申请总部后台协助取消。
3. **系统自动取消运单**：长期无任何物流轨迹的运单，由系统定时任务自动作废。

涉及的规则包括面单库存返还、手续费扣费、回单登记撤销等。

## 二、前置条件

### 2.1 账号与权限

1. **网点自主删单**：仅 **一级网点、二级网点、代理点、承包区** 账号拥有操作权限；其他机构无权限。
2. **总部删单**：仅 **总部后台管理员** 账号具备取消权限。
3. **系统自动取消**：无需人工操作，由系统定时任务自动执行。

### 2.2 运单基础限制

::: danger 重点提醒
以下运单任何渠道均无法执行取消操作：

1. **已作废** 运单；
2. **已签收** 运单；
3. **满帮来源运单**。
:::

### 2.3 核心名词说明

1. **操作轨迹**：运单产生的所有系统操作记录，包含收件、发件、中心到车、问题件、派送、签收等扫描动作；仅开单、改单、首中心到车轨迹不拦截删单。
2. **满帮运单**：第三方渠道下单的特殊运单类型，不支持网点、总部、系统自动取消。
3. **回单运单**：附加回单服务的运单，取消后需同步撤销回单登记，回单物料不予返还。
4. **自动取消JOB**：系统定时任务，**每5分钟** 扫描一次符合条件的运单，执行自动作废。

## 三、操作入口

### 3.1 网点自主取消入口

- 系统：**冷链快递运营管理平台**
- 登录入口：**系统PC后台**
- 功能模块：**订单运单-运单取消/删单管理**
- 操作路径：**订单运单管理 → 运单列表**
- 系统功能路径：登录系统 → 左侧菜单栏 **【订单运单篇】** → **运单列表**
- 快捷直达链接：**运单管理列表页**

### 3.2 总部后台取消入口

- 系统功能路径：**总部后台 → 订单运单管理 → 运单列表**

### 3.3 系统自动取消入口

无需人工进入菜单操作，系统通过自动取消JOB定时执行。

## 四、操作步骤

### 4.1 场景一：网点自主取消运单

适用于网点在规定时间内，且运单未产生限制轨迹时自行取消。

#### (1) 操作前校验

::: danger 重点提醒
网点自主取消需同时满足以下条件：

1. 当前账号所属机构为 **一级网点 / 二级网点 / 代理点 / 承包区**；
2. 必须在 **下单当日23:59:59前** 操作，跨日无法取消；
3. 运单仅存在 **寄件网点操作、首分拨/集配站到车轨迹**，无其他物流操作轨迹（含问题件）；
4. 运单不是 **已作废、已签收、满帮运单**。
:::

#### (2) 操作流程

1. 登录系统，进入 **订单运单管理 → 运单列表**。
2. 在运单列表中筛选目标运单。
3. 点击运单右侧 **【取消运单】** 按钮。
4. 在弹窗中确认取消原因。
5. 提交操作。
6. 系统自动校验规则，校验通过后，运单状态变更为 **【已取消】**。

#### (3) 取消后物料与财务规则

1. **面单库存**：
   - 运单未到达分拨中心卸车/到件时，主单、子单退回开单业务员/网点；
   - 已到分拨中心则不返还。
2. **回单物料**：无论何种情况，回单库存不予返还。
3. **保价费**：已参保订单，保价费不予退还。
4. **手续费**：网点自主取消，不收取 **3元/单** 手续费。

### 4.2 场景二：总部后台取消运单

适用于网点无法自主取消，需要总部后台管理员协助取消的场景。

#### (1) 操作前校验

::: danger 重点提醒
总部后台取消需满足以下规则：

1. 除寄件网点操作、首分拨/集配站到车外，存在其他物流轨迹（含问题件）则无法取消；
2. 总部后台取消不受 **下单当日23:59:59** 时效限制；
3. **已作废、已签收、满帮运单** 不可删。
:::

#### (2) 操作流程

1. 网点提交取消申请，并提供运单号。
2. 总部管理员进入 **总部后台 → 订单运单管理 → 运单列表**。
3. 在运单列表中检索目标运单。
4. 点击 **【总部取消】**。
5. 填写取消事由。
6. 确认提交。

#### (3) 取消后物料与财务规则

1. **面单库存**：仅轨迹为开单/改单/已受理/取消运单/撤销扫描/重算费用/收件/发件/封签发车/留仓时，返还主单、子单至业务员/网点；其余轨迹不返还。
2. **回单处理**：若为回单运单，同步撤销回单登记。
3. **财务扣费**：自动扣取 **3元/单** 手续费，并推送扣费通知。
4. **运单状态**：更新为 **【已取消】**。

### 4.3 场景三：系统自动取消运单

系统自动取消无需人工操作。系统每 **5分钟** 自动执行JOB扫描运单，满足条件后自动作废。

#### (1) 自动取消前置条件

::: danger 重点提醒
系统自动取消需同时满足以下条件：

1. 运单开单满 **72小时**；
2. 运单无任何物流操作轨迹（包含问题件轨迹，有则不自动取消）；
3. **已作废、已签收、派送中、满帮运单、回单产品运单** 不参与自动取消。
:::

#### (2) 自动取消后规则

1. **取消主体**：取消网点标记为总部。
2. **面单库存**：同总部取消返还逻辑。
3. **回单处理**：回单运单同步撤销回单登记。
4. **财务扣费**：自动收取 **3元/单** 手续费，并推送通知。
5. **订单联动**：若运单来源于电商订单，同步更新上游订单状态为 **「运单已删除」**。
6. **运单状态**：更新为 **【已取消】**。

## 五、操作结果

取消成功后，可在系统中看到以下结果：

1. 运单状态更新为 **【已取消】**。
2. 符合返还条件的主单、子单会退回至对应业务员/网点。
3. 回单运单会同步撤销回单登记，但回单物料不予返还。
4. 总部后台取消或系统自动取消时，系统自动收取 **3元/单** 手续费并推送通知。
5. 若为系统自动取消且运单来源于电商订单，上游订单状态同步更新为 **「运单已删除」**。

## 六、注意事项

::: warning 注意事项
1. 网点自主取消仅支持 **下单当日23:59:59前** 操作，跨日无法取消。
2. 运单产生问题件、派送、签收等有效物流操作轨迹后，通常无法取消。
3. 网点自主取消不收取 **3元/单** 手续费；总部后台取消和系统自动取消会自动扣取 **3元/单** 手续费。
4. 已参保订单取消后，保价费不予退还。
5. 回单物料无论哪种取消方式均不返还。
6. 满帮运单不支持网点、总部、系统自动取消。
:::

## 七、常见问题

### 7.1 Q1：网点当日超时，还能删单吗？

A：网点自主取消仅支持 **下单当日23:59:59前** 操作；超时可联系总部后台协助取消，但会收取 **3元/单** 手续费。

### 7.2 Q2：运单产生问题件，还能取消吗？

A：不能。问题件属于有效操作轨迹，网点、总部、系统自动取消均会拦截该运单。

### 7.3 Q3：取消运单后保价费、回单会退款或返还吗？

A：保价费一经参保不予退还；回单物料无论哪种取消方式都不返还，回单运单仅会撤销登记。

### 7.4 Q4：系统多久执行一次自动取消任务？

A：自动取消定时任务每 **5分钟** 运行一次，扫描全量符合 **72小时无轨迹** 的运单。

### 7.5 Q5：上游电商订单绑定运单删除后，订单状态会变化吗？

A：会。系统自动同步更新原订单状态为 **「运单已删除」**。

### 7.6 Q6：哪些运单永远无法取消？

A：**已作废、已签收、满帮渠道运单**，无论网点、总部、系统自动任务都不能删除。

### 7.7 Q7：网点点击取消运单按钮置灰不可操作怎么办？

A：可按以下方向排查：

| 序号 | ❌ 异常现象 / 报错提示 | 🔍 常见原因 | 🛠️ 解决方案 |
|------|-------------------------------|-----------------|--------------------|
| 1 | 网点点击取消运单按钮置灰不可操作 | 1.账号机构不是一二级网点/代理点/承包区；

2. 运单已跨当日23:59:59；
3. 存在首中心到车以外物流轨迹/问题件；
4. 运单为已作废/已签收/满帮运单 | 1.核对账号所属机构，无权限联系总部；
2. 超出时效走总部取消流程；
3. 查询运单轨迹，存在后续扫描无法删单；
4. 此类运单永久无法取消 |
| 2 | 总部后台取消提示“轨迹不允许删除” | 运单产生收件、派送、问题件等首中心到车以外操作记录 | 该运单已流转，无法取消，告知客户无法作废 |
| 3 | 运单开单超72小时未自动取消 | 运单存在任意物流轨迹（含问题件） | 人工核实轨迹，无法触发自动取消，如需作废走总部流程 |
| 4 | 取消后未返还面单库存 | 运单已到达分拨中心卸车/到件轨迹 | 系统规则不支持返还，无法补回面单 |
| 5 | 网点自主取消后仍被扣3元手续费 | 操作实际为总部后台取消/系统自动取消，并非网点自主操作 | 核对取消操作人，确认操作渠道 |
