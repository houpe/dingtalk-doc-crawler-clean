# 订单管理

## 一、适用场景

对接菜鸟、抖音、满帮、快手、拼多多等外部平台订单，实现订单自动接入、智能分配网点、多渠道消息提醒、时效考核、转单、撤销及费用核算全流程管理。适用于总部、省区、分拨中心、各级网点工作人员。

## 二、前置条件

- **账号与权限**：需持有鲸天系统、鲸小宝APP对应操作权限，包含 **订单查询、接单、转单、撤销、费用流水查看** 权限。无权限请联系系统管理员开通。
- **物理/环境准备**：设备正常联网，可接收站内信、钉钉、短信消息提醒；电脑端正常登录鲸天系统，移动端安装并登录鲸小宝APP。
- **配套工具**：鲸小宝APP（移动端接单、查单、转单）、鲸天系统（中通冷链官方运营后台）。

## 三、操作入口

请根据具体操作场景在鲸天系统或鲸小宝APP中进入对应菜单。各场景的具体菜单路径见操作步骤。

## 四、操作步骤

### 4.1 场景一：满帮订单自动分配规则

系统自动执行，无需手动操作。

1. 系统优先解析订单内指定网点，直接分配至该网点，不再二次解析省市区。
2. 未解析到指定网点时，通过GIS解析寄件省市区。若区域在黑名单内则直接拦截订单。
3. 非黑名单区域，匹配同省份营业分拨中心：
   - 仅1个分拨则直接分配。
   - 多个分拨则选择直线距离最近的。
   - 无匹配分拨则统一分配至总部。

::: tip 说明
自动分配过程可在订单详情中查看分配记录。
:::

### 4.2 场景二：订单查询与筛选

**系统功能路径**：鲸天系统 → 订单管理 → 全部订单

1. 筛选条件中点击 **订单归属**，默认「全部」，可切换为：
   - **我的订单**：仅查看当前机构订单。
   - **下级订单**：查看下属机构订单。
2. 在 **订单来源** 查询条件中，可精准筛选满帮订单。
3. 支持按订单时间、接单网点、订单状态等条件组合查询。

![image.png](./images/中心_网点直发操作说明_2_cfb6c218.png)

### 4.3 场景三：订单接单与消息提醒

**系统功能路径**：鲸天系统 / 鲸小宝APP → 订单管理 → 待接单

1. 订单下发后，系统根据订单来源推送提醒：
   - **满帮订单**：同步推送 **站内信、钉钉、短信** 三类提醒，同时通知所属上级分拨中心监管。
   - **其他来源订单**：仅推送站内信。
2. 接单倒计时提醒：
   - 满帮订单倒计时 **5-1分钟** 持续钉钉弹窗 / 站内信消息提醒，剩余 **2分钟** 额外发送短信预警。
   - 非满帮订单接单时效为 **60分钟**。
3. 工作人员查看提醒消息，进入待接单列表选中订单，点击 **【预约接单】** 完成接单操作。

### 4.4 场景四：接单超时与自动升级处理

**系统功能路径**：鲸天系统 → 订单管理 → 接单超时列表

1. 时效规则：
   - **满帮订单**：接单时效 **10分钟**，超时罚款 **200元/次**。
   - **其他来源订单**：接单时效 **60分钟**，超时罚款 **100元/次**。
2. 升级规则（仅满帮订单适用）：
   - 二级网点超时 → 升级至一级网点。
   - 一级网点超时 → 升级至分拨中心。
   - 分拨中心超时 → 升级至总部，总部不参与考核。
   - 其他来源订单超时不升级，仅罚款，订单仍在原寄件网点。
3. 超时后系统自动生成 **超时记录、升级处理记录**，可在订单详情内查看。

### 4.5 场景五：订单转单操作

#### (1) 鲸天系统端

**系统功能路径**：订单管理 → 待接单 → 选中订单 → **【转单】**

- **满帮订单**：
  - 二级网点：无转单权限。
  - 一级网点：可转单至自身下级网点。
  - 分拨中心：可转单至下属一级、二级网点。
  - 总部：可转单至全国所有有效网点及各级分拨中心。
- **其他来源订单**：仅总部可转单，范围同满帮总部。

转单完成后，订单详情会记录转单类型与操作记录。

#### (2) 鲸小宝APP端

**系统功能路径**：鲸小宝 → 待接单 / 待下级接单 → 订单列表 → **【转单】**

1. APP新增 **待下级接单** 入口，可单独查看下属机构订单。
2. 转单权限规则与PC端一致，同时支持按订单来源、产品类型筛选订单。

### 4.7 场景七：费用流水与定金管理

#### (1) 超时罚款查询

**系统功能路径**：鲸天系统 → 订单管理 → 费用流水

筛选流水类型为 **订单**，可查询订单超时罚款金额、收支主体、结算状态等信息。

#### (2) 定金返还规则

**系统功能路径**：订单管理 → 订单详情

1. 订单转为运单后，系统自动扣除定金与服务费，并同步告知满帮平台。
2. 订单签收回调、完成卸货后，按 **定金是否返还** 标识处理：
   - 选择 **不返还**：直接抵扣财务款项（定金抵费用）。
   - 选择 **返还**：正常扣费，不做抵扣。

## 五、操作结果

完成各场景操作后，可在对应列表或详情页查看：订单状态更新、接单记录、超时记录、升级记录、转单记录、撤销状态、费用流水及定金标识等信息。

## 六、注意事项

- ⚠️ 满帮订单接单时效严格 **10分钟**，超时罚款并自动升级，务必及时处理。
- 📌 转单权限按机构层级划分，不可越权操作。目标网点需为有效状态。
- 📌 接单提醒渠道：满帮订单为站内信+钉钉+短信；其他订单仅站内信。
- 📌 定金返还标识在订单详情中设置，一旦签收卸货后不可更改。
- 📌 超时罚款流水需等待系统结算完成后才能查询到。

## 七、常见问题

**Q1：满帮订单标准接单时效是多久？超时如何处罚？**
A：接单时效 **10分钟**，超时罚款 **200元/次**，同时订单逐级自动升级至上级机构。

**Q2：不同层级机构的转单范围有什么区别？**
A：二级网点不可转单；一级网点可转下级网点；分拨可转下属一、二级网点；总部可转全国所有网点与分拨。

**Q3：待接单和待下级接单有什么区别？**
A：**待接单** 仅查看当前登录机构订单；**待下级接单** 可查看本机构下属所有网点的订单，支持统一处理。

**Q4：订单定金在什么情况下会抵扣费用？**
A：订单详情标识为 **定金不返还** 时，完成签收卸货后定金直接抵扣财务款项；标识为 **返还** 则正常扣费。

**Q5：如何区分订单是网点、省区还是总部发起撤销？**
A：在 **撤销跟进** 页面，通过撤销分类标签即可区分网点撤销、省区撤销、总部撤销。

**Q6：收不到满帮订单接单提醒怎么办？**
A：检查站内信、钉钉、短信通知权限是否开启；刷新页面/重启APP；联系管理员核对预留联系手机号。

**Q7：找不到下级机构的满帮订单怎么办？**
A：切换「订单归属-下级订单」或进入鲸小宝「待下级接单」入口重新筛选。

**Q8：账号无转单按钮或转单失败怎么办？**
A：对照转单权限规则确认当前机构层级是否有权限，并检查目标网点状态是否正常。

**Q9：查询不到超时罚款流水怎么办？**
A：检查筛选条件是否正确，或等待系统结算完成后再次查询。

**Q10：订单被系统拦截无法接单怎么办？**
A：核对寄件地址是否命中平台黑名单，按要求调整后重新下单。
