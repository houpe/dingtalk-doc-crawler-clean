# 微信AI客服操作手册

## 一、适用场景

本文适用于冷链零担相关业务中，商家、网点在微信群内进行**报价**、**超区查询**、**查件**、**下单**等咨询的场景。

通过微信 AI 客服机器人，可在微信群内自动回复客户问题，减少一线网点人工答复工作量。

## 二、前置条件

1. 网点相关人员需注册并登录**企业微信账号**。
2. 需加入**中通冷链企业微信组织**。
3. 如需使用机器人管理功能，需具备菜单权限：**机器人客服管理**。
4. 前期推广期间，权限需先收集网点人员名单，由**总部技术支持**统一配置。
5. 如需使用存量微信群，需先将微信群转换为企业微信群。

::: warning 注意事项
注册或加入企业微信组织时，如需协助，请联系总部技术支持人员：**@李勇(总部IT-李勇)**、**@赵强如**。
:::

## 三、操作入口

- 企业微信下载地址：[https://work.weixin.qq.com/#indexDownload](https://work.weixin.qq.com/#indexDownload)
- 机器人客服管理菜单：**机器人客服管理**
- 其他系统路径请以系统实际菜单路径为准。

## 四、操作步骤

### 4.1 注册企业微信账号

1. 下载并安装企业微信。

   下载地址：[https://work.weixin.qq.com/#indexDownload](https://work.weixin.qq.com/#indexDownload)

   ![image.png](./images/微信AI客服操作手册_10_b5173766.png)

2. 注册企业微信账号。
3. 联系总部技术支持人员，加入**中通冷链企业微信组织**。

### 4.2 添加企业微信机器人账号（非必要步骤）

1. 扫码添加企业微信机器人账号。

   ![image.png](./images/微信AI客服操作手册_11_75c77d15.png)

2. 或者加入**中通冷链**企业后，在企业微信内自行搜索：**AI客服1号**。

::: warning 注意事项
后续视业务情况，机器人账号可能会增加。
:::

### 4.3 新建微信群并添加机器人

1. 在企业微信内创建客户微信群。
2. 拉入客户、企业微信机器人账号。
3. 创建群时，请确认群内有**个人微信好友**。

::: danger 重点提醒
创建微信群时，一定需要有**个人微信好友**。否则该群会成为企业内部群，后续不能添加个人微信好友。

如没有其他个人微信好友，可使用自己的企业微信添加自己的个人微信。
:::

![image.png](./images/微信AI客服操作手册_12_33663e24.png)![image.png](./images/微信AI客服操作手册_13_bbe3b1f0.png)![image.png](./images/微信AI客服操作手册_14_57d6c690.png)![image.png](./images/微信AI客服操作手册_15_03a0ebf9.png)

### 4.4 在微信群内输入激活码

1. 获取对应网点的**激活码**。
2. 在微信群内输入**激活码**。
3. 激活成功后，即可使用微信 AI 客服功能。

![image.png](./images/微信AI客服操作手册_16_b56b7155.png)

![image.png](./images/微信AI客服操作手册_17_4a1a0dc3.png)

![image.png](./images/微信AI客服操作手册_18_724f0a33.png)

::: danger 重点提醒
1. 每个**激活码**只能使用**一次**，新群需要重新复制激活码。
2. 激活码与网点绑定。省区网管代网点操作时，请在**鲸天**切换至对应网点后，再获取相应激活码。
3. 该功能主要针对**1/2级网点**。**总部/省区/分拨/集配站/财务中心请勿绑定**。
:::

## 五、业务功能使用说明

### 5.1 查件

1. 在微信群内输入**单号**。
2. 机器人会查询并回复对应运单轨迹。

![image.png](./images/微信AI客服操作手册_19_9e6a94c5.png)

::: tip 功能说明
**查件**为基础功能，默认开启。如客户群内不需要，可自行关闭。
:::

![image.png](./images/微信AI客服操作手册_1_2ece5dbd.png)

#### (1) 回复配置

1. 快件查询回复中可展示**预计时效**，默认关闭。
2. 网点可自行配置开启。
3. 开启后，会展示预计时效，即**快件跟踪内 末网点预计签收时间**。
4. **签收运单**不会显示预计时效。

![image.png](./images/微信AI客服操作手册_20_f8f9c59b.png)![image.png](./images/微信AI客服操作手册_21_0c6d863c.png)

::: danger 重点提醒
输入的运单号需属于**当前群绑定的机构**，否则不可查询。

例如：微信群绑定的是**北京网点**，单号 A 关联的寄件网点为**上海**、派件网点为**武汉**，则该单号在**北京网点**微信群内不可进行轨迹查询。
:::

### 5.2 超区查询

1. 在微信群内输入详细地址信息。
2. 机器人会进行**零担产品**的超区查询并回复结果。

![image.png](./images/微信AI客服操作手册_22_40bfaf1e.png)

::: tip 功能说明
**超区查询**为基础功能，默认开启。如客服群内不需要，可自行关闭。
:::

![image.png](./images/微信AI客服操作手册_23_1700a7d8.png)

#### (1) 回复字段配置

回复字段可由群对应的网点自行编辑。

![image.png](./images/微信AI客服操作手册_24_8be0fed6.png)

::: warning 注意事项
超区接口已做兜底查询：

1. 如收件地址为盲区，系统会查询附近**100km**内的网点信息并进行兜底回复。
2. 是否发货需由网点与客户自行协商。
3. 如附近**100km**内也无网点，则会提示盲区。
:::

![image.png](./images/微信AI客服操作手册_25_331338a7.png)

### 5.3 报价查询

1. 在微信群内输入关键字：**报价**。
2. 同时输入**收件地址信息**、**货物件数**、**重量体积**。
3. 机器人会进行报价查询并回复结果。

![image.png](./images/微信AI客服操作手册_26_b089c54d.png)

::: warning 注意事项
**报价查询**需关联寄件地址信息，默认关闭，需要网点自行配置表单信息。
:::

![image.png](./images/微信AI客服操作手册_27_e9f577d7.png)

#### (1) 报价查询回复配置

![image.png](./images/微信AI客服操作手册_28_5bc577aa.png)

::: warning 注意事项
1. 如收件地址为盲区，则会查询不到价格。
2. 报价如配置了**对客报价**，则优先取客户的报价相关信息；如未配置，则取网点统一的对外报价。
:::

### 5.4 下单

1. 在微信群内输入关键字：**下单**。
2. 同时输入**收件地址信息**、**收件人姓名/电话信息**、**货物件数**、**重量体积**。
3. 机器人会进行下单相关处理。

![image.png](./images/微信AI客服操作手册_29_227e3dc8.png)

::: warning 注意事项
**下单**需关联寄件地址信息，默认关闭，需要网点自行配置表单信息。
:::

## 七、存量微信群使用方法（省区网管/总部技术支持人员）

已经存在的客户微信群，需要先将微信群转为企业微信群，才能拉入**企业微信 - AI客服1号**。

1. 省区网管注册自己的企业微信。
2. 省区网管联系总部技术支持人员，将自己的企业微信账号变更为**企业管理员**。
3. 网点客户群将对应的省区网管微信账号拉入微信群内。
4. 网点将微信群主转交给省区网管。
5. 省区网管打开手机端企业微信，进行**转企业微信群**操作。

   ![image.png](./images/微信AI客服操作手册_4_94bdbe79.png)![image.png](./images/微信AI客服操作手册_5_288f109c.png)![image.png](./images/微信AI客服操作手册_6_a2310074.png)

6. 转为企业微信群后，拉入**AI客服1号**。
7. 由对应网点在群内输入**激活码**。
8. 后续操作参考前文步骤。
9. 将群主转移给原网点人员。

![image.png](./images/微信AI客服操作手册_7_204bf651.png)![image.png](./images/微信AI客服操作手册_8_3db9165b.png)

## 九、操作结果

完成后，应达到以下结果：

1. 群已成功绑定对应网点的**激活码**。
2. **AI客服1号**已在微信群或企业微信群内。
3. 群内可按配置使用**查件**、**超区查询**、**报价查询**、**下单**等功能。
4. 如开启相关配置，机器人会按配置内容进行自动回复。

## 十、注意事项

1. 页面展示、权限范围或业务规则如与本文不一致，请以当前系统配置和最新业务规则为准。
2. **机器人客服管理**权限在前期推广期间，需要先收集网点人员名单，由总部技术支持统一配置。
3. **查件**、**超区查询**为基础功能，默认开启；如客户群内不需要，可自行关闭。
4. **报价查询**、**下单**默认关闭，需网点自行配置表单信息。
5. 存量微信群需先转换为企业微信群，才能拉入**AI客服1号**。
6. 激活码与网点绑定，跨网点操作时需切换到对应网点后获取。

## 十一、常见问题

### 11.1 为什么输入单号后无法查询轨迹？

请确认该运单号是否属于**当前群绑定的机构**。如果运单关联的寄件网点、派件网点与当前群绑定机构不一致，可能无法查询。

### 11.2 为什么报价查询不到价格？

如收件地址为盲区，报价可能查询不到价格。

### 11.3 存量微信群可以直接拉入 AI客服1号 吗？

不可以。存量微信群需要先转为**企业微信群**，再拉入**企业微信 - AI客服1号**。
