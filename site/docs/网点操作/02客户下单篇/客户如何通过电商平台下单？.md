# 客户如何通过电商平台下单？

## 一、适用场景
客户在电商平台（如淘宝）下单时，需要使用中通冷链电子面单进行打单发货。

## 二、前置条件
- 商家已在电商平台开通店铺，且具备发货地址。
- 网点已购买对应渠道的电子面单（如菜鸟电子面单）。
- 商家已在中通冷链系统完成审核并充值电子面单余额。
- 商家已配置打单软件或ERP，并支持中通冷链面单。

## 三、操作入口
- **电商平台端**：交易 → 物流服务 → 电子面单
- **中通冷链系统（网点端）**：经营管理中心 → 商户电子面单账户管理
- **中通冷链系统（网点端）**：经营管理中心 → 电子面单申购
- **打单软件**：按软件实际菜单路径为准

## 四、操作步骤

1. **客户在电商平台申请中通冷链电子面单合作**
   进入平台 **交易（订单）→ 物流服务 → 电子面单** 板块，选择物流公司 **中通冷链**，点击 **申请合作**。

   ::: warning 注意事项
   需与商家约定发货地址。部分电商平台会根据发货地址过滤网点。例如，上海青浦网点地址为 **上海-上海-青浦区**，则商家的发货地址也必须为 **上海-上海-青浦区XX路XX号**，否则可能选不到对应网点。
   ![image.png](./images/大客户_个人下单操作说明_10_69e68144.png)
   ![image.png](./images/大客户_个人下单操作说明_11_d7788ae0.png)
   :::

2. **网点在中通冷链系统对商家审核并充值**
   - **商家审核**：
     进入 **经营管理中心 → 商户电子面单账户管理 → 审核**，点击 **审核** 即可。
     ![image.png](./images/大客户_个人下单操作说明_12_cf24529e.png)
   - **商家充值**：
     进入 **经营管理中心 → 商户电子面单账户管理 → 充值**（审核后才会显示【充值】按钮）。
     - 输入 **正整数**：为商家充值对应数量（网点账户余额需充足）。
     - 输入 **负整数**：扣减商家电子面单余额，扣减的数量返还至网点对应渠道的电子面单账户。最多可扣减至商家余额为0（例如商家有100个，则最多输入 -100）。
     ![image.png](./images/大客户_个人下单操作说明_13_26f8bc54.png)
     ![image.png](./images/大客户_个人下单操作说明_14_63a40793.png)
   - **网点购买对应渠道面单**（如还未购买）：
     进入 **经营管理中心 → 电子面单申购**。
     - 目前中通冷链已对接的平台：**淘宝（不含1688）**、~~抖音~~（暂不支持）、**快手**。
     - 若商家是淘宝商家，需购买 **菜鸟电子面单**。
     - 若此处没有对应渠道的电子面单，请联系省区财务上架该渠道面单（省区需先向总部申购后，网点才可申购）。
     ![image.png](./images/大客户_个人下单操作说明_15_eb053335.png)

3. **商家在打单软件中配置打印模板**
   - **订购打单软件**（如商家已有打单软件/ERP可跳过此步）。
     注意：并非所有打单软件都支持中通冷链，可先试用。主流支持的软件包括：旺店通、管家婆、快递管家、我打、快递助手等。若不支持，需联系软件服务商售前支持反馈，待其对接后方可使用。
     ![image.png](./images/大客户_个人下单操作说明_16_79f9e269.png)
     ![image.png](./images/大客户_个人下单操作说明_17_bac828f5.png)
   - **配置打印模板**：在面单模板维护入口，选择对应电商渠道，再选择 **中通冷链**，维护 **温区、产品类型、付款方式** 等信息。
     ![image.png](./images/大客户_个人下单操作说明_18_47a131cd.png)
     ![image.png](./images/大客户_个人下单操作说明_19_63d57e34.png)

4. **客户在打单软件取号并打印面单**
   选择对应的订单，再选择已配置的中通冷链电子面单模板，点击 **打印** 即可。
   ![image.png](./images/大客户_个人下单操作说明_1_05cb6fd5.png)

5. **订单推送至鲸天系统，网点及时接单**
   客户在电商平台下单后，订单会自动推送到鲸天系统的 **【订单管理】** 中。网点参照现有转运单流程操作即可。
   ![image.png](./images/大客户_个人下单操作说明_20_99648033.png)

::: tip 特殊说明
目前客户在电商平台下单时选择任意产品，均会转换为 **零担产品** 进入鲸天系统。
:::

## 五、操作结果
- 商家成功打印中通冷链电子面单，订单信息推送至鲸天系统。
- 网点在鲸天【订单管理】中看到该订单，可按正常转运单流程接单处理。

## 六、注意事项
- 网点充值前，需确保网点用户电子面单账户余额充足。
- 抖音渠道暂不支持电子面单。
- 若打单软件不支持中通冷链，需联系软件服务商对接后方可使用。
- 若客户自有系统且支持物流公司取号，可直接通过电子面单方式下单。
- 若客户有自研系统但不支持直接取号，可通过 **中通冷链开放平台** 接口对接下单（详情请联系侯鹏）。

## 七、常见问题
- **问：申请合作时选不到对应网点？**
  答：发货地址需与网点地址的省市县一致，否则可能过滤掉该网点。
- **问：打单软件里找不到中通冷链模板？**
  答：确认已订购支持中通冷链的打单软件，并在模板维护中选择对应电商渠道后刷新。
- **问：网点申购时没有对应渠道的面单？**
  答：需省区财务先上架该渠道面单，且省区向总部申购后，网点才能申购。
