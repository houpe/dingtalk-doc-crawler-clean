# 大客户个人下单

## 一、适用场景
面向中通冷链合作大客户，完成电子面单账户开通、申购、充值、绑定，支持线上单个下单、批量导入订单、面单打印、订单跟踪、物流查询等全流程操作，满足大客户大批量、常态化冷链寄件需求，提升下单与打单效率。

## 二、前置条件
1. **账号与权限**：需拥有鲸天系统、中通冷链客户服务平台的 **大客户操作权限** 和 **电子面单管理权限**，无权限请联系系统管理员开通。
2. **物理/环境准备**：
   - 电脑端需提前安装 **打印组件**，否则无法打印电子面单。
   - 配备常规打印机，支持电子面单打印。
   - 提前向线下网点获取 **客户编码** 和 **客户密钥**，用于绑定电子面单账户。
3. **配套工具/链接**：
   - 打印组件：页面内点击 **【下载组件】** 即可安装。
   - 鲸天系统登录入口：中通冷链后台系统（以实际地址为准）。
   - 客户服务平台入口：中通冷链官网客户端（以实际地址为准）。
   - 小程序入口：微信搜索「中通冷链」。

## 三、操作步骤

### 3.1 场景一：鲸天系统 新增&管理电子面单账户
**系统功能路径**：鲸天系统 → **经营管理中心** → **商户电子面单账户管理**

1. **绑定现有客户电子面单账户**
   进入页面后，根据商家ID/客户信息检索，选择对应客户，点击 **【绑定客户】**。
   ![image.png](./images/大客户_个人下单操作说明_10_69e68144.png)
   ::: tip 说明
   单个客户可绑定多个电子面单账户，客户手机号可与开户手机号不一致。
   :::
2. **绑定其他客户**
   输入商家ID、客户名称/手机号查询客户信息，选中目标客户完成绑定。
   ![image.png](./images/大客户_个人下单操作说明_11_d7788ae0.png)
3. **创建新客户&电子面单账户**
   选择 **【创建新客户】**，填写客户基础资料，系统将同步生成全新电子面单账户。
   ![image.png](./images/大客户_个人下单操作说明_12_cf24529e.png)

### 3.2 场景二：鲸天系统 申购客户主单
**系统功能入口**：鲸天系统 → **经营管理中心** → **电子面单申购**

::: danger 重点提醒
**必须先申购客户主单，才能为客户电子面单账户充值**。
:::

1. 进入电子面单申购页面，选择 **客户主单**。
   ![image.png](./images/大客户_个人下单操作说明_13_26f8bc54.png)

### 3.3 场景三：鲸天系统 电子面单账户充值
**系统功能入口**：鲸天系统 → **经营管理中心** → **商户电子面单账户**

1. 检索目标客户账户，点击操作栏 **【充值】**。
2. 填写充值数量，核对信息后点击 **【保存】** 完成充值。
   ::: tip 说明
   订购来源为中通冷链时，充值额度均来源于客户主单。
   :::
   ![image.png](./images/大客户_个人下单操作说明_14_63a40793.png)

### 3.4 场景四：多端绑定电子面单账户
**前置条件**：从线下网点获取 **客户编码** 和 **客户密钥**，未绑定账户无法正常下单。

#### (1) 客户服务平台绑定
1. 登录客户服务平台，进入 **【基础配置】** → **【电子面单账户管理】**。
2. 点击 **【绑定电子面单】**，填写编码与密钥，提交绑定。
   ![image.png](./images/大客户_个人下单操作说明_15_eb053335.png)

#### (2) 微信小程序绑定
1. 微信搜索进入「中通冷链」小程序，进入 **【电子面单账户管理】**。
2. 选择 **【绑定电子面单账户】**，输入客户编码、密码，确认绑定。
   ![image.png](./images/大客户_个人下单操作说明_16_79f9e269.png)
   ![image.png](./images/大客户_个人下单操作说明_17_bac828f5.png)

### 3.5 场景五：客户服务平台 大客户下单操作
**系统功能入口**：中通冷链客户服务平台 → **首页** → **大客户下单**

模块包括：**新增下单**、**批量导入**、**待处理**、**已处理**、**全部订单**。

#### (1) 单个订单下单
1. **填写寄收件人信息**
   - 方式1：从 **地址簿** 选择已有地址，系统自动填充。
   - 方式2：粘贴完整地址，系统智能识别姓名、电话、省市区。
   - 方式3：手动录入全部信息，可勾选“保存到地址簿”方便下次使用。
   - 支持设置 **默认寄件地址**，下单自动填充。
   ![image.png](./images/大客户_个人下单操作说明_18_47a131cd.png)
   ![image.png](./images/大客户_个人下单操作说明_19_63d57e34.png)

2. **填写托寄物信息**
   - 选择物品类型，也可自定义新增品类。
   - 选择温区：**冷藏（0-10℃）** / **冷冻**，选择是否为进口货物。
   - 填写货物件数、总重量、总体积，选填收派员留言。
   ![image.png](./images/大客户_个人下单操作说明_1_05cb6fd5.png)

3. **选择产品与增值服务**
   - 选择产品类型、寄派方式、派送方式。
   - 按需选择保价、增值服务。
   - 选择付款方式（默认寄付月结，支持寄付现结、到付），可选优惠券。
   ![image.png](./images/大客户_个人下单操作说明_20_99648033.png)

4. **费用查看 & 提交订单**
   - 页面查看预估费用及明细。
   - 勾选同意《电子运单契约条款》，点击 **【保存】** 完成下单。
   - 可选择 **【保存草稿】**，草稿可在草稿箱修改后再提交。
   - 下单成功后，可选择再寄一件、查看订单详情。
   ![image.png](./images/大客户_个人下单操作说明_21_3f9194c1.png)
   ![image.png](./images/大客户_个人下单操作说明_22_0e513aa8.png)

#### (2) 批量导入订单（多单快速下单）
1. 进入 **【批量导入】** 模块，点击 **【下载模板】**，获取标准Excel表格。
2. 按照模板要求填写订单信息，**单次最多导入500单**。
3. 点击上传文件或拖拽文件至上传区域，点击 **【开始导入】**。
4. 导入完成后查看结果：失败可下载失败列表排查问题，成功订单进入 **【待处理】** 列表。
   ![image.png](./images/大客户_个人下单操作说明_23_3fb94ccb.png)
   ![image.png](./images/大客户_个人下单操作说明_24_f122bc12.png)
   ![image.png](./images/大客户_个人下单操作说明_25_ea7b2919.png)

#### (3) 待处理订单操作
批量导入/单个下单保存后的订单统一进入 **【待处理】** 列表：
1. **仅下单**：选中订单，选择可用电子面单账户，完成下单（后续到已处理列表打印）。
2. **下单并打印**：选中订单，选择电子面单账户、面单模板、打印机，直接完成下单+打印。
3. 若提示无可用余额，需前往鲸天系统为账户充值；提示未检测打印组件，需下载并安装组件。
   ![image.png](./images/大客户_个人下单操作说明_26_9d56125d.png)
   ![image.png](./images/大客户_个人下单操作说明_27_e3214a99.png)
   ![image.png](./images/大客户_个人下单操作说明_28_c588dd60.png)

#### (4) 已处理订单操作
1. 进入 **【已处理】** 列表，可查看所有已下单订单的打印状态、运单号、费用等信息。
2. 针对 **打印失败** 的订单，选中后重新执行打印操作，流程同“下单并打印”。
3. 支持订单导出、撤销操作。
   ![image.png](./images/大客户_个人下单操作说明_29_b2e73e5e.png)

#### (5) 全部订单 & 查询功能
1. 点击 **【全部订单】**，跳转至 **【物流跟踪-订单查询】**。
2. 可查看 **订单基础信息、实时物流轨迹、回单状态、打印日志**。
3. 可依据运单号、时间、收件人等条件筛选订单。

## 四、操作结果
- 绑定电子面单账户后，账户状态显示“已绑定”，可在对应端正常下单。
- 申购客户主单后，等待审核通过方可充值；充值后账户余额增加，即可用于下单打印。
- 下单成功或批量导入成功后，订单进入待处理/已处理列表，可打印面单、跟踪物流。
- 微信小程序下单成功后，可在查件页面查看实时物流。

## 五、注意事项
1. **申购主单需审核**：电子面单申购存在审核流程，需等待状态变为「审核通过」后，再执行充值操作。
2. **打印组件必须安装**：若执行“下单并打印”时提示未检测到打印组件，请点击页面 **【下载组件】** 安装并启动，刷新页面重试；仍异常则重启电脑。
3. **批量导入限制**：单次批量导入最多支持500条订单，超出数量请分多次上传。
4. **客户编码/密钥**：必须从线下网点获取，丢失后联系网点工作人员查询。
5. **小程序与PC端数据互通**：两端均可查询订单、物流轨迹与打印记录。

## 六、常见问题

### 6.1 Q1：为什么申购完主单，还是无法给客户账户充值？
A：电子面单申购存在审核流程，需等待状态变为「审核通过」后，再执行充值操作。

### 6.2 Q2：小程序和电脑端的订单数据是否互通？
A：数据完全互通，两端均可查询订单、物流轨迹与打印记录。

### 6.3 Q3：批量导入订单有数量限制吗？
A：单次批量导入最多支持500条订单，超出数量请分多次上传。

### 6.4 Q4：面单打印失败后，如何重新打印？
A：进入 **【已处理】** 订单列表，选中打印失败的订单，重新选择模板和打印机再次打印即可。

### 6.5 Q5：忘记客户编码和密钥该如何处理？
A：直接联系为您开通电子面单的线下网点工作人员查询。

### 6.6 Q6：客户可自主下单的方式有哪些？
A：中通冷链（小程序）、客户服务平台（PC端）、扫码下单（小程序）。
