# 扫码下单

## 一、适用场景

本文适用于网点员工、业务人员、合作客户使用**员工专属下单二维码**完成扫码下单、订单来源查询及异常处理。

日常工作中，客户委托网点代下单、反复查询物流轨迹的情况较多，会增加网点工作量。使用**员工专属下单二维码**后，可引导客户自主扫码下单，订单自动指派至对应网点；客户也可自行在小程序查询物流信息，减少网点人工处理压力，同时支持线上引流拓展客户。

## 二、前置条件

### 2.1 账号与权限

1. 员工需登录**鲸小宝APP**。
2. 仅**网点直属正式员工**可使用专属下单二维码功能，非授权人员无对应权限。
3. 后台查看订单 **BD字段**，需具备**订单管理**、**运单管理**权限；无权限请联系系统管理员。

### 2.2 设备与环境

1. 手机正常联网，并已安装**鲸小宝APP**、**微信**。
2. 如需纸质推广，需准备打印机用于打印二维码。
3. 客户端仅需安装**微信**，无需额外下载APP。

### 2.3 配套工具

- 🛠️ **鲸小宝APP**：网点内部客户端
- 🌐 **中通冷链小程序**：微信端直接搜索进入
- **订单/运单管理后台**：中通冷链运营管理系统

## 三、操作入口

### 3.1 员工获取专属下单二维码

**系统路径**：登录**鲸小宝APP** → **我的** → **我的二维码**

**快捷入口**：鲸小宝底部导航栏【**我的**】页面

### 3.2 客户扫码下单

**系统路径**：微信扫码 → 自动跳转**中通冷链小程序**

### 3.3 后台查询订单来源

- **订单管理查询路径**：**运营后台** → **订单管理** → **订单详情**
- **运单管理查询路径**：**运营后台** → **寄件运单管理** → **运单详情**

## 四、操作步骤

### 4.1 场景一：员工获取并分享专属下单二维码

#### (1) 打印纸质二维码分享

1. 登录**鲸小宝APP**，进入【**我的**】页面。
2. 点击【**我的二维码**】，进入二维码页面。
3. 点击【**打印二维码**】。
4. 连接打印机完成打印。
5. 将纸质二维码张贴至客户门店、合作点位，供客户扫码下单。

📷 ![image.png](./images/大客户_个人下单操作说明_10_69e68144.png)

![image.png](./images/大客户_个人下单操作说明_11_d7788ae0.png)

#### (2) 下载二维码图片单独分享

1. 在【**我的二维码**】页面，点击【**下载二维码**】。
2. 二维码图片保存至手机本地。
3. 通过微信或其他聊天软件，将二维码图片单独发送给对应客户。

![lADPJwKt7CJFlzzNA4TNAo4_654_900.jpeg](./images/大客户_个人下单操作说明_12_cf24529e.png)

#### (3) 分享至微信朋友圈引流

1. 在二维码页面点击【**分享二维码**】。
2. 选择分享到**微信朋友圈**。
3. 补充文案后发布，用于线上引流拓展客户。

::: warning 注意事项
苹果 **IOS系统**暂不支持直接分享操作，可选择**下载图片**或**打印二维码**的方式替代。
:::

![617f69c6fb46c1ac033a10f398564740.jpeg](./images/大客户_个人下单操作说明_13_26f8bc54.png)

### 4.2 场景二：客户扫码完成自主下单

#### (1) 基础下单操作

1. 客户使用微信扫描员工专属二维码。
2. 系统自动进入专属下单页面，页面默认展示**绑定网点**、**联系电话**。
3. 客户依次填写**寄件人信息**、**收件人信息**、**物品信息**。
4. 如弹出“**揽收范围与寄件地址不匹配**”提示，根据实际情况选择【**继续下单**】或【**按寄件地址下单**】。
5. 阅读并勾选同意《**电子运单契约条款**》。
6. 核对**预估运费**后提交订单。

![676a1afb25d50102defdb7b292a8f95c.jpeg](./images/大客户_个人下单操作说明_14_63a40793.png)

#### (2) 订单物流查询

1. 下单完成后，客户打开**中通冷链小程序**。
2. 进入【**我的寄件**】。
3. 查看订单配送状态和物流轨迹。

::: tip 补充说明
**扫码下单**：自动绑定指定网点与对应员工，订单定向指派，报价为小程序统一报价。

**小程序直接下单**：无固定绑定网点，系统自动匹配就近网点。
:::

### 4.3 场景三：后台查询订单来源

#### (1) 在订单管理页面查询

1. 进入**运营后台** → **订单管理**。
2. 打开目标订单的**订单详情**页。
3. 查找页面内 **BD字段**，内容格式为「**员工编号/员工姓名**」。
4. 根据 **BD字段** 判断扫码下单来源员工。

**状态说明：**

- 员工正常在岗：仅展示**编号+姓名**。
- 员工已离职：标注「**（已离职）**」。
- 员工更换所属组织：标注「**（已更换组织）**」。

![](./images/大客户_个人下单操作说明_15_eb053335.png)

![](./images/大客户_个人下单操作说明_16_79f9e269.png)

#### (2) 在运单管理页面查询

1. 进入**运营后台** → **寄件运单管理**。
2. 打开目标运单的**运单详情**。
3. 在运单基本信息区域查看 **BD字段**，确认扫码下单对应员工。

![](./images/大客户_个人下单操作说明_17_bac828f5.png)

## 五、操作结果

完成以上操作后，可达到以下结果：

1. 员工可通过**打印二维码**、**下载二维码**或**分享二维码**的方式，将专属下单二维码提供给客户。
2. 客户扫码后可进入**中通冷链小程序**完成自主下单。
3. 扫码下单的订单会自动关联对应员工与所属网点。
4. 后台可在订单或运单详情中通过 **BD字段** 查询订单来源员工。
5. 客户可在**中通冷链小程序**【**我的寄件**】中查看订单和物流轨迹。

## 六、注意事项

::: danger 重点提醒
**专属下单二维码**仅限**网点直属正式员工**使用。外部授权人员、合作方无此功能。
:::

::: warning 注意事项
1. **专属下单二维码**为长期有效，下载、打印后可重复使用。
2. **扫码下单**目前统一使用小程序前端标准报价。
3. 如客户扫码后提示“**揽收范围与寄件地址不匹配**”，可引导客户选择【**按寄件地址下单**】，系统重新匹配就近网点；或确认后选择【**继续下单**】。
4. 如后台看不到 **BD字段**，可先刷新页面；仍无法查看时，联系管理员确认是否具备订单/运单详情查看权限。
5. 苹果手机无法直接分享二维码时，改用**下载二维码图片**或**打印纸质二维码**方式发送给客户。
:::

## 七、常见问题

### 7.1 常见异常与处理方式

| 序号 | ❌ 异常现象 / 报错提示 | 🔍 常见原因 | 🛠️ 解决方案 |
|------|-------------------------------|-----------------|--------------------|
| 1 | 鲸小宝无“我的二维码”功能入口 | 账号为非网点直属员工，权限不足 | 联系网点管理员开通对应权限，非授权人员无法使用该功能 |
| 2 | 苹果手机无法直接分享二维码 | IOS系统限制，不支持APP内一键分享 | 改用**下载二维码图片**或**打印纸质二维码**的方式发给客户 |
| 3 | 客户扫码后提示揽收地址不匹配 | 寄件地址超出绑定网点常规揽收范围 | 引导客户选择「按寄件地址下单」，系统重新匹配就近网点；或确认后继续下单 |
| 4 | 订单/运单页面找不到BD字段 | 后台账号权限不足，或页面未加载完成 | 1. 刷新页面重试；2. 联系管理员开通订单/运单详情查看权限 |
| 5 | 客户扫码无法跳转小程序 | 微信缓存异常、网络中断、二维码失效 | 1. 检查手机网络；2. 客户重启微信重新扫码；3. 员工重新下载二维码使用 |

### 7.2 FAQ

**Q1：专属下单二维码是否有使用有效期？**

A：二维码为长期有效，下载、打印后可重复使用。

**Q2：扫码下单的价格可以单独定制特殊报价吗？**

A：目前扫码下单统一使用小程序前端标准报价，个性化专属报价功能后续会上线，现阶段可提前和客户沟通价格。

**Q3：哪些人员可以生成专属下单二维码？**

A：仅网点机构下**所有直属正式员工**拥有该权限，外部授权人员、合作方无此功能。

**Q4：员工离职/调岗后，历史订单的BD字段会如何显示？**

A：员工离职会标注「**（已离职）**」，更换所属组织会标注「**（已更换组织）**」，依然可追溯原始下单来源。

**Q5：客户下单后，如何自己查看物流？**

A：客户打开**中通冷链微信小程序**，进入【**我的寄件**】，即可查看全部订单与实时配送轨迹。
