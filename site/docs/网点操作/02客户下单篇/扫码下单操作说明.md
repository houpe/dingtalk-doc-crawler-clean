# 扫码下单

## 一、适用场景

**为什么用？** 日常工作中客户委托网点代下单、反复查询物流轨迹，增加网点工作量。使用 **员工专属下单二维码**，可引导客户自主扫码下单，订单自动指派至对应网点，客户自行在小程序查询物流信息，减轻网点人力压力，同时支持线上引流拓展客户。

## 二、前置条件

- **账号与权限**：需登录 **鲸小宝APP**，仅**网点直属正式员工**可使用二维码功能；后台查询订单/运单BD字段需对应管理权限，无权限请联系系统管理员。
- **物理/环境准备**：
  1. 手机正常联网，安装 **鲸小宝APP**、**微信**。
  2. 如需纸质推广，请准备打印机。
  3. 客户端仅需安装微信，无需额外下载APP。
- **配套工具**：
  - 🛠️ 鲸小宝APP（网点内部客户端）
  - 🌐 中通冷链微信小程序
  - 运营后台（订单/运单管理）

## 三、操作入口

- **员工获取二维码**：鲸小宝APP → **我的** → **我的二维码**
- **客户扫码下单**：微信扫码 → 自动跳转中通冷链小程序
- **后台查询订单来源**：运营后台 → **订单管理** / **寄件运单管理** → 订单/运单详情

## 四、操作步骤

### 4.1 员工获取并分享专属下单二维码

1. 登录鲸小宝APP，进入 **我的二维码** 页面。
2. 根据需要选择分享方式：
   - **打印纸质二维码**：点击【打印二维码】，连接打印机打印，张贴至客户门店或合作点位。
   - **下载二维码图片**：点击【下载二维码】，图片保存至手机本地，单独发送给客户。
   - **分享至微信朋友圈**：点击【分享二维码】，选择分享到微信朋友圈，补充文案发布。

::: warning 注意事项
苹果IOS系统暂不支持直接分享操作，请使用**下载图片**或**打印二维码**替代。
:::

### 4.2 客户扫码完成自主下单

1. 客户使用微信扫描员工专属二维码，自动进入专属下单页面，页面默认展示绑定网点和联系电话。
2. 依次填写 **寄件人信息**、**收件人信息**、**物品信息**。
3. 若弹出“揽收范围与寄件地址不匹配”提示，按需选择 **【继续下单】** 或 **【按寄件地址下单】**。
4. 阅读并勾选同意《电子运单契约条款》，核对预估运费后提交订单。
5. 完成后客户进入中通冷链小程序 **【我的寄件】**，可实时跟踪订单物流轨迹。

### 4.3 后台查询订单来源（扫码对应员工）

**场景3.1 订单管理页面查询**

1. 进入运营后台 → **订单管理**，打开目标订单详情页。
2. 查找 **BD字段**（格式：员工编号/员工姓名）。
3. 状态说明：
   - 员工正常在岗：仅展示编号+姓名。
   - 员工已离职：标注「（已离职）」。
   - 员工更换所属组织：标注「（已更换组织）」。

**场景3.2 运单管理页面查询**

1. 进入运营后台 → **寄件运单管理**，打开目标运单详情。
2. 在运单基本信息区域，查看 **BD字段**，即可确认扫码下单的对应员工。

## 五、操作结果

- 员工成功生成专属二维码，可打印、下载或分享至朋友圈。
- 客户扫码后自动进入专属下单页，提交订单后订单归属该员工及对应网点；客户可在小程序 **【我的寄件】** 查看物流。
- 后台订单/运单详情中的 **BD字段** 会显示下单员工的编号和姓名，员工异动时标注状态。

## 六、注意事项

- 仅**网点直属正式员工**有生成二维码的权限，外部授权人员无此功能。
- 苹果IOS系统暂不支持APP内一键分享，请使用下载图片或打印纸质二维码方式。
- 客户扫码后若提示“揽收地址不匹配”，可选【继续下单】或【按寄件地址下单】。
- 扫码下单价格为小程序统一报价，暂不支持个性化专属报价，需提前与客户沟通。
- 员工离职或调岗后，历史订单BD字段会标注「已离职」或「已更换组织」，可追溯原始来源。
- 后台查询BD字段若找不到，请刷新页面或确认账号是否有订单/运单详情查看权限。

## 七、常见问题

**Q1：专属下单二维码是否有使用有效期？**
A：二维码长期有效，下载、打印后可重复使用。

**Q2：扫码下单的价格可以单独定制特殊报价吗？**
A：目前扫码下单统一使用小程序前端标准报价，个性化专属报价功能后续上线。

**Q3：哪些人员可以生成专属下单二维码？**
A：仅网点机构下**所有直属正式员工**拥有权限，外部授权人员无此功能。

**Q4：员工离职/调岗后，历史订单的BD字段会如何显示？**
A：离职标注「（已离职）」，更换组织标注「（已更换组织）」，可追溯原始来源。

**Q5：客户下单后如何自己查看物流？**
A：打开中通冷链微信小程序，进入【我的寄件】即可查看全部订单与实时配送轨迹。

**常见异常与原因/解决方案**

| 异常现象 | 常见原因 | 解决方案 |
|-------------------------------|-----------------|--------------------|
| 鲸小宝无“我的二维码”功能入口 | 账号非网点直属员工，权限不足 | 联系网点管理员开通权限 |
| 苹果手机无法直接分享二维码 | IOS系统限制 | 改用下载图片或打印纸质二维码 |
| 客户扫码后提示揽收地址不匹配 | 寄件地址超出网点揽收范围 | 选择「按寄件地址下单」或确认后继续下单 |
| 订单/运单页面找不到BD字段 | 后台权限不足或页面未加载 | 刷新页面；联系管理员开通查看权限 |
| 客户扫码无法跳转小程序 | 微信缓存异常、网络中断、二维码失效 | 检查网络；重启微信重新扫码；员工重新下载二维码 |
