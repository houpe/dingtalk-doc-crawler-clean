# 一. 维保工单

## 1.1 业务场景与名词解释

### 1.1.1 业务场景

本手册适用于中通冷链全体司机，依托<strong>中通冷链司机版微信小程序</strong>完成车辆定期保养、故障维修全流程线上工单管理。车辆到达保养周期或出现故障时，司机线上发起<strong>维保前申请</strong>，经后勤专员审批后到指定站点维保；维保结束后线上提交<strong>维保后申请</strong>并补充凭证，完成工单闭环，实现车辆维保记录、费用、配件全流程线上留痕、统一管控。

### 1.1.2 核心名词解释

- <strong>维保申请</strong>：小程序小工具模块内的工单功能，分为<strong>维保前申请</strong>和<strong>维保后申请</strong>两个环节。
- <strong>保内保养</strong>：车辆在质保周期内的常规定期保养，为日常主流保养类型。
- <strong>维保前申请</strong>：车辆进场维保/维修前，线上填报车辆信息、里程、故障、预估费用并提交审批的前置工单。
- <strong>维保后申请</strong>：车辆完成维保/维修后，补充实际维修项目、配件、费用、票据及现场照片的收尾工单。
- <strong>维保审批</strong>：由车辆后勤专员对前后两阶段工单进行审核，审批通过后方视为工单办结。
- <strong>里程照片</strong>：车辆仪表盘里程实拍图，作为里程登记凭证。
- <strong>维修前/后图片</strong>：车辆故障部位、维保现场、维修票据、配件清单等佐证照片。

## 1.2 前置准备与环境配置

1. <strong>账号与权限</strong> 使用个人微信登录<strong>中通冷链司机版</strong>小程序，司机账号默认开通维保申请权限；找不到入口请联系网点管理员核查。
2. <strong>设备与权限要求</strong> 使用智能手机，需开启小程序<strong>相机、相册</strong>权限，用于拍摄并上传里程、车辆、票据照片。
3. <strong>使用提醒</strong> 优先选择运输空档期安排维保，<strong>禁止在运输任务途中进行车辆保养维修</strong>。
4. <strong>配套入口</strong> 微信搜索并进入【中通冷链司机版】小程序。

## 1.3 场景化标准操作步骤

### 1.3.1 场景一：维保前申请（维保进场前置流程）

- 系统功能路径：中通冷链司机版小程序 → 首页 → 小工具 → 维保申请

![image.png](./images/维保工单操作说明书_1_65ebb6fc.png)

1. 确认车辆已达到保养周期或存在故障，合理规划维保时间，避开运输作业时段。
2. 打开小程序，在首页找到<strong>小工具</strong>板块，点击进入<strong>维保申请</strong>功能。
3. 按页面要求逐项填写信息：

- 车牌号码：填写当前运营车辆号牌；
- 当前里程：如实填写车辆仪表盘行驶里程；
- 保养类型：常规保养选择<strong>保内</strong>；
- 维保时间、维修项目、故障说明：详细填写保养项目或车辆故障问题；
- 维修耗时、预估维修金额：填写预估时长与费用；
- 保养公司、当前地点：系统自动带出，核对信息即可。

4. 上传凭证图片：按要求上传<strong>里程照片、维修前图片</strong>，两类图片均为必填，单次可上传1-5张。
5. 所有信息及照片核对无误后，点击<strong>提交</strong>，等待车辆后勤专员审批。
6. 待工单状态显示<strong>维保前审核通过</strong>，即可前往指定站点开展维保作业。

### 1.3.2 场景二：维保后申请（维保完成收尾流程）

- 系统功能路径：中通冷链司机版小程序 → 首页 → 小工具 → 维保申请

![image.png](./images/维保工单操作说明书_2_e1d68e65.png)

1. 车辆完成保养/维修后，再次进入<strong>维保申请</strong>页面。
2. 在工单列表中找到已通过的维保前工单，点击<strong>维保后申请</strong>进入编辑页面。
3. 补充完整实际维保信息：

- 维修项目清单：选择或填写实际维修、更换的配件名称及数量；
- 维修工时、维修实际金额：如实填写实际耗时与结算费用；
- 完善故障说明、当前地点等内容。

4. 上传凭证图片：拍摄并上传<strong>维修现场、维修发票、配件清单</strong>等维修后照片（必填1-5张）。
5. 信息、图片全部确认完毕，点击<strong>提交</strong>。
6. 等待车辆后勤专员二次审批，页面显示<strong>维保后审核通过</strong>，整份维保工单办结。

## 1.4 常见异常与兜底方案

| 序号 | 异常现象 / 报错提示 | 常见原因 | 解决方案 |
|------|---------------------------|------------|------------|
| 1 | 小程序找不到【维保申请】入口 | 账号权限未开通、小程序缓存异常 | 1. 联系网点管理员开通对应权限；2. 关闭小程序后台，重新进入刷新页面。 |
| 2 | 无法拍摄、上传照片 | 未开启小程序相机/相册权限、图片过大 | 1. 进入小程序设置，开启相机与相册权限；2. 重新拍摄清晰照片，压缩大小后再上传。 |
| 3 | 维保工单提交失败 | 网络中断、带\*必填项未填写完整 | 1. 切换稳定网络重新提交；2. 检查页面必填字段、图片是否齐全，补充完整后再次提交。 |
| 4 | 保养公司/当前地点信息错误 | 系统定位异常、基础信息有误 | 手动核对修正，或联系后勤专员更新车辆绑定站点信息。 |
| 5 | 提交后长时间未收到审批结果 | 后勤专员未处理、工单状态延迟 | 耐心等待，必要时联系车辆后勤专员提醒审核。 |
| 6 | 找不到历史维保前工单 | 筛选条件错误、工单已办结 | 在维保申请列表调整筛选条件，查看全部工单记录。 |

## 1.5 高频常见问题（FAQ）

### 1.5.1 Q1：车辆维保可以在运输任务途中进行吗？

A：不建议，需提前规划时间，在无运输任务的空档期安排保养维修，避免影响正常运输作业。

### 1.5.2 Q2：保养类型默认选择“保内”就可以吗？

A：常规周期保养统一选择<strong>保内</strong>；车辆故障维修、出保后保养根据实际情况选择对应类型。

### 1.5.3 Q3：里程照片、维修前后照片可以不上传吗？

A：不可以，里程照、维修前后照片均为必填凭证，缺少资料将无法提交工单，也会导致审批不通过。

### 1.5.4 Q4：维保前申请提交后，多久可以进场维保？

A：等待后勤专员审批通过，工单状态显示「维保前审核通过」后，再前往指定服务站维保。

### 1.5.5 Q5：维保完成后，忘记提交维保后申请有什么影响？

A：工单无法闭环，维保记录、费用、配件信息无法归档，会影响车辆台账及后续维保判定，维保完成后务必及时提交维保后申请。

### 1.5.6 Q6：预估金额和实际维修金额不一致怎么办？

A：维保后申请环节按照实际结算金额如实填写即可，后勤专员会根据票据及实际费用审核。

### 1.5.7 Q7：车辆出现故障，和常规保养操作流程一样吗？

A：流程完全一致，均需要先发起维保前申请，审批通过后维修，完工再提交维保后申请。
