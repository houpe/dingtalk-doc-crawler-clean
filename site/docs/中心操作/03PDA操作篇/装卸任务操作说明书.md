# 装卸任务

## 适用场景

本文适用于中通冷链分拨中心现场管理人员、装卸人员、扫描员、叉车工等岗位。

现场人员可通过 **中通冷链司机版微信小程序** 与 **PDA手持设备**，完成车辆到港后的 **卸车、装车** 标准化作业，包括任务查看、人员打卡、封签核验、测温拍照、扫码分拣、现场验收等环节，保障冷链货物温区合规、作业安全、数据可追溯。

### 核心名词解释

- **卸车任务**：运输车辆抵达分拨月台后，按流程完成单据交接、测温、分拣、入库的作业任务。
- **装车任务**：根据调度计划，完成货物集托、扫码、装车、封签、月台交接的出库作业任务。
- **封签**：用于锁定车厢、保障货物运输安全的铅封，卸车前必须核验封签完整性。
- **冷链测温标准**：冷藏货物温度标准 **2℃~8℃**，冷冻货物温度标准 **≤-15℃**。
- **集托**：将同流向、同温区货物码放至统一托盘，便于批量扫描、转运与装卸。
- **问题件**：作业中发现的破损件、有单无货、有货无单、无头件等异常货物。
- **劳务打卡**：装卸人员通过小程序完成上班打卡、单任务打卡，记录作业工时。
- **月台**：分拨中心车辆停靠、装卸货的专用作业区域。

## 前置条件

1. **账号与权限**  
   所有现场作业人员使用个人微信登录 **中通冷链司机版** 小程序；PDA设备使用专属工号登录。无账号或无操作权限时，请联系分拨中心管理员配置。

2. **设备与工具**  
   准备智能手机（用于小程序打卡、任务查看、拍照测温）、PDA手持终端、测温枪、三角木、托盘、封签、拍照设备、月台二维码。

3. **网络要求**  
   作业区域需保证无线网络/移动信号稳定，PDA与小程序需联网同步数据。

4. **配套入口**  
   - 微信小程序：微信搜索进入 **中通冷链司机版**。
   - 现场终端：使用分拨专用 **PDA设备**。

## 操作入口

- 小程序路径：**微信 → 中通冷链司机版小程序 → 我的 → 登录 → 首页 → 任务管理**
- 打卡入口：**中通冷链司机版小程序 → 首页 → 打卡**
- PDA入口：请使用分拨专用PDA设备，以现场实际菜单为准。

## 操作步骤

### 通用前置操作：查看任务信息与人员上班打卡

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bJa9dbklxo/img/f453667c-0d6a-4afc-be28-bd52bf310f59.png?Expires=1783781685&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=KsDiJ3l16npfzHWGbKw0JuEhV2U%3D "")

1. 微信搜索 **中通冷链司机版**，点击右下角 **【我的】→【去登录】**，完成账号登录。
2. 回到小程序首页，点击 **【任务管理】**。
3. 查看 **待处理、进行中、已完成** 的装卸任务。
4. 进入任务详情，核对车辆信息、预计到达时间、货量、温层、票件数、规划作业时长、对应月台。
5. 装卸人员提前完成上班打卡：在首页点击 **【打卡】**，扫描分拨中心二维码，完成上班签到。

### 卸车全流程操作

完整流程：确认卸车信息 → 单据交接 → 引导车辆靠台 → 拍照测温 → 劳务任务打卡 → 现场卸车 → 清扫车厢 → 关闭月台门 → 卸车任务验收

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bJa9dbklxo/img/8c2af6b8-488d-4f39-995e-5ba5c48d4462.png?Expires=1783781685&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=NMQwqKnChIDGoaCru3Pi%2F4k8L9E%3D "")

1. **确认卸车信息**
   - 在 **【任务管理】** 中筛选 **卸车** 类任务，按车辆预计到达时间排序查看。
   - 查看货量明细、流向、冷冻/冷藏温区、重量、体积、票件数量，提前规划月台与人力。
   - 时间标红代表车辆晚于规划时间到达，需重点跟进超时车辆。

2. **单据交接**
   - 车辆抵达分拨中心后，任务状态显示 **待靠台**。
   - 点击任务内车牌号，获取司机联系方式。
   - 与司机完成纸质/电子单据核对交接。

3. **引导车辆靠台**
   - 现场人员引导车辆驶入指定月台，并规范停靠。
   - 司机在小程序提交 **靠台** 操作。
   - 页面状态更新为 **已靠台待卸车** 后，现场方可开展作业。

4. **拍照测温（冷链必做）**
   - 进入卸车编辑页面，填写 **封签号、月台号**。
   - 选择对应装卸工、扫描工、叉车工。
   - 进行 **封签拍照**：拍摄车辆尾部完整封签照片，确认封签完好无破损、无撬动痕迹。
   - 车轮前后垫放 **三角木**，做好安全防护后，再打开车厢门。
   - 进行 **车厢拍照**：拍摄车厢内货物全景照片留存。
   - 进行 **货物测温**：将车门/隔温板开启缝隙，使用测温枪检测货物表面温度。
     - 冷藏标准：**2℃~8℃**
     - 冷冻标准：**≤-15℃**
   - 将测温结果、现场照片依次上传至小程序。

::: danger 重点提醒
温度达标后方可正常卸车；温度不达标时，需等待温度恢复标准后再作业。
:::

5. **劳务任务打卡**
   - 装卸人员在小程序首页点击 **【打卡】**。
   - 扫描当前 **月台二维码**。
   - 点击 **【开始装卸车】**，页面提示打卡成功。

::: danger 重点提醒
劳务任务打卡仅限本任务分配人员操作。
:::

6. **现场卸车（PDA操作+现场作业）**
   - 打开PDA设备，进入 **【卸车任务】**。
   - 选择对应月台任务，点击 **【集托】** 开始作业。
   - 按要求规范码托：
     - 严禁不同温区货物混放。
     - 遵循 **重不压轻、大不压小** 原则。
     - 禁止踩踏货物。
     - 所有货物 **100%上托盘**，不得直接接触地面。
     - 同票货物同位摆放。
   - 使用PDA对所有货物 **100%逐票扫描**。
   - 带托货物使用临时托盘转运。
   - 发现破损件、有单无货、有货无单、无头件时，立即在PDA **【问题件登记】** 上报，并现场处理。
   - 扫描完成后，根据流向将货物转运至对应库区。
   - 扫描库区二维码完成入库。

::: danger 重点提醒
货物穿堂停留时长不得超过 **30分钟**，严禁单票多库区混放。
:::

7. **清扫车厢**  
   整车货物完全卸出后，及时清理车厢内杂物、垃圾、残留货物，保证车厢整洁。

8. **关闭月台门**  
   车厢清理完毕、车辆准备离场后，现场关闭月台门。

9. **卸车任务验收**
   - 在小程序/PDA查看卸车数据，重点核对以下四类状态：
     - **完成卸车并入库**
     - **完成卸车未入库**
     - **未卸出货物**
     - **运单差异**
   - 核对实际货量、票数、件数与系统数据。
   - 如存在差异，及时登记上报。
   - 全部核对无误后，确认任务完成，系统更新为 **已完成**。

### 装车全流程操作

完整流程：接收装车任务 → 核对装车单据 → 货物集托分拣 → 劳务任务打卡 → 现场装车 → 封签核验 → 拍照留存 → 车辆离场 → 装车任务验收

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/r4mlQ5bJa9dbklxo/img/8c2af6b8-488d-4f39-995e-5ba5c48d4462.png?Expires=1783781685&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=NMQwqKnChIDGoaCru3Pi%2F4k8L9E%3D "")

1. **接收装车任务**
   - 在小程序 **【任务管理】** 中筛选 **装车** 任务。
   - 查看调度线路、目的站点、货量、温层、车辆、月台信息。
   - 核对装车单据、运单明细，确认货物流向、票件、温区与任务一致。

2. **货物集托分拣**
   - 根据装车线路、温区对货物分类集托。
   - 严格区分冷冻、冷藏货物，禁止混托。
   - 按照 **重下轻上、大下小上** 规范码托。
   - 易碎、易损货物单独摆放防护。
   - 使用PDA逐票扫描待装车货物，核对运单信息。
   - 异常件提前登记隔离。

3. **劳务任务打卡**
   - 装卸、扫描、叉车人员扫描月台二维码。
   - 完成 **装车任务打卡**。

4. **现场装车**
   - 引导车辆停靠指定月台。
   - 车轮垫放三角木，做好安全防护。
   - 按照装车顺序、流向依次装车。
   - 合理规划车厢空间，确保货物摆放稳固，防止运输途中倾倒、破损。
   - 装车过程全程使用PDA扫码复核，确保票、货、单三者一致。

5. **封签与拍照留存**
   - 货物全部装车完毕后，关闭车厢门。
   - 现场施加新封签，并记录封签编号。
   - 拍摄 **车厢全貌、封签特写** 照片，上传至小程序存档。
   - 核对车厢温度，确认符合冷链运输标准。

6. **车辆离场**
   - 再次核对单据、货物、封签无误后，司机办理交接手续。
   - 现场移除三角木。
   - 引导车辆驶离月台。

7. **装车任务验收**
   - 操作人员在小程序/PDA内提交装车完成。
   - 核对装车总量、票数、件数。
   - 如存在货差、破损、漏装等问题，立即上报管理人员并登记异常。
   - 任务状态更新为 **已完成**，整项装车作业结束。

## 操作结果

- 上班打卡完成后，系统记录现场人员签到信息。
- 任务打卡完成后，系统记录本次装卸任务作业人员信息。
- 卸车完成后，可在小程序/PDA中查看 **完成卸车并入库、完成卸车未入库、未卸出货物、运单差异** 等数据状态。
- 装车完成后，可在小程序/PDA中核对装车总量、票数、件数。
- 作业验收无误后，任务状态更新为 **已完成**。
- 封签、车厢、测温等照片上传后，可作为作业凭证与追溯依据。

::: danger 重点提醒

- 温度不达标时，禁止直接装卸，需等待温度恢复至标准范围。
- 卸车前必须核验封签完整性。
- 所有货物必须 **100%上托盘**，不得直接接触地面。
- 货物穿堂停留时间不得超过 **30分钟**。
- 严禁不同温区货物混放，严禁单票多库区混放。
- 参与本次装卸任务的人员需完成任务打卡，便于工时与作业追溯。
:::

::: warning 注意事项
- 作业前请确认账号已登录、权限正常、网络稳定。
- PDA数据同步异常时，先检查网络和设备状态，再联系调度或管理员处理。
- 时间标红代表车辆晚于规划时间到达，需重点跟进。
- 发现破损件、有单无货、有货无单、无头件等异常，应及时在PDA **【问题件登记】** 上报。

:::

## 常见问题

### 常见异常与兜底方案

| 序号 | 异常现象 / 报错提示 | 常见原因 | 解决方案 |
|------|---------------------------|------------|------------|
| 1 | 小程序无法查看装卸任务 | 账号无权限、网络中断、未登录账号 | 1. 确认账号已正常登录；2. 切换稳定网络，刷新页面；3. 联系管理员开通任务查看权限。 |
| 2 | PDA无法扫描、数据不同步 | PDA断网、设备卡顿、任务未下发 | 1. 检查PDA网络，重启设备；2. 刷新任务列表，等待后台数据同步；3. 联系调度重新下发任务。 |
| 3 | 货物测温不达标（温度超标） | 车厢预冷不足、货物脱温 | 暂停装卸作业，等待车厢降温至标准区间；溯源货物转运环节，登记异常上报。 |
| 4 | 封签破损、缺失、被撬动 | 运输途中异常、封签人为损坏 | 立即拍照留证，登记异常问题件，同步上报分拨负责人与对应调度。 |
| 5 | 有货无单/有单无货、票件不符 | 分拣错误、漏扫、单据异常 | 停止作业，使用PDA登记问题件，逐票复盘核对，补齐单据或找回漏货。 |
| 6 | 人员无法完成劳务打卡 | 二维码失效、不在任务人员名单、定位异常 | 1. 重新刷新月台二维码；2. 联系管理员添加作业人员名单；3. 开启小程序定位权限。 |
| 7 | 货物混温区、码放不规范 | 作业人员操作不规范 | 立即重新分拣、重新码托，严格按照冷冻/冷藏分区作业，现场管理人员现场督导。 |
| 8 | 任务列表为空 | 调度未下发装卸任务、筛选条件错误 | 1. 取消筛选，重新查询；2. 联系调度确认是否已下发对应任务。 |

### 冷链货物测温标准是多少？温度不达标可以直接作业吗？

冷藏货物标准 **2℃~8℃**，冷冻货物标准 **≤-15℃**。温度不达标时禁止开门装卸，需等待温度恢复至标准范围。

### 封签核验有问题该怎么处理？

发现封签破损、缺失、撬动痕迹时，第一时间多角度拍照留存，停止作业并上报现场负责人，按异常件流程登记处理。

### 装卸作业发现破损货、无头件如何操作？

立即使用PDA进入 **【问题件登记】** 模块上报，单独隔离存放，禁止继续装车/入库，并同步告知管理人员。

### 劳务打卡必须每一个装卸人员都打卡吗？

是的，参与本次装卸任务的所有人员均需扫描月台码完成任务打卡，用于工时与作业人员追溯。

### 货物可以直接放在地面上吗？

不可以。所有货物必须 **100%上托盘**，严禁直接接触地面，同时遵循 **重不压轻、大不压小** 原则。

### 货物入库后在穿堂区域最长能停留多久？

货物穿堂停留时间不得超过 **30分钟**，需及时转运至对应库区。

### 装车、卸车照片、测温记录需要上传吗？

需要。封签、车厢、测温照片均需按要求上传至小程序，作为作业凭证与追溯依据。

### 任务显示数据和实际货量不一致怎么办？

暂停收尾验收，逐票扫码复盘核对，查找差异原因，登记异常并上报管理人员处理。
