# 车辆交接

## 一、适用场景

本手册适用于以下两类车辆交接场景，由总部调度、省区调度、交车司机、接车司机按角色操作：

- **换人交接**：司机班组更换、车辆移交其他驾驶员，需完整核验车辆全部车况、里程、冷机数据。
- **月末加油交接**：月度固定满油交接，仅需完成满油核验与交车提交，无需接车确认。

::: tip 核心名词解释
- **交车司机**：当前持有车辆、发起交车操作的在岗驾驶员。
- **接车司机**：接收车辆、核验车况并完成接车确认的新驾驶员。
- **ENG小时 / SON小时**：车辆两套冷机运行累计工时，由车载设备自动同步至小程序。
- **车辆交接照片**：必填影像凭证，包含车头、左右两侧、车尾、工具箱、撑杆隔温板、仪表盘码表。
- **转交功能**：交接任务中途更换对接人员，仅支持中心/集配人员线上转交，司机更换需在PC后台修改。
- **抄送提醒**：交接完成后，钉钉鲸天审批中心推送流程抄送通知，管理员可查看完整交接记录。
- **车辆状态**：分为**启用**（司机在运营）、**闲置**（停放在分拨/集配，无司机使用）。
:::

## 二、前置条件

1. **账号权限**
   - PC端：鲸天系统账号，开通**运营运输管理 → 车辆管理 → 车辆交接**调度权限。
   - 移动端：司机微信登录**中通冷链司机版**，账号已绑定对应车辆，自动开放车线任务 → 车辆交接功能。

2. **设备权限**
   手机需开启小程序**定位、相机、相册**，用于拍摄车辆交接照片；车载冷机设备保持通电，保证工时数据正常读取。

## 三、操作入口

- **PC后台**：鲸天系统 → **运营运输管理** → **车辆管理** → **车辆交接**
- **小程序**：中通冷链司机版 → 底部【**车线任务**】 → 车辆交接卡片

## 四、操作步骤

### 4.1 创建车辆交接任务（PC调度后台）

1. 选择**车辆号码**，系统自动带出该车绑定调度单号、班次、原有主副驾信息。
2. 选择**任务类型**：**换人交接**或**月末加油交接**。
3. 填写**接车主驾**、**接车副驾**人员信息。
4. 选定**交接地点**（分拨/途径站点，系统校验交接位置合规性）。
5. 填写期望**交接日期**（系统用于校验后续运输任务冲突）。
6. 提交全部信息，任务下发至对应司机小程序【车线任务】列表。

![image.png](./images/调度任务操作说明书_10_3dc077b4.png)

### 4.2 交车司机提交交车（换人/加油交接通用）

1. 进入车线任务页面，找到待交接车辆卡片，向右滑动【**交车**】滑块。
2. 系统自动读取仪表盘码表里程、ENG/SON冷机工时、加油时间。若识别里程错误，**手动修改数值**；冷机工时读取失败，检查冷机供电后刷新页面。
3. 拍摄并上传全套车辆照片：**车头、车辆左侧、车辆右侧、车尾、工具箱、撑杆与隔温板、码表截图**（缺一不可）。
4. 核对接车人员、交接地点、各项数据。如需更换对接人，点击【**转交**】，搜索并选择目标人员确认。
5. 确认无误后点击【**提交**】完成交车操作。
   - **月末加油交接**：交车提交后流程直接完结，无需接车确认。
   - **换人交接**：交车提交完成，等待接车司机核验确认。

![image.png](./images/调度任务操作说明书_11_b5eb8418.png)

### 4.3 接车司机核验并确认接车（仅换人交接）

1. 在任务列表滑动【**接车**】滑块，进入车辆交接详情页。
2. 逐项核对：**车辆总里程、ENG冷机工时、SON冷机工时、加油记录、全套车辆照片**。
3. **无车辆异常**：直接点击【**提交**】，接车确认完成，整单车辆交接闭环。
4. **发现车辆破损、设备故障、油量异常等问题**：
   1. 点击页面【**异常上报**】。
   2. 选择异常分类，拍摄故障实拍图片，填写100字内异常描述。
   3. 提交异常记录后再完成接车确认，异常数据同步后台存档。

![image.png](./images/调度任务操作说明书_12_ea060445.png)

### 4.4 交接任务人员转交操作

1. 在交车/接车详情页底部点击【**转交**】按钮。
2. 在人员搜索框输入姓名/手机号，筛选交接地点所属调度、集配工作人员。
3. 选中目标转交人员，点击【**确认**】完成线上转交。

::: warning 注意事项
司机之间更换交接人，不可小程序转交，必须前往PC鲸天后台修改交接人员。
:::

![image.png](./images/调度任务操作说明书_13_0edadde6.png)

### 4.5 管理人员查看交接抄送（钉钉鲸天审批中心）

1. 在钉钉接收流程抄送推送消息，点击【**查看详情**】。
2. 进入审批中心车辆交接页面，查看**交车提交时间、接车提交时间、参与人员、车辆牌照、交接类型**。
3. 可查看完整交接影像、里程工时、异常上报记录，用于对账、责任追溯。

![image.png](./images/调度任务操作说明书_14_98dda60e.png)

### 4.6 PC后台车辆状态查询

1. 进入鲸天系统 → **车辆管理** → **车辆信息**。
2. 按**车牌、车队、车辆状态（闲置/启用）、证件有效期**筛选。
3. 列表字段说明：
   - **启用**：车辆分配司机，处于运营状态。
   - **闲置**：车辆停放在指定分拨/集配，无在岗司机。
   - **停靠地点**：闲置车辆固定停放园区。
4. 支持车辆信息编辑、导出台账、年审/运输证过期预警查看。

![image.png](./images/调度任务操作说明书_15_cd846373.png)

## 五、操作结果

- 换人交接：交车提交 + 接车核验确认后，交接闭环，数据线上留痕。
- 月末加油交接：交车提交后流程直接结束。
- 异常上报：异常记录同步后台，供管理人员追溯。
- 抄送提醒：管理员在钉钉审批中心可查看完整交接记录。

## 六、注意事项

::: danger 重点提醒
- 交接拍照必须包含：**车头、车身左侧、车身右侧、车尾、工具箱、撑杆隔温板、仪表盘码表**，缺一不可，否则无法提交。
- 车辆外观刮擦、冷机故障**必须**在接车前走【异常上报】上传照片说明，未报备直接确认接车，后续故障由接车司机承担。
- 司机之间换人交接**不可**使用小程序转交，必须由调度在PC鲸天后台修改接车人。
- 月末加油交接提交后仍显示待处理：可能是任务类型选错（选为换人交接），需联系调度撤回，重新创建月末加油交接任务。
:::

::: warning 注意事项
- 小程序看不到车辆交接任务：调度未下发、车辆未绑定、定位异常。可联系调度重新下发，核对绑定手机号，开启定位权限后刷新。
- 冷机ENG/SON工时读取为空：检查冷机供电，通电等待5分钟后重新进入页面同步数据。
- 码表里程识别错误：手动修改正确数值，重新拍摄清晰仪表盘照片。
- 无法上传照片：检查小程序相机/相册权限。
- 接车无法提交提示存在未上报车况：先完成异常上报再提交接车。
- 钉钉收不到交接抄送通知：联系管理员开通流程抄送权限，解除消息拦截。
:::

## 七、常见问题

### 7.1 Q1 换人交接和月末加油交接流程区别是什么？
A：月末加油交接只需交车司机满油拍照提交，流程直接结束，不需要接车确认；换人交接必须交车提交 + 接车核验确认两步才算完成。

### 7.2 Q2 交接拍照最少需要拍摄哪些部位？
A：车头、车身左侧、车身右侧、车尾、工具箱、撑杆隔温板、仪表盘码表，全部缺一不可，缺项无法提交。

### 7.3 Q3 车辆外观刮擦、冷机故障一定要上报吗？
A：必须在接车前走【异常上报】上传照片说明，未报备直接确认接车，后续故障损耗由接车司机承担。

### 7.4 Q4 中途更换司机，小程序转交能用吗？
A：不能，转交功能仅针对中心管理人员切换；司机换人交接，由调度在PC鲸天后台修改接车人。

### 7.5 Q5 车辆状态“闲置”代表什么？
A：车辆没有分配在岗司机，停放在对应分拨/集配中心。闲置车辆如需启用，调度创建交接任务分配司机即可。

### 7.6 Q6 交接完成后去哪里查历史交接记录？
A：管理人员可在钉钉审批中心抄送消息查看；司机可在车线任务【已完成】分类查看全部交接台账。

### 7.7 Q7 冷机工时一直不更新怎么办？
A：检查冷机电源是否正常通电，断电重启冷机，等待5分钟后重新进入交接页面同步数据。

### 7.8 Q8 交车忘记拍码表照片可以事后补传吗？
A：不可以，交车提交后无法二次修改照片，必须重新发起交接任务补齐凭证。
