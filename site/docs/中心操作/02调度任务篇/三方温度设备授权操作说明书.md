# 三方温度设备授权

## 一、适用场景

物流调度、设备管理员需要对**易流**和**捷依**两类主流车载/便携冷链温控设备进行跨企业数据共享授权，实现中通云冷实时读取外部承运车辆轨迹、车厢温湿度、冷机运行、报警数据，满足冷链全程温控追溯和在途监控业务需求。

## 二、前置条件

- **账号权限**：
  - PC端：易流云/捷依管理后台管理员账号，开通 **共享中心/设备共享** 操作权限。
  - APP端：易流云、捷依官方管理APP管理员账号。
  - 中通接收账号：开通 **使用共享授权** 查看权限。
- **系统入口**：
  - 易流PC：易流云管理后台（精益管车 → 共享中心）
  - 易流APP：易流云移动端（共享中心）
  - 捷依PC：捷依后台报表中心（设备管理）
  - 捷依APP：捷依管理手机客户端
- **前期核对**：确认车辆、便携设备已完成基础建档且在线；提前对接合作方获取对应合作识别码。

- 核心名词
  - **共享授权**：设备所属企业将车辆/便携设备监控数据开放给合作企业的查看权限，无编辑、设置权限。
  - **合作码/共享识别码**：跨企业对接唯一凭证。易流统一合作码 `32903033`（纯数字），捷依识别码 `39698G`（区分大小写，末尾大写G）。
  - **监控授权**：仅开放查看车辆轨迹、温度、报警、运行数据，不可修改设备参数、司机绑定、报警规则。
  - **三方承运车辆**：外部外协物流公司名下冷链运输车辆。
  - **便携设备**：分拨月台、仓库使用的便携式温湿度采集终端。
  - **使用共享授权**：中通总部端操作，接收外部物流公司开放的车辆/设备监控数据。
  - **限时/永久共享**：捷依设备专属配置，可自定义数据共享起止时间。

## 三、操作入口

- **易流PC端**：易流云首页 → **精益管车** → **共享中心** → **提供共享授权**
- **易流APP端**：易流云APP → **共享中心** → **提供授权**
- **捷依PC端**：捷依后台 → **报表中心** → **设备管理** → **+新增共享**
- **捷依APP端**：捷依管理APP → 设备共享功能入口
- **中通接收（PC端）**：易流云 → **精益管车** → **共享中心** → **使用共享授权**
- **中通接收（APP端）**：易流云APP → **共享中心** → **使用授权**

## 四、操作步骤

### 4.1 场景一：易流温度设备授权

#### (1) 三方承运车辆授权（易流PC端）

1. 进入 **提供共享授权** 页面，点击 **添加合作**。
2. 在对方合作码输入框填写统一编码：`32903033`，点击进行授权。
3. 新增合作记录生成后，在列表中找到「中通云冷网络科技(浙江)有限公司(IOT)」，点击 **修改**。
4. 共享数据勾选 **车辆数据**，勾选需要开放监控的目标车辆。
5. 确认授权方式为 **监控授权**，点击 **保存** 完成数据共享开通。

::: tip 补充说明
监控授权可重复授予多家合作企业，仅开放查看权限。
:::

#### (2) 三方承运车辆授权（易流APP端）

1. 进入共享中心页面，点击 **新增/提供授权**。
2. 选择 **添加合作**，输入合作码 `32903033`。
3. 系统自动匹配中通合作企业，授权数据类型选择 **车辆数据**。
4. 选择需要授权的车辆范围（全局/自定义车辆），点击确定提交授权。

#### (3) 分拨便携设备授权（易流PC端）

1. 同1-1步骤，先添加合作，输入合作码 `32903033`，完成企业合作绑定。
2. 进入 **设备管理** → **便携式设备管理**。
3. 找到需要共享的便携温湿度设备，点击 **快速绑定**。
4. 确认设备编号、使用仓库/月台信息无误，提交绑定。
5. 回到共享中心，修改合作授权，追加勾选 **仓库/便携设备数据**，保存。

#### (4) 中通总部接收易流共享数据（PC端）

1. 切换至 **使用共享授权** 标签，页面展示所有外部已授权企业。
2. 筛选合作企业，点击 **开始使用** 即可查看对方开放车辆、便携设备温度、轨迹数据。
3. 无需查看时点击 **结束使用** 关闭临时读取权限。

#### (5) 中通总部接收易流共享数据（APP端）

1. 进入使用授权列表，查看所有外协物流企业。
2. 选中企业，点击 **开始使用** 调取对方共享车辆温控监控。
3. 任务结束或不再需要监控时点击 **结束使用**。

### 4.2 场景二：捷依温度设备授权

#### (1) 三方承运车辆授权（捷依PC端）

1. 进入 **新增共享**，共享识别码输入 `39698G`（严格区分大小写）。
2. 系统自动带出中通冷链合作主体。
3. 勾选需要授权的车牌号码。
4. 设置共享模式：**限时共享**（填写开始/结束时间）或 **永久共享**。
5. 配置数据权限：可选择仅生效时段查看或全部历史数据。
6. 点击 **共享** 提交，完成温控数据开放。

#### (2) 三方承运车辆授权（捷依APP端）

1. 登录捷依管理APP，找到设备共享功能入口。
2. 输入共享识别码 `39698G`。
3. 选定授权车辆，设置共享有效期。
4. 提交完成授权。

## 五、操作结果

- **授权方**：在共享中心或设备共享列表中可看到已开放的合作企业和车辆/设备信息。
- **接收方（中通）**：在 **使用共享授权** 页面可看到外部企业列表，点击 **开始使用** 后即可查看轨迹、实时温度、报警记录等数据。
- **授权即时生效**：正常提交后即可读取数据；若延迟5分钟以上无数据，退出系统重新登录刷新。

## 六、注意事项

::: warning 合作码填写
- 易流合作码：`32903033`（纯数字，无大小写区分）
- 捷依识别码：`39698G`（末尾大写G，必须严格区分大小写，输错无法匹配企业）
:::

::: danger 权限说明
监控授权仅开放数据查看权限，中通**不可**修改车辆设备参数、报警规则、司机绑定等信息。
:::

::: tip 便携设备授权提醒
仅车辆授权看不到仓库温感数据，需在修改授权中追加勾选 **仓库/便携设备数据**。
:::

::: warning 共享时效
- 捷依支持限时共享和永久共享，短期项目选限时共享，长期合作选永久共享。
- 到期自动关闭数据读取，可重新共享或延长有效期。
:::

::: danger 解除授权
- 易流：在共享中心找到对应企业，点击 **解除授权**。
- 捷依：进入共享列表删除该条共享记录，或设置共享结束时间为当日。
- 注意：若当前存在正在使用的共享会话，需先通知中通端结束使用，再解除授权。
:::

## 七、常见问题

### 7.1 Q1 易流和捷依合作码分别是什么？区分大小写吗？

**A**：易流固定合作码 `32903033`（纯数字，无大小写区分）；捷依识别码 `39698G`（末尾大写G，必须严格区分大小写，输错匹配不到企业）。

### 7.2 Q2 监控授权后，中通可以修改车辆设备参数吗？

**A**：不可以。监控授权仅开放数据查看权限，无设备配置、报警规则、司机绑定修改权限。

### 7.3 Q3 外协车辆到期不再合作，如何关闭温度共享？

**A**：易流：共享中心找到对应企业，点击 **解除授权**；捷依：进入共享列表删除该条共享记录，或设置共享结束时间为当日。

### 7.4 Q4 便携月台温感设备需要单独授权吗？

**A**：需要。仅车辆授权看不到仓库温感，添加合作后需在修改授权中勾选 **仓库/便携设备数据**。

### 7.5 Q5 中通调度在哪查看外协车辆温度？

**A**：PC/易流APP **使用共享授权** 页面，对应企业点击 **开始使用**，即可查看轨迹、实时温度、报警记录。

### 7.6 Q6 捷依限时共享和永久共享怎么选？

**A**：短期运输项目选 **限时共享**，填写项目起止日期；长期稳定外协车队选择 **永久共享**。

### 7.7 Q7 授权完成多久后可以读取温度数据？

**A**：正常提交授权即时生效；若延迟5分钟以上无数据，退出系统重新登录刷新。

### 7.8 Q8 多家外协车队需要分别添加合作吗？

**A**：是的，每家物流公司独立创建合作授权，分开勾选对应名下车辆。

### 7.9 常见异常与解决方案

| 异常现象 | 常见原因 | 解决方案 |
|----------|----------|----------|
| 输入合作码无匹配中通企业 | 合作码输错、大小写错误、企业未开通共享账户 | 易流填 `32903033`；捷依填写 `39698G` 区分大小写，联系设备厂商开通合作账户 |
| 授权后中通看不到车辆温度 | 仅添加合作未勾选车辆数据、设备离线、共享未保存 | 进入修改授权页面，勾选车辆；确认车载冷机设备在线，重新保存授权 |
| 便携设备授权后无仓库温感数据 | 共享仅勾选车辆，未追加仓库/便携设备权限 | 在提供共享授权页面修改，新增勾选仓库/便携设备数据项 |
| 捷依共享过期看不到数据 | 设置了限时共享，到期自动关闭 | 重新发起共享，延长共享结束时间或选择永久共享 |
| 使用共享授权列表空白 | 外协企业未提交共享、账号无读取权限 | 联系对方管理员完成车辆授权；联系厂商开通使用共享权限 |
| APP添加合作提交失败 | 网络中断、设备未完成基础建档 | 切换稳定网络；先在PC后台完善车辆/设备信息再授权 |
| 授权后只能看轨迹，无温度数据 | 共享仅开通基础定位，未开放温控数据 | 修改共享配置，勾选温湿度、冷机、报警相关数据权限 |
| 无法解除企业共享授权 | 当前存在正在使用的共享会话 | 通知中通端结束使用，再点击解除授权 |
