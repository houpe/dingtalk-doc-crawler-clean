# 一. 三方温度设备授权

## 1.1 业务场景与名词解释

### 1.1.1 业务场景

本手册适用于物流调度、设备管理员，针对<strong>易流、捷依</strong>两类主流车载/便携冷链温控设备，规范三方承运车辆、分拨便携温感设备跨企业数据共享授权操作。通过PC管理后台、手机APP两种渠道完成设备监控权限开通，实现中通云冷可实时读取外部承运车辆轨迹、车厢温湿度、冷机运行、报警数据，满足冷链全程温控追溯、在途监控业务需求，同时明确总部接收共享数据操作流程。

### 1.1.2 核心名词解释

- <strong>共享授权</strong>：设备所属企业将车辆/便携设备监控数据开放给合作企业的权限操作，仅开放监控查看权限，无设备编辑、设置权限。
- <strong>合作码/共享识别码</strong>：跨企业对接唯一凭证；易流统一合作码`32903033`，捷依识别码`39698G`（区分大小写）。
- <strong>监控授权</strong>：仅支持查看车辆轨迹、温度、报警、运行数据，不可修改设备参数、司机绑定、报警规则。
- <strong>三方承运车辆</strong>：外部外协物流公司名下冷链运输车辆，需授权中通查看温控数据。
- <strong>便携设备</strong>：分拨月台、仓库使用便携式温湿度采集终端，需绑定并共享数据。
- <strong>使用共享授权</strong>：中通总部端操作，接收外部物流公司开放的车辆/设备监控数据。
- <strong>限时/永久共享</strong>：捷依设备专属配置，可自定义数据共享起止时间。

## 1.2 前置准备与环境配置

1. <strong>账号权限</strong>

- PC端：易流云/捷依管理后台管理员账号，开通【共享中心/设备共享】操作权限；
- APP端：易流云、捷依官方管理APP管理员账号；
- 中通接收账号：开通「使用共享授权」查看权限。

2. <strong>配套系统入口</strong>

- 易流PC：易流云管理后台；易流APP：易流云移动端；
- 捷依PC：捷依后台报表中心；捷依APP：捷依管理手机客户端。

3. <strong>前期核对</strong> 确认车辆、便携设备已完成基础建档、设备在线；提前对接合作方获取对应合作识别码。

## 1.3 场景化标准操作步骤

## 1.4 场景11 三方承运车辆授权（易流PC端）

- 系统路径：易流云首页 → 精益管车 → 共享中心 → 提供共享授权

1. 进入【提供共享授权】页面，点击【添加合作】；
2. 在对方合作码输入框填写统一编码：`32903033`，点击进行授权；
3. 新增合作记录生成后，列表找到「中通云冷网络科技(浙江)有限公司(IOT)」，点击【修改】；
4. 共享数据勾选【车辆数据】，勾选需要开放监控的目标车辆；
5. 确认授权方式为「监控授权」，点击保存完成数据共享开通。

补充：监控授权可重复授予多家合作企业，仅开放查看权限。

## 1.5 场景12 三方承运车辆授权（易流APP端）

- 系统路径：易流云APP → 共享中心 → 提供授权

1. 进入共享中心页面，点击【新增/提供授权】；
2. 选择【添加合作】，输入合作码`32903033`；
3. 系统自动匹配中通合作企业，授权数据类型选择「车辆数据」；
4. 选择需要授权的车辆范围（全局/自定义车辆），点击确定提交授权。

## 1.6 场景13 分拨便携设备授权（易流PC端）

- 第一步共享企业授权：同1-1添加合作，输入合作码32903033完成企业合作绑定；
- 第二步设备绑定共享

1. 路径：易流云首页 → 设备管理 → 便携式设备管理；
2. 找到需要共享的便携温湿度设备，点击【快速绑定】；
3. 确认设备编号、使用仓库/月台信息无误，提交绑定；
4. 回到共享中心，修改合作授权，追加勾选仓库/便携设备数据，保存。

## 1.7 场景14 中通总部接收易流共享数据（PC端）

- 系统路径：易流云 → 精益管车 → 共享中心 → 使用共享授权

1. 切换至【使用共享授权】标签，页面展示所有外部已授权企业；
2. 筛选合作企业，点击【开始使用】即可查看对方开放车辆、便携设备温度、轨迹数据；
3. 无需查看时点击【结束使用】关闭临时读取权限。

## 1.8 场景15 中通总部接收易流共享数据（APP端）

- 系统路径：易流云APP → 共享中心 → 使用授权

1. 进入使用授权列表，查看所有外协物流企业；
2. 选中企业，点击【开始使用】调取对方共享车辆温控监控；
3. 任务结束、不再需要监控时点击【结束使用】。

## 1.9 模块二：捷依温度设备授权操作

### 1.9.1 场景21 三方承运车辆授权（捷依PC端）

- 系统路径：捷依后台 → 报表中心 → 设备管理 → +新增共享

### 1.9.3 常见异常与兜底方案

| 序号 | 异常现象 / 报错提示 | 常见原因 | 解决方案 |
|------|---------------------------|------------|------------|
| 1 | 输入合作码无匹配中通企业 | 合作码输错、大小写错误、企业未开通共享账户 | 易流填32903033；捷依填写39698G区分大小写，联系设备厂商开通合作账户。 |
| 2 | 授权后中通看不到车辆温度 | 仅添加合作未勾选车辆数据、设备离线、共享未保存 | 进入修改授权页面，勾选车辆；确认车载冷机设备在线，重新保存授权。 |
| 3 | 便携设备授权后无仓库温感数据 | 共享仅勾选车辆，未追加仓库/便携设备权限 | 提供共享授权页面修改，新增勾选仓库/便携设备数据项。 |
| 4 | 捷依共享过期看不到数据 | 设置了限时共享，到期自动关闭 | 重新发起共享，延长共享结束时间或选择永久共享。 |
| 5 | 使用共享授权列表空白 | 外协企业未提交共享、账号无读取权限 | 联系对方管理员完成车辆授权；联系厂商开通使用共享权限。 |
| 6 | APP添加合作提交失败 | 网络中断、设备未完成基础建档 | 切换稳定网络；先在PC后台完善车辆/设备信息再授权。 |
| 7 | 授权后只能看轨迹，无温度数据 | 共享仅开通基础定位，未开放温控数据 | 修改共享配置，勾选温湿度、冷机、报警相关数据权限。 |
| 8 | 无法解除企业共享授权 | 当前存在正在使用的共享会话 | 通知中通端结束使用，再点击解除授权。 |

### 1.9.4 高频常见问题（FAQ）

### 1.9.5 Q1 易流和捷依合作码分别是什么？区分大小写吗？

A：易流固定合作码：32903033（纯数字无大小写区分）；捷依识别码：39698G（末尾大写G，必须严格区分大小写，输错匹配不到企业）。

### 1.9.6 Q2 监控授权后，中通可以修改车辆设备参数吗？

A：不可以，监控授权仅开放数据查看权限，无设备配置、报警规则、司机绑定修改权限。

### 1.9.7 Q3 外协车辆到期不再合作，如何关闭温度共享？

A：易流：共享中心找到对应企业，点击【解除授权】；捷依：进入共享列表删除该条共享记录，或设置共享结束时间为当日。

### 1.9.8 Q4 便携月台温感设备需要单独授权吗？

A：需要，仅车辆授权看不到仓库温感，添加合作后需在修改授权中勾选仓库/便携设备数据。

### 1.9.9 Q5 中通调度在哪查看外协车辆温度？

A：PC/易流APP「使用共享授权」页面，对应企业点击开始使用，即可查看轨迹、实时温度、报警记录。

### 1.9.10 Q6 捷依限时共享和永久共享怎么选？

A：短期运输项目选限时共享，填写项目起止日期；长期稳定外协车队选择永久共享。

### 1.9.11 Q7 授权完成多久后可以读取温度数据？

A：正常提交授权即时生效；若延迟5分钟以上无数据，退出系统重新登录刷新。

### 1.9.12 Q8 多家外协车队需要分别添加合作吗？

A：是的，每家物流公司独立创建合作授权，分开勾选对应名下车辆。
