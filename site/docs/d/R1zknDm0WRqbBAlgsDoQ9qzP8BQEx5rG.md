# 问题件如何有效回复

## 一、适用场景

本文适用于网点在系统中进行**问题件发布、查询、回复、举证、申诉、撤销**等操作。

## 二、前置条件

1. 已具备对应系统账号，并已开通相关菜单权限。
2. 网点可操作权限如下（**品控工作台-操作**）：
   - **发布**：发布问题件
   - **问题件查询**：问题件查询、回复、举证、撤销
   - **举证申诉**：申诉
   - **超时未回复申诉**：申诉

## 三、操作入口

**操作入口**：**服务质量**

- 流程节点人员：在 **【操作】** 中处理、查看数据。
- 非流程节点人员：在 **【查询】** 中查看数据。

## 四、操作步骤

### 4.1 发布问题件

1. 进入问题件页面后，点击 **【发布】**。
2. 在弹窗中输入发布内容。
3. 确认内容无误后，点击 **【确认】** 完成发布。

::: tip 说明
问题件支持**单条发布**和**批量发布**两种模式。
:::

![image.png](./images/R1zknDm0WRqbBAlgsDoQ9qzP8BQEx5rG/仲裁如何申报、申诉？_11_14378a46.png)

### 4.2 回复问题件

1. 在问题件列表中找到需要处理的问题件。
2. 点击对应的 **【回复】** 操作。
3. 按页面要求填写回复内容并提交。

::: danger 重点提醒
**【回复】时效要求**：

- 发布时间为 **T 0:00:00-16:00:00**：被通知网点需在**当天 23:59:59 前回复**。
- 发布时间为 **T+0 16:00:00-T+0 23:59:59**：被通知网点需在**次日 11 点前回复**。
:::

### 4.3 举证问题件

1. 在问题件列表中找到需要举证的问题件。
2. 点击对应的 **【举证】** 操作。
3. 按页面要求填写举证内容并提交。

::: danger 重点提醒
**【举证】规则**：

1. 举证登记不规范、虚假问题件：**必须是问题件被通知方才能举证**，且必须在**自问题件发布之日起的 72 小时内举证**，超时不允许。
2. 举证回复不规范：**必须是问题件发布方才能举证**，且必须在**回复后的 72 小时之内举证**，超时不允许。
:::

### 4.4 撤销问题件

1. 在问题件列表中找到需要撤销的问题件。
2. 点击对应的 **【撤销】** 操作。
3. 按页面提示完成撤销。

::: danger 重点提醒
**【撤销】仅发布方可操作。**
:::

![image.png](./images/R1zknDm0WRqbBAlgsDoQ9qzP8BQEx5rG/仲裁如何申报、申诉？_1_575eadf9.png)

### 4.5 回复/发布问题件（鲸小宝）

1. 进入鲸小宝问题件页面。
2. 根据实际需要，对鲸小宝问题件进行 **回复**、**完结**、**撤销** 操作。

::: tip 说明
支持鲸小宝问题件进行**回复 / 完结 / 撤销**操作。
:::

![image.jpeg](./images/R1zknDm0WRqbBAlgsDoQ9qzP8BQEx5rG/仲裁如何申报、申诉？_2_1b53f941.png)

## 五、操作结果

完成操作后，可在问题件列表中查看对应问题件的处理结果，包括发布、回复、举证、撤销或鲸小宝问题件的回复、完结、撤销结果。

## 六、注意事项

::: warning 注意事项
1. 流程节点人员通过 **【操作】** 处理和查看数据；非流程节点人员通过 **【查询】** 查看数据。
2. 举证、回复、撤销均有权限或时效要求，操作前请确认当前账号是否符合条件。
3. 超出规定时效后，系统可能不允许继续举证或回复，请按页面提示处理。
:::
