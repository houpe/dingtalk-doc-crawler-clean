# 调度任务

## 一、适用场景

本手册基于**鲸天系统**编制，适用于运输调度岗位人员。

当需要完成以下操作时，可参考本文：

- 运输基础数据维护：运输公司、车辆、司机档案维护。
- 班线/整车调度计划创建。
- 车辆、司机派单。
- 运单预配载、调度单打印。
- 运输任务跟踪。
- 运输费用审批。
- 配套司机端小程序任务协同。

通过以上操作，实现冷链运输线上化、标准化管控，保障干线、支线、整车运输任务有序执行。

## 二、前置条件

1. **账号与权限**
   使用专属工号登录**鲸天系统**。调度岗位需开通**运营运输管理**全模块权限；如无权限，请联系系统管理员配置。

2. **设备与环境**
   使用办公电脑操作，并确保网络稳定；如需打印调度单，请提前连接打印机。

3. **系统入口**
   官方系统入口：**鲸天系统后台（内网访问）**。

4. **基础数据准备**
   调度前需维护运输公司、车辆、司机等基础档案。

## 三、操作入口

主要操作入口如下：

- **鲸天系统 → 运营运输管理 → 基础资料 → 运输公司维护**
- **鲸天系统 → 运营运输管理 → 车辆管理 → 车辆信息**
- **鲸天系统 → 运营运输管理 → 司机管理 → 司机信息**
- **鲸天系统 → 运营运输管理 → 运输计划 → 调度计划**
- **鲸天系统 → 运营运输管理 → 整车管理 → 整车订单**
- **鲸天系统 → 运营运输管理 → 运输监控 → 运输任务管理**
- **鲸天系统 → 运营运输管理 → 整车管理 → 运输费用审批**

## 四、核心名词解释

- **基础数据**：包含运输公司、车辆、司机三大基础档案，是开展调度派单的前置数据源。
- **固定班线**：常态化运行的固定线路班次，系统自动生成调度计划，直接安排车辆与司机即可。
- **临时班线**：非常规临时线路、加班车，需手动新增调度计划，提交总部值班调度审核后生效。
- **预配载**：调度根据库存、在途运单提前完成货物分配，指导现场 PDA 装车作业。
- **整车订单**：客户整车运输需求单据，需完成录入、编辑、审批、派车全流程操作。
- **运输任务管理**：集中查看所有调度单、子任务、发车及运单明细，跟踪运输全流程状态。
- **运输费用审批**：对已完结调度任务的运费、补扣款进行审核，数据同步至天玑系统。

## 五、操作步骤

## 六、基础数据配置

### 6.1 新增运输公司

**操作入口**：**鲸天系统 → 运营运输管理 → 基础资料 → 运输公司维护**

1. 进入页面后，点击上方**新增**按钮。
2. 按要求填写运输公司信息：
   - 公司名称填写全称。
   - 选择对应签署车队。
   - 完善联系电话、地址、资质证件等信息。
   - 带 **\*** 的字段为必填项。
   - 备注栏填写主营线路。
3. 上传法人身份证、营业执照、道路运输许可证等资质附件。
4. 确认信息无误后，点击**保存**。
5. 提交后联系**总部运输部纪宗君**完成审核。

::: danger 重点提醒
未审核的运输公司无法正常选用；已审核信息如需修改，也需联系总部处理。
:::

### 6.2 新增车辆信息

**操作入口**：**鲸天系统 → 运营运输管理 → 车辆管理 → 车辆信息**

1. 点击列表上方**新增**按钮。
2. 填写车辆信息：
   - 区分**内部/外部**车辆属性。
   - 选择所属车队、运输公司、车型、额定载重/体积等必填项。
   - 按需设置分配车队。
   - 按需填写挂车号牌。
3. 信息核对无误后，点击**保存**。

::: tip 说明
车辆信息无需审核。若调度时无法选择该车辆，可等待 **5分钟** 后退出账号重新登录。
:::

### 6.3 修改车辆信息

**操作入口**：**鲸天系统 → 运营运输管理 → 车辆管理 → 车辆信息**

1. 在车辆列表中勾选目标车辆。
2. 点击**编辑**。
3. 修改对应内容。
4. 编辑完成后，点击**保存**提交。

::: warning 注意事项
仅车辆**所属车队**管理员可编辑车辆信息。
:::

### 6.4 新增司机信息

**操作入口**：**鲸天系统 → 运营运输管理 → 司机管理 → 司机信息**

1. 点击**新增**按钮。
2. 填写司机资料：
   - 司机姓名。
   - 手机号。
   - 身份证。
   - 驾驶证。
   - 从业资格证。
   - 区分内部/外部/平台司机来源。
   - 选择所属车队。
   - 上传各类证件图片。
3. 核对信息后，点击**保存**。

::: tip 说明
司机信息无需审核。调度无法选中司机时，退出账号重新登录即可。

## 七、班线车辆调度

### 7.1 固定班线调度

**操作入口**：**鲸天系统 → 运营运输管理 → 运输计划 → 调度计划**

1. 在调度计划页面，将调度状态筛选为**待调度**。
2. 点击**查询**，加载对应班次。
3. 选中目标车线。
4. 点击**调度**按钮。
5. 依次选择或填写以下信息：
   - 车牌号。
   - 挂车号。
   - 运输公司。
   - 主/副班司机。
   - 是否冷藏。
   - 计费方式。
   - 运费价格。
6. 信息确认无误后，点击**确认**，完成调度。

::: tip 补充说明
车辆、司机无法选择时，先完成基础数据录入。

如需换车、换司机，可执行**重新调度**；如需增减经停点，选择**更改经停**。以上操作均需总部值班调度审核。
:::

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/XNkOM5j4x8NMAOY7/img/7bbc2ecb-4f30-489f-8188-c5e3f69570f1.png?Expires=1783781679&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=2Zky2ic47%2BeFHBWpaHJmILDssyc%3D "")

### 7.2 临时班线/加班车调度

**操作入口**：**鲸天系统 → 运营运输管理 → 运输计划 → 调度计划**

1. 点击页面**新增临时调度计划**。
2. 填写基础信息：
   - 始发站点。
   - 目的站点。
   - 规划发车/到达时间。
   - 总里程。
3. 如有多个经停点，点击**添加**进行补充。
4. 选择或填写车辆、司机及费用信息：
   - 车牌号。
   - 挂车号。
   - 司机。
   - 运输公司。
   - 是否冷藏。
   - 报价及计费信息。
5. 点击**保存提交**。
6. 联系**总部值班调度**完成审核。

::: danger 重点提醒
临时班线/加班车审核通过后，司机方可接收任务。
:::

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/XNkOM5j4x8NMAOY7/img/216b9b18-38ac-46c6-9f71-17bf70829e3e.png?Expires=1783781679&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=DyH7JLIG0U8oRH7jXPbyI4807Bw%3D "")

### 7.3 调度单预配载

**操作入口**：**鲸天系统 → 运营运输管理 → 运输计划 → 调度计划**

1. 选中对应调度单号。
2. 点击操作列**货物配载**。
3. 选择发车信息。
4. 点击**配载**，加载在库、在途运单。
5. 勾选需要加入配载列表的运单。
6. 确认后，点击**提交**保存数据。

::: warning 注意事项
已发车站点不支持预配载。配载结果用于指导现场 PDA 装车。
:::

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/XNkOM5j4x8NMAOY7/img/5d80befb-a534-469b-9cdc-633dd06563e7.png?Expires=1783781679&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=9xLNhWLoqxGQIlF%2FcrRQLa9EXhI%3D "")

### 7.4 调度单打印

**操作入口**：**鲸天系统 → 运营运输管理 → 运输计划 → 调度计划**

1. 勾选一条或多条需要打印的调度单。
2. 点击**调度单打印**，输出单据。

::: tip 说明
调度单包含调度单号、车辆、司机、途经站点、时间等信息，用于装卸车扫码使用。
:::

## 八、整车订单全流程操作

### 8.1 新增整车订单

**操作入口**：**鲸天系统 → 运营运输管理 → 整车管理 → 整车订单**

1. 点击**新增整车订单**。
2. 填写订单信息：
   - 客户类型。
   - 温控要求。
   - 指定车队。
   - 申请人及联系方式。
   - 装/卸货地址、联系人等途径信息。
3. 录入货物信息：
   - 货物名称。
   - 重量。
   - 体积。
   - 件数。
   - 货物价值。
4. 按要求选择计费方式，并填写整车价格。
5. 全部信息核对完成后，点击**保存**。

::: tip 填写说明
货物价值按实际填写，保险费按货值千分之三收取。
:::

## 九、运输任务管理

**操作入口**：**鲸天系统 → 运营运输管理 → 运输监控 → 运输任务管理**

1. 通过调度单号、车牌号、发车时间、站点等条件筛选数据。
2. 点击**查询**。
3. 选中调度单，向右滑动或双击查看明细。
4. 可查看以下信息：
   - 调度列表：查看运输计划、费用、经停站点。
   - 运输子任务：查看分段运输、封签、装卸图片。
   - 发车信息：查看货量、票数、体积、重量。
   - 运单列表：查看每笔运单详细资料。

## 十、运输费用审批

**操作入口**：**鲸天系统 → 运营运输管理 → 整车管理 → 运输费用审批**

1. 按车牌、调度单号、时间筛选待审核单据。
2. 点击**审核**。
3. 查看车辆、司机、线路、里程、时效等基础信息。
4. 按需填写补款/扣款金额。
5. 选择审批意见（同意/不同意）。
6. 填写备注。
7. 点击**保存**提交。

::: tip 说明
审批结果实时同步至天玑系统；审批驳回将退回上一环节。
:::

## 十一、操作结果

完成以上操作后，可得到对应结果：

- 运输公司、车辆、司机基础档案维护完成，可用于后续调度。
- 固定班线或临时班线调度完成，车辆、司机安排生效。
- 调度单完成预配载后，可指导现场 PDA 装车作业。
- 调度单打印后，可用于现场装卸、扫码交接。
- 整车订单完成新增、编辑、审批、派车及费用追加。
- 运输任务管理页面可跟踪调度单、运输子任务、发车信息和运单明细。
- 运输费用审批完成后，审批结果会**实时同步至天玑系统**。

## 十二、注意事项

::: warning 注意事项
- 运输公司新增完成后，需联系**总部运输部纪宗君**审核，审核通过后才可正常选用。
- 车辆信息无需审核；若调度时无法选择车辆，可等待 **5分钟** 后退出账号重新登录。
- 司机信息无需审核；调度无法选中司机时，退出账号重新登录即可。
- 仅车辆**所属车队**管理员可编辑车辆信息。
- 仅司机**所属车队**管理员可编辑司机信息。
- 临时班线/加班车提交后，需联系**总部值班调度**审核，审核通过后司机方可接收任务。
- 换车、换司机可执行**重新调度**；增减经停点选择**更改经停**，以上操作均需总部值班调度审核。
- 已发车站点不支持预配载。
- 整车订单货物价值按实际填写，保险费按货值千分之三收取；若不需要投保，货物价值填写**0.01**。
- 完成车辆、司机调度安排后即可打印调度单。
- 费用审批驳回将退回上一环节。
:::

## 十三、常见问题

### 13.1 常见异常与兜底方案

| 序号 | 异常现象 / 报错提示 | 常见原因 | 解决方案 |
|------|---------------------------|------------|------------|
| 1 | 新增调度时，无法选择车辆/司机 | 未提前录入车辆、司机基础数据；数据录入后未生效 | 1. 前往车辆/司机信息模块补充档案；2. 退出系统重新登录，或等待5分钟再操作。 |
| 2 | 临时调度计划提交后，司机收不到任务 | 未联系总部值班调度完成审核 | 提交计划后第一时间对接总部值班调度审核。 |
| 3 | 车辆/司机信息无法编辑 | 当前登录账号无对应车队编辑权限 | 联系对应车队管理员进行信息修改。 |
| 4 | 整车订单提交审批后状态无变化 | 审批人员未处理、系统缓存异常 | 1. 提醒对应审批人处理单据；2. 刷新页面或重新登录系统。 |
| 5 | 调度单预配载功能无法操作 | 该班次车辆已发车 | 已发车站点不支持配载，仅可在发车前操作。 |
| 6 | 运输费用审批提交失败 | 网络卡顿、单据状态异常 | 1. 切换稳定网络重试；2. 核对调度单状态，完结单据才可进行费用审批。 |
| 7 | 司机小程序看不到调度下发任务 | 调度单未审核、手机号绑定错误、小程序无定位权限 | 1. 确认调度单已完成总部审核；2. 核对司机绑定手机号与系统一致；3. 指导司机开启小程序定位权限。 |

### 13.2 新增运输公司后，为什么不能直接选用？

运输公司新增完成后，必须联系总部运输部纪宗君审核，**审核通过**才可在调度、订单模块正常选择使用。

### 13.3 固定班线和临时班线调度有什么区别？

固定班线为常态化线路，系统自动生成计划，直接调度即可；临时班线/加班车需要手动新增计划，且必须经总部值班调度审核，司机才能接单。

### 13.4 车辆故障需要更换车辆，如何操作？

在对应调度计划页面点击**重新调度**，更换新车辆及司机，操作完成后提交总部值班调度审核。

### 13.5 整车订单里的货物价值该如何填写？

按货物实际价值填写，保险费按货值千分之三计收；若不需要投保，货物价值填写**0.01**。

### 13.6 运输途中司机上报异常，调度需要做什么？

及时查看司机提交的异常类型、文字说明及图片凭证，根据实际情况审核处理，同步更新运输任务状态。

### 13.7 调度单什么时候可以打印？

完成车辆、司机调度安排后即可打印，单据用于现场装卸、扫码交接使用。

### 13.8 运输费用审批完成后，数据会同步到哪里？

审批结果会**实时同步至天玑系统**，无需二次录入。
