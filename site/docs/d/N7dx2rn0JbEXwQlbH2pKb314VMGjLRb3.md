# 基础数据商品管理

## 一、适用场景

本文适用于在 **OMS订单中心** 维护商品基础资料，并将商品信息同步到 **WMS** 的操作场景。

- **ERP客户**：完成 **ERP对接** 后，由 **ERP创建商品基础资料**，并自动同步商品基础资料到 **OMS**。
- **非ERP客户**：需要手动在 **OMS** 创建商品资料。

## 二、前置条件

1. 已具备系统登录账号，并拥有 **OMS订单中心 > 基础数据 > 商品管理** 的操作权限。
2. 已确认商品基础信息，例如：**货主、商品名称、商家商品编码、条形码、品牌、品类、产地** 等。
3. 如需同步到 **WMS**，请确认相关基础数据已在 WMS 中维护完整，例如 **包装单位**。

## 三、操作入口

- **系统功能路径**：`登录系统` -> `进入左侧菜单栏` -> `[OMS订单中心]` -> `[基础数据]` -> `[商品管理]`
- **官方系统登录入口**：👉 [[https://wms.ztocc.com/app/#/base/goods](https://wms.ztocc.com/app/#/base/goods)]
- **快捷直达链接**：👉 [[https://wms.ztocc.com/app/#/base/goods](https://wms.ztocc.com/app/#/base/goods)]

## 四、操作步骤

### 4.1 新增商品

1. 进入 **商品管理** 页面。

2. 根据客户类型选择对应操作方式：

   - **非ERP客户**：进入页面后，点击上方 **新增** 按钮。
   - **ERP客户**：由 **ERP同步商品信息到OMS**。

3. 非ERP客户点击 **新增** 后，按页面要求填写商品信息。

   需填写的信息包括：**货主、商品名称、商家商品编码、条形码、品牌、品类、产地、商品成本、商品标签、商品类型、温层、允收期(天)、保质期(天)、允发期(天)、体积、毛重、净重、有无批次、匹配规则、是否易碎品、是否液体、原装发货、型号、规格、包装材料类型、包装单位**。

   ::: danger 重点提醒
   带 **\*** 的字段为必填项。
   :::

   ![image.png](./images/N7dx2rn0JbEXwQlbH2pKb314VMGjLRb3/单据-_入库单_1_b9ab1010.png)

4. 保存商品信息。

5. **WMS对接**：OMS商品信息保存后，会自动同步到 **WMS** 中。

## 五、操作结果

商品资料创建或同步完成后：

1. **OMS** 中可查看到对应商品基础资料。
2. 商品信息保存后会自动同步到 **WMS**。
3. 如同步失败，可根据报错信息检查相关基础数据是否已维护。

## 六、注意事项

::: warning 注意事项
- **ERP客户** 的商品资料由 **ERP同步到OMS**，一般无需在 OMS 手动新增。
- **非ERP客户** 需要在 **OMS商品管理** 页面手动新增商品资料。
- 如商品需要同步到 **WMS**，请确保 **WMS** 中已维护相关基础数据，例如 **包装单位**。
:::

## 七、常见问题

### 7.1 Q1：商品未推送到WMS，报“商品单位不存在”怎么办？

**A**：通常是因为 **OMS中包装单位在WMS中未维护**，导致推送失败。请先在 **WMS** 中维护对应包装单位后，再处理商品同步。
