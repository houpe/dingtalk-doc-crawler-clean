# 系统账号开通及权限配置

## 一、适用场景

本文适用于网点、省区等人员在系统中进行以下操作：

- 开通鲸天系统账号及对应宝盒账号。
- 新增员工账号。
- 给员工配置组织和角色权限。
- 开通或维护鲸小宝、PDA 等账号。
- 区分**员工管理**和**用户管理**的使用场景。

## 二、前置条件

操作前请确认：

1. 已具备对应系统的登录账号。
2. 当前账号具备新增员工、授权、用户管理等相关权限。
3. 已确认员工所属机构（网点）及需要配置的岗位角色。
4. 如需开通鲸小宝、PDA 等账号，需能进入**用户管理**页面。

::: tip 账号创建说明
若在**鲸天系统**中添加账号，系统会自动创建**鲸天系统账号**及对应的**宝盒账号**。
:::

账号创建权限如下：

1. **省区网点管理员账号**：由**总部网管**添加。
2. **一级网点网点管理员账号**：由**省区网管**添加。
3. **一级网点员工**及**下属二级网点网点管理员账号**：由**一级网点网点管理员**添加。
4. **二级网点员工账号**：由**二级网点网点管理员**添加，或由**一级网点网点管理员**添加。

## 三、操作入口

### 3.1 员工管理入口

可通过以下方式进入：

- 菜单搜索：搜索**“员工管理”**
- 菜单路径：**基础配置-用户中心-员工管理**

### 3.2 用户管理入口

菜单路径：**系统管理/用户中心/用户管理**

## 四、操作步骤

### 4.1 添加员工账号

功能说明：网点或下属网点创建新员工账号。

1. 进入**员工管理**页面。
2. 新增员工，并填写员工基础信息、登录账号、密码等信息。
3. 根据员工是否需要查看录单成本，选择对应配置：
   - 需要查看录单成本：选择**“是”**
   - 不需要查看录单成本：选择**“否”**
4. 保存员工信息。

::: warning 注意事项
目前录单成本查看配置默认是**“空白”**。网点管理员给本网点创建员工时，必须注意填写该项。
:::

### 4.2 给员工授权

功能说明：配置员工账号可以查看哪些组织的数据。

1. 进入员工授权页面。
2. 配置**角色类型**。
3. 在**选择角色**中，按照员工岗位选择对应权限。
4. 在**授权组织**中，选择该员工所属的网点组织。
5. 保存授权信息。

::: tip 示例
如果员工同时负责 **A 网点**和 **B 网点**的网点客服岗位，则在**选择组织**时，需要同时勾选 **A 网点**和 **B 网点**。
:::

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnWN5YEy0AlvDe/img/151dc288-e2a0-42ad-a6ce-fd17160e34d0.png?Expires=1784089884&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=VgOiNbDZRgFu7PKj34TZoNgrvEQ%3D "")

::: danger 重点提醒
1. 授权后，员工账号用户状态才是**“已开通”**，才可以登录系统；否则无权限。
2. **角色类型**、**选择角色**需按员工实际岗位选择对应权限。
3. **授权组织**需选择该员工所属的网点组织。
:::

### 4.3 开通鲸小宝、PDA 等账号

功能说明：在**用户管理**中可编辑、删除员工，重置密码，重置 PDA 密码，启用 PDA 用户。

1. 进入**系统管理/用户中心/用户管理**。
2. 找到需要开通或维护的员工账号。
3. 如需开通鲸小宝、PDA 等账号，点击对应位置进行**重置密码**。
4. 重置后，直接使用密码登录即可。
5. 如需维护 PDA 用户，可进行**重置 PDA 密码**或**启用 PDA 用户**等操作。

## 五、操作结果

完成后应达到以下结果：

1. 员工账号已创建。
2. 员工已配置对应的**授权组织**和**授权角色**。
3. 授权完成后，员工账号用户状态为**“已开通”**，可登录系统。
4. 如已在**用户管理**中完成密码重置，员工可使用密码登录鲸小宝、PDA 等账号。

## 六、注意事项

### 6.1 预制基本角色

系统当前预制基本角色如下。

**网点配置的角色：**

- **ZTO 网点管理员**
- **ZTO 网点客服**
- **ZTO 网点操作员**
- **ZTO 代理点业务员**
- **ZTO 一级网点财务/ZTO 二级网点财务**

**省区配置的角色：**

- **ZTO 省区网管**
- **ZTO 代理点业务员**
- **ZTO 网点管理员**
- **ZTO 网点客服**
- **ZTO 网点操作员**
- **ZTO 一级网点财务/ZTO 二级网点财务**

### 6.2 员工管理和用户管理的区别

**员工管理**：用于新增员工、设置登录账号、密码、授权组织、授权角色、编辑基础信息。

**用户管理**：用于编辑登录手机、重置鲸小宝 APP 密码、重置 PDA 密码。

::: warning 注意事项
员工离职时，在**员工管理**中删除即可，删除后宝盒就无法正常登录。
:::

## 七、常见问题

### 7.1 员工账号为什么无法登录系统？

请检查员工账号是否已完成授权。授权后，员工账号用户状态才是**“已开通”**，才可以登录系统；否则无权限。

### 7.2 员工同时负责多个网点，授权组织怎么选？

在**选择组织**时，需要同时勾选该员工负责的多个网点。例如同时负责 **A 网点**和 **B 网点**，则同时勾选 **A 网点**和 **B 网点**。

### 7.3 鲸小宝、PDA 账号在哪里开通？

进入**系统管理/用户中心/用户管理**，找到员工账号后进行**重置密码**，重置后可直接用密码登录。
