# 客户如何通过电商平台下单？

## 一、适用场景

客户需要通过电商平台申请中通冷链电子面单，并在打单软件中取号、打印面单、完成下单时，可参考本文操作。

适用对象包括：商家、网点及相关操作人员。

## 二、前置条件

1. 客户已在对应电商平台开店，并可进入平台的**交易（订单）**、**物流服务**、**电子面单**等相关功能。
2. 网点可登录中通冷链系统，并具备**商户电子面单账户管理**、**电子面单申购**等相关权限。
3. 商家需准备好发货地址，并与网点提前确认。
4. 商家如需打印面单，需使用支持中通冷链的打单软件或 ERP。

## 三、操作入口

- 电商平台入口：**交易（订单）——物流服务——电子面单**
- 中通冷链系统入口：
  - **经营管理中心——商户电子面单账户管理——审核**
  - **经营管理中心——商户电子面单账户管理——充值**
  - **经营管理中心——电子面单申购**
- 打单软件入口：请以商家实际使用的打单软件或 ERP 菜单路径为准。

## 四、操作步骤

### 4.1 客户在电商平台申请中通冷链电子面单

1. 客户登录电商平台，进入**交易（订单）——物流服务——电子面单**板块。
2. 进入物流公司选择页面，找到**中通冷链**。
3. 点击**申请合作**。

::: warning 注意事项
需要和商家约定发货地址。部分电商平台的网点选择会根据发货地址过滤网点。

例如：上海青浦网点地址为**上海-上海-青浦区**，则客户的发货地址也需为**上海-上海-青浦区 XX路XX号**，否则可能会出现选择不到对应网点的情况。

### 4.2 网点在中通冷链系统审核商家并充值

#### (1) 商家审核

1. 网点登录中通冷链系统。
2. 进入**经营管理中心——商户电子面单账户管理——审核**。
3. 找到对应商家，点击**审核**即可。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/YdgOkpNv2MJ6l4BX/img/afadd10f-d3fb-40e8-a0de-fa42cef127b1.png?Expires=1783781509&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=GmxKdGZIh%2BuIqzQ06FpYBrWs9NE%3D "")

#### (2) 商家充值

1. 商家审核通过后，进入**经营管理中心——商户电子面单账户管理——充值**。
2. 找到对应商家，点击**充值**。
3. 根据实际需要输入充值或扣减数量：
   - 输入**正整数**：给商家充值电子面单。
   - 输入**负整数**：扣减商家电子面单余额，并将扣减数量返还至网点对应渠道的电子面单账户。

::: warning 注意事项
- **审核后才有充值按钮**。
- 网点给商家充值前，充值数量需小于网点用户的面单余额。
- 如商家有**100个**面单余额，则最多允许输入**-100**。
:::

### 4.3 网点购买对应渠道面单

1. 网点进入**经营管理中心——电子面单申购**。
2. 根据商家对接的平台，购买对应渠道的电子面单。
   - 目前中通冷链已对接的平台有：**淘宝（不含1688）**、~~**抖音**~~、**快手**。
   - 若商家为淘宝商家，则需要购买**菜鸟电子面单**。

::: warning 注意事项
若在此处没有对应渠道的电子面单，需要联系**省区财务**上架对应渠道的面单。

同时需要省区先向总部申购后，网点才可以申购。

**抖音暂不支持。**
:::

### 4.4 商家在打单软件配置打印模板

#### (1) 订购对应的打单软件服务

1. 商家在平台订购对应的打单软件服务。
2. 若商家已有在使用的打单软件或 ERP，则无需执行该步骤。

::: warning 注意事项
目前各打单软件并非全部支持中通冷链。商家可以先点击**适用版**，免费适用后确认是否支持中通冷链。

目前主流的 ERP、打单软件均支持，例如：**旺店通**、**管家婆**、**快递管家**、**我打**、**快递助手**等。

若不支持，需要商家侧与对应的**软件服务商**售前支持人员反馈，软件服务商对接后即可使用打单功能。
:::

#### (2) 配置打印模板

1. 在打单软件中进入**面单模版维护**入口。
2. 选择对应电商渠道。
3. 选择**中通冷链**。

### 4.5 客户在打单软件取号并打印面单

1. 商家在打单软件中选择对应订单。
2. 选择已配置的**电子面单模版**。
3. 执行打印即可。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/YdgOkpNv2MJ6l4BX/img/327f8676-63c0-4a65-961c-225d40801598.png?Expires=1783781509&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=T1aTexEU6psBR4gxSF%2FVOKT5Tgo%3D "")

::: tip 特殊说明
目前客户在电商平台上下单，选择任意产品，目前均会转换为**零担产品**至鲸天系统内。
:::

### 4.6 网点在鲸天系统接单

1. 客户在平台直接下单后，订单会推送到鲸天系统的**订单管理**中。
2. 网点需及时接单。
3. 后续操作与现有订单操作相同，正常进行转运单等操作即可。

## 五、操作结果

完成以上操作后：

1. 商家可在电商平台申请并使用**中通冷链电子面单**。
2. 网点可在中通冷链系统完成商家**审核**、**充值**及对应渠道面单申购。
3. 商家可在打单软件中配置中通冷链面单模板，并完成取号打印。
4. 客户通过平台下单后，订单会推送至鲸天系统**订单管理**中，网点可按现有流程接单并处理。

## 六、注意事项

1. 发货地址需与网点地址匹配，否则可能无法选择到对应网点。
2. 商家审核通过后，才会出现**充值**按钮。
3. 网点给商家充值前，充值数量需小于网点用户的面单余额。
4. 输入负整数可扣减商家电子面单余额；扣减数量不可超过商家当前面单余额。
5. 若无对应渠道电子面单，需联系**省区财务**上架对应渠道面单，并由省区先向总部申购后，网点才可以申购。
6. **抖音暂不支持**。
7. 打单软件或 ERP 是否支持中通冷链，以软件实际支持情况为准；如不支持，需商家联系对应**软件服务商**售前支持人员反馈。
8. 目前客户在电商平台上下单时，选择任意产品，均会转换为**零担产品**至鲸天系统内。
9. 客户在平台直接下单后，会推送到鲸天**订单管理**中，请及时接单。

## 七、常见问题

### 7.1 客户申请合作时选择不到对应网点怎么办？

请先核对客户发货地址是否与网点地址匹配。部分电商平台会根据发货地址过滤网点，例如上海青浦网点地址为**上海-上海-青浦区**，客户发货地址也需为**上海-上海-青浦区 XX路XX号**。

### 7.2 商户电子面单账户管理中没有充值按钮怎么办？

需先完成商家**审核**。审核后才会出现**充值**按钮。

### 7.3 找不到对应渠道的电子面单怎么办？

需要联系**省区财务**上架对应渠道的面单。同时需要省区先向总部申购后，网点才可以申购。

### 7.4 商家使用的打单软件不支持中通冷链怎么办？

需要商家侧与对应的**软件服务商**售前支持人员反馈，软件服务商对接后即可使用打单功能。

## 八、补充说明

1. 若客户自有系统，支持对接物流公司取号，可直接通过**电子面单**方式下单。
2. 若客户有其他自研系统，不支持物流公司直接取号，且对方有研发能力，可通过接口对接**中通冷链开放平台**的方式下单。详情联系 @侯鹏。
