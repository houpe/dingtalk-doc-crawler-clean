# 差重差方如何申诉

## 适用场景

本文适用于责任网点需要对差重差方进行申诉的业务场景。

## 前置条件

请确认账号已开通对应菜单权限，并已准备好申诉所需的业务数据、申诉内容和附件。

## 操作入口

请以系统实际菜单路径为准。

## 操作步骤

1. 责任网点在规定时间内进入差重差方申诉页面。责任网点可在 **72小时内** 进行申诉。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Lk3lbVmx2dkGlm96/img/7ebfdecc-5595-44e3-853e-94fbe4f580dd.png?Expires=1783781578&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=vF684Aq5R7IKX3HCi%2FRvn1fMC%2FI%3D "")

2. 点击 **【处理】**，进入详情页。
3. 在详情页填写申诉内容，并上传附件。
4. 提交申诉。提交后，流程将流转至总部，由总部审核。

![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/Lk3lbVmx2dkGlm96/img/01661711-c7f5-4f81-93d6-0b21a66dbca7.png?Expires=1783781578&OSSAccessKeyId=LTAI5tKTjg4Kq1HCdBJ8qpSp&Signature=Lrg0VMysUA%2Fr9S6ptY70RKgvPho%3D "")

## 操作结果

提交成功后，申诉流程流转至总部审核。请在系统页面确认申诉状态或处理结果是否符合预期。

## 注意事项

::: warning 注意事项
累计责任：同个责任网点当月被成功投诉 **大于5次后（不含第5次）**，每票累计处罚 **1000元**。

如页面展示、权限范围或业务规则与本文不一致，请以当前系统配置和最新业务规则为准。
:::
