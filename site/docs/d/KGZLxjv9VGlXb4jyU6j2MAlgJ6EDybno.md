# 三方温度设备授权

## 一、适用场景

本手册适用于物流调度、设备管理员，用于指导 **易流**、**捷依** 两类主流车载/便携冷链温控设备的数据共享授权操作。

适用业务包括：

- 三方承运车辆温控设备授权给中通查看。
- 分拨月台、仓库便携式温湿度采集终端授权给中通查看。
- 中通总部接收外部物流公司开放的车辆/设备监控数据。

授权完成后，中通云冷可实时读取外部承运车辆的 **轨迹、车厢温湿度、冷机运行、报警数据**，用于冷链全程温控追溯和在途监控。

## 二、前置条件

1. **账号权限**
   - PC端：具备 **易流云/捷依管理后台管理员账号**，并开通 **共享中心/设备共享** 操作权限。
   - APP端：具备 **易流云、捷依官方管理APP管理员账号**。
   - 中通接收账号：已开通 **使用共享授权** 查看权限。

2. **设备与数据准备**
   - 车辆、便携设备已完成基础建档。
   - 设备处于在线状态。
   - 已提前获取对应合作识别码：
     - 易流统一合作码：`32903033`
     - 捷依识别码：`39698G`

3. **核心名词说明**
   - **共享授权**：设备所属企业将车辆/便携设备监控数据开放给合作企业，仅开放监控查看权限，无设备编辑、设置权限。
   - **合作码/共享识别码**：跨企业对接唯一凭证；易流统一合作码 `32903033`，捷依识别码 `39698G`（区分大小写）。
   - **监控授权**：仅支持查看车辆轨迹、温度、报警、运行数据，不可修改设备参数、司机绑定、报警规则。
   - **三方承运车辆**：外部外协物流公司名下冷链运输车辆，需授权中通查看温控数据。
   - **便携设备**：分拨月台、仓库使用的便携式温湿度采集终端，需绑定并共享数据。
   - **使用共享授权**：中通总部端操作，用于接收外部物流公司开放的车辆/设备监控数据。
   - **限时/永久共享**：捷依设备专属配置，可自定义数据共享起止时间。

::: danger 重点提醒
捷依识别码 `39698G` **严格区分大小写**，末尾为大写 **G**。
:::

## 三、操作入口

### 3.1 易流

- 易流PC：**易流云管理后台**
- 易流APP：**易流云移动端**
- 三方承运车辆授权PC路径：**易流云首页 → 精益管车 → 共享中心 → 提供共享授权**
- 三方承运车辆授权APP路径：**易流云APP → 共享中心 → 提供授权**
- 分拨便携设备管理路径：**易流云首页 → 设备管理 → 便携式设备管理**
- 中通总部接收共享数据PC路径：**易流云 → 精益管车 → 共享中心 → 使用共享授权**
- 中通总部接收共享数据APP路径：**易流云APP → 共享中心 → 使用授权**

### 3.2 捷依

- 捷依PC：**捷依后台报表中心**
- 捷依APP：**捷依管理手机客户端**
- 三方承运车辆授权PC路径：**捷依后台 → 报表中心 → 设备管理 → +新增共享**
- 捷依APP设备共享入口：请以系统实际菜单路径为准。

## 四、操作步骤

### 4.1 易流：三方承运车辆授权（PC端）

系统路径：**易流云首页 → 精益管车 → 共享中心 → 提供共享授权**

1. 进入 **提供共享授权** 页面，点击 **添加合作**。
2. 在对方合作码输入框填写统一编码：`32903033`，点击进行授权。
3. 新增合作记录生成后，在列表中找到 **中通云冷网络科技(浙江)有限公司(IOT)**，点击 **修改**。
4. 在共享数据中勾选 **车辆数据**，再勾选需要开放监控的目标车辆。
5. 确认授权方式为 **监控授权**，点击保存，完成数据共享开通。

::: tip 补充说明
监控授权可重复授予多家合作企业，仅开放查看权限。
:::

### 4.2 易流：三方承运车辆授权（APP端）

系统路径：**易流云APP → 共享中心 → 提供授权**

1. 进入 **共享中心** 页面，点击 **新增/提供授权**。
2. 选择 **添加合作**，输入合作码 `32903033`。
3. 系统自动匹配中通合作企业后，授权数据类型选择 **车辆数据**。
4. 选择需要授权的车辆范围（全局/自定义车辆）。
5. 点击确定，提交授权。

### 4.3 易流：分拨便携设备授权（PC端）

#### (1) 共享企业授权

1. 参考“易流：三方承运车辆授权（PC端）”添加合作。
2. 输入合作码 `32903033`，完成企业合作绑定。

#### (2) 设备绑定共享

1. 进入路径：**易流云首页 → 设备管理 → 便携式设备管理**。
2. 找到需要共享的便携温湿度设备，点击 **快速绑定**。
3. 确认设备编号、使用仓库/月台信息无误后，提交绑定。
4. 返回 **共享中心**。
5. 修改合作授权，追加勾选 **仓库/便携设备数据**。
6. 点击保存，完成便携设备共享。

### 4.4 易流：中通总部接收共享数据（PC端）

系统路径：**易流云 → 精益管车 → 共享中心 → 使用共享授权**

1. 切换至 **使用共享授权** 标签。
2. 页面展示所有外部已授权企业。
3. 筛选合作企业，点击 **开始使用**。
4. 查看对方开放的车辆、便携设备温度和轨迹数据。
5. 无需查看时，点击 **结束使用**，关闭临时读取权限。

### 4.5 易流：中通总部接收共享数据（APP端）

系统路径：**易流云APP → 共享中心 → 使用授权**

1. 进入 **使用授权** 列表，查看所有外协物流企业。
2. 选中企业，点击 **开始使用**。
3. 调取对方共享车辆的温控监控数据。
4. 任务结束或不再需要监控时，点击 **结束使用**。

### 4.6 捷依：三方承运车辆授权（PC端）

系统路径：**捷依后台 → 报表中心 → 设备管理 → +新增共享**

1. 点击 **新增共享**。
2. 在共享识别码中输入 `39698G`（严格区分大小写）。
3. 系统自动带出中通冷链合作主体。
4. 勾选需要授权的车牌号码。
5. 设置共享模式：
   - **限时共享**：填写开始/结束时间。
   - **永久共享**：按页面要求选择永久共享。
6. 配置数据权限，可选择 **仅生效时段查看** 或 **全部历史数据**。
7. 点击 **共享** 提交，完成温控数据开放。

::: danger 重点提醒
`39698G` 必须严格区分大小写，输错会匹配不到企业。
:::

### 4.7 捷依：三方承运车辆授权（APP端）

1. 登录 **捷依管理APP**。
2. 找到设备共享功能入口。
3. 输入共享识别码 `39698G`。
4. 选定授权车辆。
5. 设置共享有效期。
6. 提交后完成授权。

技术咨询热线：**4008204680**

## 五、操作结果

授权完成后，应达到以下结果：

- 外部承运车辆或便携设备已成功共享给中通。
- 中通可通过 **使用共享授权** 查看被授权企业开放的车辆/设备数据。
- 可查看的数据包括：**轨迹、实时温度、报警记录、运行数据** 等已授权内容。
- 如不再需要查看，可通过 **结束使用** 关闭临时读取权限。
- 如需关闭企业共享，可按对应系统操作解除或删除共享记录。

## 六、注意事项

::: danger 重点提醒
- **监控授权** 仅开放数据查看权限，不可修改设备参数、司机绑定、报警规则。
- 便携月台温感设备需要单独授权，仅车辆授权无法查看仓库温感数据。
- 捷依设置为 **限时共享** 后，到期会自动关闭共享。
- 授权前请确认车辆、便携设备已完成基础建档且设备在线。
:::

::: warning 注意事项
- 易流固定合作码为 `32903033`，为纯数字，无大小写区分。
- 捷依识别码为 `39698G`，末尾为大写 **G**。
- 多家外协车队需要分别添加合作授权，并分别勾选对应名下车辆。
- 授权后如中通仍看不到数据，可先检查是否已勾选车辆/便携设备数据、设备是否在线、共享是否保存。
:::

## 七、常见问题

### 7.1 常见异常与处理方案

| 序号 | 异常现象 / 报错提示 | 常见原因 | 解决方案 |
|------|---------------------------|------------|------------|
| 1 | 输入合作码无匹配中通企业 | 合作码输错、大小写错误、企业未开通共享账户 | 易流填32903033；捷依填写39698G区分大小写，联系设备厂商开通合作账户。 |
| 2 | 授权后中通看不到车辆温度 | 仅添加合作未勾选车辆数据、设备离线、共享未保存 | 进入修改授权页面，勾选车辆；确认车载冷机设备在线，重新保存授权。 |
| 3 | 便携设备授权后无仓库温感数据 | 共享仅勾选车辆，未追加仓库/便携设备权限 | 提供共享授权页面修改，新增勾选仓库/便携设备数据项。 |
| 4 | 捷依共享过期看不到数据 | 设置了限时共享，到期自动关闭 | 重新发起共享，延长共享结束时间或选择永久共享。 |
| 5 | 使用共享授权列表空白 | 外协企业未提交共享、账号无读取权限 | 联系对方管理员完成车辆授权；联系厂商开通使用共享权限。 |
| 6 | APP添加合作提交失败 | 网络中断、设备未完成基础建档 | 切换稳定网络；先在PC后台完善车辆/设备信息再授权。 |
| 7 | 授权后只能看轨迹，无温度数据 | 共享仅开通基础定位，未开放温控数据 | 修改共享配置，勾选温湿度、冷机、报警相关数据权限。 |
| 8 | 无法解除企业共享授权 | 当前存在正在使用的共享会话 | 通知中通端结束使用，再点击解除授权。 |

### 7.2 易流和捷依合作码分别是什么？区分大小写吗？

易流固定合作码：`32903033`（纯数字，无大小写区分）。

捷依识别码：`39698G`（末尾大写 **G**，必须严格区分大小写，输错匹配不到企业）。

### 7.3 监控授权后，中通可以修改车辆设备参数吗？

不可以。监控授权仅开放数据查看权限，无设备配置、报警规则、司机绑定修改权限。

### 7.4 外协车辆到期不再合作，如何关闭温度共享？

- 易流：进入 **共享中心**，找到对应企业，点击 **解除授权**。
- 捷依：进入共享列表删除该条共享记录，或设置共享结束时间为当日。

### 7.5 便携月台温感设备需要单独授权吗？

需要。仅车辆授权看不到仓库温感，添加合作后需在修改授权中勾选 **仓库/便携设备数据**。

### 7.6 中通调度在哪查看外协车辆温度？

在 PC/易流APP 的 **使用共享授权** 页面，对应企业点击 **开始使用**，即可查看轨迹、实时温度、报警记录。

### 7.7 捷依限时共享和永久共享怎么选？

短期运输项目选择 **限时共享**，填写项目起止日期；长期稳定外协车队选择 **永久共享**。

### 7.8 授权完成多久后可以读取温度数据？

正常提交授权后即时生效；若延迟 **5分钟以上** 仍无数据，可退出系统后重新登录刷新。

### 7.9 多家外协车队需要分别添加合作吗？

是的。每家物流公司需要独立创建合作授权，并分别勾选对应名下车辆。
