# 系统访问及APP下载

## 适用场景

本文适用于需要访问 **鲸天系统**、下载 **中通宝盒**、下载 **鲸小宝**，或登录 **中通冷链司机小程序** 的人员。

## 前置条件

1. 登录 **鲸天系统** 需要使用 **中通宝盒账号**进行鉴权。
2. 登录前需确认上级已按手机号开通 **鲸天系统账号**。
3. 若未开通账号，直接登录时会出现左侧 **“未办理入职”** 提示，此时需联系对应上级人员开通账号。
4. **鲸天系统**添加员工账号后，会根据手机号自动同步入职信息到 **宝盒**；同步完成后，才可登录。

## 常见问题

### 登录时提示“未办理入职”怎么办？

说明当前手机号可能未开通 **鲸天系统账号**，请联系对应上级人员开通账号。

### 已添加员工账号后，为什么仍无法登录？

**鲸天系统**添加员工账号后，需要根据手机号同步入职信息到 **宝盒**。请等待同步完成后再登录；如仍无法登录，请联系对应上级人员或系统管理员排查。
