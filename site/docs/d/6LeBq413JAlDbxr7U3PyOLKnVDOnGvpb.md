# 网点录单

## 适用场景

本文适用于**全网网点操作人员**，在日常承接客户寄件需求时，通过**鲸天系统（PC端）**或**鲸小宝APP（移动端）**完成手工录单，录入寄收件信息、货物信息、服务类型等内容，并生成正式运单。

- 文档名称：**中通冷链网点录单操作说明书**
- 更新日期：**2026年3月31日**
- 涉及系统：**鲸天系统（PC端）**、**鲸小宝APP（移动端）**

本次同步完成全线产品体系升级，在原有产品基础上新增多款冷链产品，统一录单流程、产品选择规则、改单限制及面单展示样式，规范全网点录单标准，保障运单信息准确、流转顺畅。

## 操作入口

- PC端入口：**鲸天系统 → 订单管理 → 网点录单**
- 移动端入口：**鲸小宝APP → 网点录单**

## 名词解释

- **产品分类**：录单一级选择类目，分为**冷链快运（原零担/小件）**、**冷链快递**，用于快速区分业务类型。
- **冷链快运**：主流冷链寄件类型，包含多款细分产品，适配不同时效、重量、价位的货物运输需求。
- **冷链快递**：独立产品类型，按专属规则下单、流转与结算。
- **细分产品**：冷链快运下包含多款差异化产品（含鲸确达、鲸速达、鲸惠达等），各产品时效、载重、定价、服务标准不同。
- **改单**：运单生成后，对订单信息、产品类型等内容进行修改的操作。
- **面单**：货物运输凭证，分为子单、签收联、留存联，是货物交接、派送、签收的依据。

## 操作步骤

### PC端录单

1. 进入**鲸天系统 → 订单管理 → 网点录单**。
2. 选择**产品分类**。
   - 可选择：**冷链快运**或**冷链快递**。
   - 录单时先选择一级产品分类，无需直接选择细分产品。
3. 选择**细分产品**。
   - 选择**冷链快运**时，系统会根据**寄收地址GIS定位、运输时效、货物重量、产品成本**四项规则，自动筛选当前可使用的细分产品，并按固定顺序展示，默认选中主推产品。
   - 选择**冷链快递**时，直接按指定产品类型下单，无自动筛选逻辑。
4. 录入基础信息。
   - 依次完整填写寄件人、收件人、省市区、详细地址、联系电话等必填信息。
   - 地址需填写至街道/乡镇、门牌号/园区，确保地址精准。
5. 录入货物与服务信息。
   - 填写货物名称、重量、件数、体积、包装类型。
   - 按需选择保价、增值服务、付款方式、上门时间等内容。
   - 页面会同步展示对应产品的时效、费用及服务说明。
6. 核对全部信息无误后，点击**保存**。
7. 系统生成正式运单。

### 移动端录单

1. 打开**鲸小宝APP**。
2. 进入**网点录单**。
3. 按照与PC端一致的流程操作：
   - 先选择**产品分类**；
   - 再选择对应**细分产品**；
   - 完整录入寄收件信息、货物信息和服务信息；
   - 核对无误后提交生成运单。
4. 如需管理订单，可在APP内订单列表中使用筛选功能，按全品类产品查询、管理订单。

::: tip 说明
移动端录单流程、产品选择逻辑与PC端**完全一致**。
:::

## 操作结果

完成录单并点击**保存**后，系统会生成正式运单。

运单信息可用于后续货物交接、派送、签收等环节；面单可按新版电子面单模板打印，作为货物运输凭证。

## 注意事项

::: danger 重点提醒
订单流转至**中心到件、卸车环节**后，**禁止修改产品类型**，仅可调整收货网点；该订单不再保障原承诺时效。
:::

::: warning 注意事项
- 录单时需确保寄件人、收件人、省市区、详细地址、联系电话等必填信息完整准确。
- 地址需填写至**街道/乡镇、门牌号/园区**，避免因地址不清影响流转。
- 如打印面单标识或格式异常，请在打印设置中切换为**官方新版面单模板**后重新打印。
- 如菜单内找不到对应产品筛选项，可先刷新页面，或重启系统/APP后再次查询。
:::

### 常见异常与处理方式

| 序号 | 异常现象 / 报错提示 | 常见原因 | 解决方案 |
|------|---------------------------|------------|------------|
| 1 | 冷链快运下部分细分产品无法选择 | 货物重量、运输时效、线路成本不满足该产品使用规则，系统自动过滤 | 按照页面展示的可用产品下单，或核对货物、地址信息后重新选择 |
| 2 | 订单无法修改产品类型 | 订单已流转至中心到件、卸车环节，系统限制改产品 | 仅可修改目的网点，该订单不再保障原承诺时效 |
| 3 | 打印面单标识、格式异常 | 未使用最新版面单模板 | 在打印设置中切换为官方新版面单模板，重新打印 |
| 4 | 菜单内找不到对应产品筛选项 | 系统未完成更新或页面缓存异常 | 刷新页面，或重启系统/APP后再次查询 |

## 常见问题

### Q1：录单时为什么要先选择产品分类，再选细分产品？

A：产品分类为一级大类，用于区分**冷链快运**、**冷链快递**两大业务板块；细分产品根据时效、载重、价格划分。系统会自动匹配可用品类，简化操作并符合业务规则。

### Q2：PC端和手机APP录单规则、产品选择逻辑一样吗？

A：**完全一致**。两端流程、筛选规则、改单限制、面单标准全部统一。

### Q3：订单什么情况下不能修改产品类型？

A：订单流转至**中心到件、卸车环节**后，禁止修改产品类型，仅可调整收货网点。

### Q4：批量匹配网点功能，默认选用哪类产品？

A：批量匹配网点默认选用**经济型主推产品**。

### Q5：面单上的简写标识代表什么作用？

A：用于网点、派送人员快速识别订单所属产品类型，提升货物分拣、交接效率。
