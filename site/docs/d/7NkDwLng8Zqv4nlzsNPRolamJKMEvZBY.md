# 司机端注册

## 一、适用场景

本文适用于司机在中通冷运网络货运平台接单前，完成司机端 APP 注册、司机认证和车辆信息提交的操作。

- **司机**：指在中通冷运网络货运平台上的承运方。
- **网络货运平台**：即中通冷运网络货运平台，是专为网点、省公司、生态圈以及 KA 客户等解决整车业务场景的网络货运承运平台。

## 二、前置条件

1. 准备司机本人手机号码，用于注册和登录。
2. 准备一部 **安卓或 iOS 系统的智能手机**，并确保网络连接正常（**4G/5G/Wi-Fi**）。
3. 下载并安装司机端 APP：
   - 👉 [点击下载中通冷运司机端 APP （安卓）](https://zmas.zto.com/download/NTMS)
   - 下载鲸小宝 APP（IOS）：**敬请期待**

## 三、操作入口

- **司机注册入口**：登录 APP -> 使用手机号码 + 验证码登录，即完成注册。
- **司机认证入口**：登录 APP -> **我的** -> **立即认证**。

## 四、操作步骤

### 4.1 司机注册

1. 打开司机端 APP。
2. 使用司机本人手机号码获取验证码。
3. 输入验证码并登录，登录成功后即完成注册。

### 4.2 司机认证

1. 登录 APP 后，进入 **我的** 页面。
2. 点击 **立即认证**。
3. 进入认证页面后，填写认证相关信息，包括 **姓名、身份证、驾驶证** 等。
4. 上传 **身份证、驾驶证照片**。系统会自动识别并回填信息，请仔细核对填写内容，避免系统识别错误导致信息有误。

## 五、操作结果

认证提交后，平台会在 **1-3 个工作日** 内完成审核。请在 APP 内关注审核进度。

## 六、注意事项

::: warning 注意事项
- 上传证件照片后，系统会自动识别信息，请务必核对 **姓名、身份证、驾驶证、车辆信息** 等内容是否正确。
- 如系统识别内容有误，请及时修改后再提交，避免因信息错误影响审核。
- 提交认证后，请关注审核进度。
:::

::: danger 重点提醒
根据网络货运承运平台的国家规定，平台需要对在平台承运的司机进行 **身份证、驾驶证、行驶证、从业资格证、道路运输许可证、道路运输经营许可证** 等证件以及有效期进行严格监管，且挂车需要分别针对车头和挂车进行双重监管。
:::

### 7.1 为什么要填写那么多信息？

根据网络货运承运平台的国家规定，平台需要对在平台承运的司机进行 **身份证、驾驶证、行驶证、从业资格证、道路运输许可证、道路运输经营许可证** 等证件以及有效期进行严格监管，且挂车需要分别针对车头和挂车进行双重监管。
