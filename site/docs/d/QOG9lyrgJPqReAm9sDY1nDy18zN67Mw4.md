# 包仓仓位用的好快，哪里能看到仓位的使用明细呢？

## 一、适用场景

本文适用于需要查看包仓仓位使用明细，或需要确认包仓运单占仓情况的场景。

## 二、前置条件

请确认账号已开通 **财务管理-账单管理-包仓账单** 菜单权限，并已存在需要查看的包仓运单数据。

## 三、操作入口

**财务管理-账单管理-包仓账单**

## 四、操作步骤

1. 登录鲸天系统。
2. 打开 **财务管理-账单管理-包仓账单**。
3. 在列表中找到需要查看的包仓运单。
4. 如果该包仓运单存在使用明细，**操作** 列会展示 **查看详情** 按钮。
5. 点击 **查看详情**，系统会弹出该运单的使用明细。

![image.png](./images/QOG9lyrgJPqReAm9sDY1nDy18zN67Mw4/包仓仓位用的好快，哪里能看到仓位的使用明细呢？_1_4b707d03.png)

## 五、操作结果

点击 **查看详情** 后，可在弹窗中查看运单的包仓使用明细。

## 六、注意事项

::: warning 注意事项
查看运单占仓情况时，需重点关注 **占仓重量**。

当仓位不足时，运单只会部分占仓。
:::

如页面展示、权限范围或业务规则与本文不一致，请以当前系统配置和最新业务规则为准。
