# 车辆交接

## 适用场景

本文适用于**总部调度、省区调度、交车司机、接车司机**进行车辆交接相关操作。

车辆交接分为两个操作端：

- **PC鲸天调度后台**：创建车辆交接任务、修改交接人员、查询车辆状态。
- **中通冷链司机小程序**：司机线上交车、接车确认、异常上报、人员转交。

主要适用于以下两类场景：

- **司机换人交接**：司机班组更换、车辆移交其他驾驶员，需要核验车辆车况、里程、冷机数据，并由接车司机确认。
- **月末加油交接**：月度固定满油交接，仅需完成满油核验与交车提交，无接车确认环节。

通过线上交接，可完成车辆燃油、里程、冷机工时、外观车况核验、提交确认、异常上报、人员转交等流程，便于车辆交接数据留痕、权责划分和后续追溯。

### 核心名词说明

- **换人交接**：司机班组更换、车辆移交其他驾驶员，需要完整核验车辆全部车况、里程、冷机数据的交接类型。
- **月末加油交接**：月度固定满油交接，仅需完成满油核验与交车提交，无接车确认环节。
- **交车司机**：当前持有车辆、发起交车操作的在岗驾驶员。
- **接车司机**：接收车辆、核验车况并完成接车确认的新驾驶员。
- **ENG小时/SON小时**：车辆两套冷机运行累计工时，由车载设备自动同步至小程序。
- **车辆交接照片**：交接必填影像凭证，包含车头、左右两侧、车尾、工具箱、撑杆隔温板、仪表盘码表。
- **转交功能**：交接任务中途更换对接人员，仅支持中心/集配人员线上转交，司机更换需在PC后台修改。
- **抄送提醒**：车辆交接完成后，钉钉鲸天审批中心推送流程抄送通知，管理人员可查看完整交接记录。
- **车辆状态**：分为**启用**（司机在运营）、**闲置**（停放在分拨/集配，无司机使用）。

## 前置条件

1. **账号权限**
   - PC端：需有**鲸天系统账号**，并开通**运营运输管理-车辆管理-车辆交接**调度权限。
   - 移动端：司机需使用微信登录**中通冷链司机版**，账号已绑定对应车辆，系统自动开放**车线任务-车辆交接**功能。

2. **设备权限**
   - 手机需开启小程序**定位、相机、相册**权限，用于定位和拍摄车辆交接照片。
   - 车载冷机设备需保持通电，确保工时数据可正常读取。

3. **基础数据**
   - PC后台需已维护车辆、司机、调度单号、班次等相关信息。
   - 交接车辆需与对应司机账号完成绑定。

## 操作入口

- **PC后台**：**鲸天系统 → 运营运输管理 → 车辆管理 → 车辆交接**
- **小程序**：**中通冷链司机版 → 底部【车线任务】**
- **车辆状态查询**：**鲸天系统 → 车辆管理 → 车辆信息**
- **抄送查看**：**钉钉 → 鲸天审批中心 → 流程抄送消息**

## 操作步骤

### 创建车辆交接任务（PC调度后台）

适用人员：**总部调度、省区调度**。

操作路径：**鲸天系统 → 运营运输管理 → 车辆管理 → 车辆交接**

1. 进入**车辆交接**页面。
2. 选择**车辆号码**，系统会自动带出该车绑定的调度单号、班次、原有主副驾信息。
3. 选择**任务类型**：**换人交接**或**月末加油交接**。
4. 填写**接车主驾**、**接车副驾**人员信息。
5. 选择**交接地点**（分拨/途径站点，系统会校验交接位置合规性）。
6. 填写期望**交接日期**，系统用于校验后续运输任务冲突。
7. 确认信息无误后提交，任务会下发至对应司机小程序的**【车线任务】**列表。

### 提交交车（司机小程序）

适用场景：**换人交接**、**月末加油交接**通用。

操作路径：**中通冷链司机版 → 底部【车线任务】→ 车辆交接卡片**

1. 打开**中通冷链司机版**，进入底部**【车线任务】**页面。
2. 找到待交接车辆卡片，向右滑动**【交车】**滑块，进入交车详情。
3. 查看系统自动读取的数据，包括：
   - **仪表盘码表里程**
   - **ENG冷机工时**
   - **SON冷机工时**
   - **加油时间**
4. 如识别的里程有误，手动修改为正确数值。
5. 如冷机工时读取失败，先检查冷机供电，再刷新页面。
6. 按要求拍摄并上传全套车辆照片：
   - **车头**
   - **车辆左侧**
   - **车辆右侧**
   - **车尾**
   - **工具箱**
   - **撑杆与隔温板**
   - **码表截图**
7. 核对**接车人员、交接地点、里程、冷机工时、照片**等信息。
8. 如需更换对接人，点击**【转交】**，搜索并选择目标人员后确认。
9. 确认无车辆问题后，点击**【提交】**，完成交车操作。

::: tip 流程说明
- **月末加油交接**：交车提交后，流程直接完结，无需接车确认。
- **换人交接**：交车提交后，需等待接车司机核验并确认接车。
:::

### 核验并确认接车（司机小程序）

适用场景：**仅换人交接**。

操作路径：**中通冷链司机版 → 底部【车线任务】→ 车辆交接卡片**

1. 打开**【车线任务】**列表。
2. 找到车辆交接任务，滑动**【接车】**滑块，进入车辆交接详情页。
3. 逐项核对以下内容：
   - **车辆总里程**
   - **ENG冷机工时**
   - **SON冷机工时**
   - **加油记录**
   - **全套车辆照片**
4. 如**无车辆异常**，直接点击**【提交】**，完成接车确认，整单车辆交接闭环。
5. 如发现**车辆破损、设备故障、油量异常等问题**，按以下方式处理：
   1. 点击页面**【异常上报】**。
   2. 选择异常分类。
   3. 拍摄故障实拍图片。
   4. 填写**100字内**异常描述。
   5. 提交异常记录后，再完成接车确认。异常数据会同步后台存档。

### 转交交接任务

适用场景：交接任务中途需要更换对接人员。

1. 在交车或接车详情页面底部，点击**【转交】**按钮。
2. 在人员搜索框中输入**姓名/手机号**。
3. 筛选交接地点所属调度、集配工作人员。
4. 选中目标转交人员，点击**【确认】**，完成线上转交。

::: danger 重点提醒
司机之间更换交接人，**不可**通过小程序转交，**必须**前往**PC鲸天后台**修改交接人员。
:::

### 查看抄送交接流程（钉钉鲸天审批中心）

适用人员：管理人员。

1. 在钉钉中接收流程抄送推送消息。
2. 点击**【查看详情】**。
3. 进入审批中心车辆交接页面。
4. 查看以下交接信息：
   - **交车提交时间**
   - **接车提交时间**
   - **参与人员**
   - **车辆牌照**
   - **交接类型**
5. 可继续查看完整交接影像、里程工时、异常上报记录，用于对账和责任追溯。

### 查询PC后台车辆状态

操作路径：**鲸天系统 → 车辆管理 → 车辆信息**

1. 进入**车辆信息**页面。
2. 按需选择筛选条件：
   - **车牌**
   - **车队**
   - **车辆状态**（**闲置/启用**）
   - **证件有效期**
3. 查看列表字段：
   - **启用**：车辆已分配司机，处于运营状态。
   - **闲置**：车辆停放在指定分拨/集配，无在岗司机。
   - **停靠地点**：闲置车辆固定停放园区。
4. 页面支持进行车辆信息编辑、导出台账、年审/运输证过期预警查看。

## 操作结果

完成车辆交接后，可达到以下结果：

- **PC调度后台创建任务后**：对应司机可在小程序**【车线任务】**中看到车辆交接任务。
- **月末加油交接提交后**：交车司机提交完成，流程直接完结。
- **换人交接提交后**：交车司机提交后，接车司机完成核验并点击**【提交】**，整单车辆交接闭环。
- **存在异常上报时**：异常图片、异常描述等记录同步后台存档。
- **流程完成后**：管理人员可通过钉钉鲸天审批中心抄送消息查看完整交接记录。

## 注意事项

::: danger 重点提醒
- **车辆交接照片**为必填影像凭证，包含**车头、左右两侧、车尾、工具箱、撑杆隔温板、仪表盘码表**。
- 接车时发现车辆破损、设备故障、油量异常等问题，需先通过**【异常上报】**提交异常图片和文字说明，再完成接车确认。
- 司机之间更换交接人，**不可**使用小程序转交，需由调度在**PC鲸天后台**修改交接人员。
- **月末加油交接**无接车确认环节；如任务类型选错为**换人交接**，需联系调度处理。
:::

::: warning 注意事项
- 手机需开启小程序**定位、相机、相册**权限，否则可能影响任务展示、拍照或上传。
- 冷机工时依赖车载设备同步，如读取为空，可先检查冷机设备供电和车载终端状态。
- 仪表盘码表照片应清晰，避免模糊、反光导致里程识别错误。
- 交车提交后如照片无法二次修改，需按实际流程重新处理。
:::

## 常见问题

### 常见异常与处理方式

| 序号 | 异常现象 / 报错提示 | 常见原因 | 解决方案 |
|------|---------------------------|------------|------------|
| 1 | 小程序看不到车辆交接任务 | 调度未在PC后台下发、车辆未绑定本人账号、定位异常 | 1. 联系调度重新下发交接任务；2. 核对车辆绑定司机手机号；3. 开启小程序定位权限，刷新车线任务。 |
| 2 | 冷机ENG/SON工时读取为空 | 冷机设备断电、车载终端离线 | 1. 检查冷机供电开关，通电等待5分钟；2. 退出页面重新进入同步数据。 |
| 3 | 码表里程识别错误 | 照片模糊反光 | 手动修改正确里程数值，重新拍摄清晰仪表盘照片备用。 |
| 4 | 无法上传车辆交接照片 | 小程序无相机权限 | 小程序设置中开启相机、相册权限，重新拍摄上传。 |
| 5 | 转交功能找不到目标司机 | 转交仅支持中心/集配人员，司机不可线上转交 | 前往鲸天PC后台车辆交接页面，手动更换接车/交车司机。 |
| 6 | 接车无法提交，提示存在未上报车况 | 车辆存在破损未走异常上报流程 | 先完成异常图片+文字报备，再提交接车确认。 |
| 7 | 月末加油交接提交后仍显示待处理 | 任务类型选错（选为换人交接） | 联系调度撤回任务，重新创建「月末加油交接」任务。 |
| 8 | 钉钉收不到交接抄送通知 | 审批权限未分配、消息屏蔽 | 联系后台管理员开通流程抄送查看权限，解除消息拦截。 |

### 换人交接和月末加油交接流程区别是什么？

月末加油交接只需交车司机满油拍照提交，流程直接结束，不需要接车司机确认；换人交接必须完成**交车提交 + 接车核验确认**两步才算完成。

### 交接拍照最少需要拍摄哪些部位？

需要拍摄**车头、车身左侧、车身右侧、车尾、工具箱、撑杆隔温板、仪表盘码表**。

::: danger 重点提醒
车辆交接照片全部缺一不可，缺项无法提交。
:::

### 车辆外观刮擦、冷机故障一定要上报吗？

必须在接车前走**【异常上报】**，上传照片并填写说明。未报备直接确认接车，后续故障损耗由接车司机承担。

### 中途更换司机，小程序转交能用吗？

不能。**转交功能**仅针对中心管理人员切换；司机换人交接，需由调度在**PC鲸天后台**修改接车人。

### 车辆状态“闲置”代表什么？

**闲置**表示车辆没有分配在岗司机，停放在对应分拨/集配中心。闲置车辆如需启用，调度创建交接任务分配司机即可。

### 交接完成后去哪里查历史交接记录？

管理人员可在**钉钉审批中心抄送消息**中查看；司机可在车线任务**【已完成】**分类查看全部交接台账。

### 冷机工时一直不更新怎么办？

检查冷机电源是否正常通电，断电重启冷机，等待**5分钟**后重新进入交接页面同步数据。

### 交车忘记拍码表照片可以事后补传吗？

不可以。交车提交后无法二次修改照片，必须重新发起交接任务补齐凭证。
