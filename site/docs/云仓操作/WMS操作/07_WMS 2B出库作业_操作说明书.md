# WMS 2B出库作业

::: danger 重点提醒
本文档覆盖WMS 2B出库全流程：**出库订单 → 出库计划 → 拣货任务（波次/分配/调整）→ 拣货确认 → 交接发货**。当前功能存在局限性，部分未交付功能详见适用场景说明。
:::

## 一、适用场景

2B出库适用于企业对企业的大批量、计划性出库场景。与2C出库不同，2B出库以 **出库计划** 为操作单位，支持多次分批出库、多种拣货模式（批拣/按单拣/边拣边分）以及集货交接。

当前功能局限：
- 不能同2C出库一样自动取号、设置承运方案、灵活指定承运商、自动打印。
- 集货功能代码暂不在交付范围，仅可用的功能可用2C出库包裹替代。
- 取快运子单数量功能：出库计划中 **单据类型=冷链B2B出库（2B）** 时，自动根据SKU总数量取云冷子单数量（新开发功能）。

## 二、前置条件

- **权限要求**：需有仓库管理员权限，完成仓库货主配置中的2B自动化参数设置。
- **系统配置**：
  - 单据类型设置为2B类型（THCK退货出库、QTCK其他出库）。
  - 配置B2B波次规则、任务拆分规则、B2B分配条件、承运商配置。
- **设备要求**：
  - PC端操作：出库订单、出库计划、波次、交接。
  - PDA端操作：B2B拣货、交接发运。

## 四、操作步骤

### 4.1 出库订单 → 生成出库计划

1. **进入菜单**：出库 → **出库订单**。
2. **确认单据类型为2B**：
   - 出库订单类型必须为 **THCK（退货出库）** 或 **QTCK（其他出库）**，否则无法创建出库计划。
   - ![](./images/01_创建仓库、货主、员工_操作说明书_10_e94b96dc.png)
3. **（可选）变更订单信息**：如需修改订单信息，变更后会作废已创建的出库计划，需重新生成。
4. **生成计划**：点击 **生成计划** 按钮，手动创建本次出库数量的出库计划。
   - 是否自动创建由 **仓库货主配置** 中“手动生成计划单据类型”控制。
   - ![](./images/01_创建仓库、货主、员工_操作说明书_11_e213f0ae.png)

### 4.2 出库计划（2B）

- **菜单路径**：出库 → **出库计划（2B）**。
- 出库计划是2B出库订单的一次出库任务，一个订单可拆分为多次出库计划。
- ![](./images/01_创建仓库、货主、员工_操作说明书_12_4b5dcb0d.png)

### 4.3 生成拣货任务（三种方式）

创建拣货任务前需先分配库存。以下三种方式可以组合使用、互相补充。

#### (1) 方式A：加入波次（批量操作，推荐）

多个出库计划一起加入波次，按B2B波次规则自动分配库存。

**波次类型说明**：

| 波次类型 | 说明 |
|----------|------|
| **B2B批拣** | 多单加入一个波次，按拣货任务总拣。 |
| **B2B按单拣选** | 每个出库计划生成一个拣货任务，一单一拣。 |
| **B2B边拣边分** | 每个出库计划分配一个集货位，按SKU总拣后立即分拣到各集货位。若波次跨库区，按库区拆分多个拣选任务。 |

**波次操作流程**：

1. **创建波次**：进入 **出库 → B2B波次**，点击 **创建波次**。
   - ![](./images/01_创建仓库、货主、员工_操作说明书_13_2ccb6f1a.png)
2. **运行波次**：点击 **运行波次**。
3. **（可选）更改集货区**：如需调整集货区，在此步骤操作。
4. **释放波次**：点击 **释放波次**，系统自动生成B2B拣货任务。
   - ![](./images/01_创建仓库、货主、员工_操作说明书_14_457f7d24.png)
5. **打印单据**（务必先释放波次再打印）：
   - **B2B批拣单**：每个拣货任务打印一页，用于总拣。
   - **按单拣货单**：每个出库计划打印一页，用于大2B订单。
   - **送货单**：每个出库计划打印一页，用于与司机/门店交接。

#### (2) 方式B：分配库存（按单操作，用于有特殊分配要求的货主）

1. 在 **出库计划（2B）** 列表选择计划，点击 **分配库存**。
   - 系统根据B2B分配条件自动分配。
2. 分配成功后仍可手动调整。
3. 生成拣货任务（可手动生成或自动生成，视配置而定）。
   - ![](./images/01_创建仓库、货主、员工_操作说明书_15_7567dfc0.png)
   - ![](./images/01_创建仓库、货主、员工_操作说明书_16_ef5e73cc.png)
   - ![](./images/01_创建仓库、货主、员工_操作说明书_17_9cc56e8f.png)

#### (3) 方式C：调整分配（手动指定库存）

当不需要系统规则分配时，可手动指定库存并生成拣货任务。

1. 在 **出库计划（2B）** 列表点击计划 **详情**。
2. 点击 **调整分配**。
3. 检索库位，指定具体库位库存（如仅出临期、仅从退货区出库等）。
4. 调整完成后 → 点击 **生成拣货任务**。
   - ![](./images/01_创建仓库、货主、员工_操作说明书_18_42ee3433.png)
   - ![](./images/01_创建仓库、货主、员工_操作说明书_19_730c34e3.png)
   - ![](./images/01_创建仓库、货主、员工_操作说明书_1_68925fe0.png)

### 4.4 拣货任务确认

以下方式二选一：

#### (1) 方式一：PC端确认

- 路径：出库 → **B2B拣货** → **任务确认**。
- 说明：拣货任务不能更改库位，但可以少拣或多拣（是否允许取决于操作配置 → 拣货设置）。

#### (2) 方式二：PDA拣货

1. 使用PDA登录，进入 **B2B拣货**。
2. 选择对应的拣选模式（B2B批拣 / B2B按单拣货 / B2B边拣边分）。
   - ![](./images/01_创建仓库、货主、员工_操作说明书_20_e59e47f8.png)
   - ![](./images/01_创建仓库、货主、员工_操作说明书_2_5e45b2a3.png)
3. 具体操作说明参考《中通仓链20260423系统更新日志》。

以下方式二选一：

#### (1) PC端交接发货

- 路径：出库 → **交接发货**。
- ![](./images/01_创建仓库、货主、员工_操作说明书_3_925b7af9.png)

#### (2) PDA交接发运

- PDA菜单 → **B2B交接发运**。
- ![](./images/01_创建仓库、货主、员工_操作说明书_4_211d769e.png)

## 五、操作结果

完成以上步骤后：
- 出库订单状态更新，出库计划创建成功。
- 拣货任务生成并完成确认。
- 交接发货后，系统触发发运，库存扣减。
- 可在 **出库计划明细** 总览每SKU的分配、拣货、交接进度。

## 六、注意事项

- **权限**：必须拥有仓库管理员权限，且完成仓库货主配置中的2B自动化参数设置。
- **单据类型**：只有 **THCK（退货出库）** 和 **QTCK（其他出库）** 才能生成2B出库计划。
- **变更订单**：变更出库订单信息后，已创建的计划会被作废，需重新生成。
- **波次释放与打印顺序**：**务必先释放波次，再打印单据**，否则可能导致打印失败或数据错误。
- **集货功能**：当前集货功能代码暂不在交付范围，如需集货可暂时使用2C出库包裹替代。
- **取号与打印**：2B出库的取号打印功能较弱，承运商配置需要提前检查。
- **拣货任务调整**：PC端确认拣货任务时不能更改库位，但可以调整数量（受配置限制）。

## 七、常见问题

### 7.1 Q1：2B出库和2C出库有什么区别？

**A**：2B以“出库计划”为操作单位，支持多次分批出库、多种拣货模式、集货交接。2C以“出库包裹”为单位，支持自动预处理（取号、承运、包装）。2B的取号打印功能相对薄弱。

### 7.2 Q2：三种波次类型如何选择？

**A**：
- **B2B批拣**：多单合并总拣，适合杂乱小订单。
- **B2B按单拣选**：一单一拣，适合大订单。
- **B2B边拣边分**：按集货位分拣，适合面向门店/车线的出库。

### 7.3 Q3：出库订单可以分多次出库吗？

**A**：可以。一个出库订单可以拆分为多次出库计划，分批发货。

### 7.4 Q4：无法创建出库计划怎么办？

**A**：检查出库订单类型是否为2B（THCK/QTCK），并确认 **仓库货主配置** 中“手动生成计划单据类型”已正确设置。

### 7.5 Q5：波次释放后无法打印怎么办？

**A**：请确认打印操作是在释放波次之后进行的。先释放波次，再打印单据。

### 7.6 Q6：取号失败怎么办？

**A**：检查承运商配置，确保已为2B出库单据类型配置承运方案。
