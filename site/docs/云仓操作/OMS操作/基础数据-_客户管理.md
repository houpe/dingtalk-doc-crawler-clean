# 基础数据客户管理

## 一、适用场景

当仓库发起客户入参流程，需要配置新的货主信息时，可参考本文在系统中新增货主。

## 二、前置条件

- 已具备系统登录账号，并有 **OMS订单中心 / 基础数据 / 客户管理** 相关操作权限。
- 已准备好新增货主所需信息，包括：**仓库**、**货主编码**、**货主名称**、**销售负责人**、**行业**、**货主类型**、**货主种类**、**企业类型**、**默认物流公司**、**合伙人/网点**、**erp编码**、**erp类型**、**启用报价确认** 等。
- 配套工具/链接：
  - 🌐 官方系统登录入口：👉 [[https://wms.ztocc.com/app/#/base/customer](https://wms.ztocc.com/app/#/base/customer)]

## 三、操作入口

- 系统功能路径：**登录系统** -> **进入左侧菜单栏** -> **[OMS订单中心]** -> **[基础数据]** -> **[客户管理]**
- 快捷直达链接：👉 [[https://wms.ztocc.com/app/#/base/customer](https://wms.ztocc.com/app/#/base/customer)]

## 四、操作步骤

1. 进入 **客户管理** 页面后，点击上方 **新增** 按钮。

2. 按页面要求填写货主信息，字段包括：**仓库**、**货主编码**、**货主名称**、**销售负责人**、**行业**、**货主类型**、**货主种类**、**企业类型**、**默认物流公司**、**合伙人/网点**、**erp编码**、**erp类型**、**启用报价确认** 等。

   ::: tip 特殊字段说明
   - **仓库**：货主需要把商品存放的仓库，可多选。
   - **erp编码**：ERP 对接时与 ERP 约定的对接编码。
   - **开启报价确认**：若需要仓库对网点报价时开启；若不需要则关闭。
   :::

   ![image.png](./images/单据-_入库单_1_b9ab1010.png)

3. 填写完成后保存货主信息。

4. 如需进行 **WMS对接**，在 OMS 创建货主并保存后，系统会自动同步货主信息到 WMS 中。

5. 如需进行 **ERP对接**，创建货主后，点击 **详情**，再点击 **获取对接参数**，将对接信息提供给 ERP 侧发起对接请求。

   ![image.png](./images/单据-_入库单_2_9b4425c3.png)

## 五、操作结果

- 货主信息创建完成后，可在 **客户管理** 页面查看新增的货主记录。
- 若涉及 **WMS对接**，OMS 保存货主后会自动同步货主信息到 WMS。
- 若涉及 **ERP对接**，可在货主 **详情** 中获取对接参数，并提供给 ERP 侧完成后续对接。

## 六、注意事项

::: warning 注意事项
- **仓库** 可多选，请按货主实际需要存放商品的仓库选择。
- **erp编码** 需与 ERP 侧约定一致，用于后续 ERP 对接。
- **开启报价确认** 需根据是否需要仓库对网点报价来决定：需要则开启，不需要则关闭。
:::
