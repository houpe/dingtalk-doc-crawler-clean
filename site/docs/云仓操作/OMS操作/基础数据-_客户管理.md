# 基础数据客户管理

## 一、适用场景
仓库发起客户入参流程，需要配置新的货主信息时使用。

## 二、前置条件
- 拥有对应系统的账号和操作权限（菜单路径：**OMS订单中心 → 基础数据 → 客户管理**）。
- 已准备好的货主基础信息（如编码、名称、仓库、负责人等）。

## 三、操作入口
- **系统菜单路径**：登录系统 → 左侧菜单栏 → `OMS订单中心` → `基础数据` → `客户管理`
- **快捷直达链接**：[https://wms.ztocc.com/app/#/base/customer](https://wms.ztocc.com/app/#/base/customer)

## 四、操作步骤
1. **新增货主**
   进入客户管理页面后，点击页面顶部的 **新增** 按钮。
2. **填写关键字段**
   按以下要求填写信息（带 `*` 的为必填项）：
   - **仓库**：必填，可多选，指货主存放商品的仓库。
   - **货主编码**：需唯一。
   - **货主名称**：必填。
   - **销售负责人**、**行业**、**货主类型**、**货主种类**、**企业类型**、**默认物流公司**、**合伙人/网点** 等按实际情况填写。
   - **ERP编码**：若与ERP对接，需填写与ERP约定的对接编码。
   - **ERP类型**：选择对应ERP系统类型。
   - **启用报价确认**：若需要仓库对网点进行报价，则开启；若不需要则关闭。

   ::: tip 字段说明
   上图中字段均为示例，请根据实际业务填写。
   ![image.png](./images/单据-_入库单_1_b9ab1010.png)
   :::

3. **保存货主信息**
   填写完成后点击保存，系统自动同步货主信息到 **WMS**。
4. **（可选）ERP对接**
   如果货主需要与ERP系统对接，在保存后按以下步骤操作：
   - 进入该货主的 **详情** 页面。
   - 点击 **获取对接参数**，将对接信息提供给ERP方请求对接。

   ![image.png](./images/单据-_入库单_2_9b4425c3.png)

## 五、操作结果
- 新增货主保存成功后，在客户管理列表中能看到该货主记录。
- 货主信息会自动同步到 **WMS** 系统。
- 若启用了ERP对接，ERP方收到对接参数后可进行后续对接配置。

## 六、注意事项
::: danger 重点提醒
- **ERP编码** 必须与ERP系统约定的编码一致，否则会导致对接失败。
- **启用报价确认** 开关一旦开启，仓库对网点报价时需走确认流程；关闭则直接生效。请按实际业务需求设置。
- 仓库字段 **必须至少选择一个仓库**，且该仓库需已在系统中维护。
- 货主编码一经创建不可修改，请谨慎填写。
:::
