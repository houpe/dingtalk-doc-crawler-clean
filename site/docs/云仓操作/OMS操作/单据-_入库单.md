# 单据入库单

## 一、适用场景

- **ERP客户**：已完成ERP对接，ERP创建入库单后同步到OMS。
- **非ERP客户**：无ERP系统，需手动在OMS创建入库单。

## 二、前置条件

- **配套工具/链接**：官方系统登录入口：[入库单-鲸天系统](https://wms.ztocc.com/app/#/back/warehousing)
- 账号已开通OMS订单中心相关权限。
- 对于非ERP客户，需提前准备好入库商品的基础信息（商品编码、单位等）。

## 三、操作入口

登录系统后，进入左侧菜单栏 → **[OMS订单中心]** → **[入库管理]** → **[入库单]**。

快捷直达链接：[入库单-鲸天系统](https://wms.ztocc.com/app/#/back/warehousing)

## 四、操作步骤

### 4.1 场景一：ERP客户 → 入库单

1. **ERP同步入库单到OMS**
   在ERP中创建入库单，系统自动同步至OMS。
   ![image.png](./images/单据-_入库单_1_b9ab1010.png)

2. **WMS对接**
   OMS接收入库单后，自动同步到WMS系统。

3. **入库单回传**
   WMS入库上架完成后，自动回传入库信息至OMS，再由OMS回传至ERP。

---

### 4.2 场景二：非ERP客户 → 入库单

1. **在OMS创建/导入入库单**
   - 点击 **新增** 或 **导入** 按钮。
   - 填写以下必填信息：
     - **头表信息**：货主、仓库、单据类型、外部单号、物流公司、物流单号、备注、联系人、联系电话、寄件省/市/区、详细地址。
     - **明细信息**：商品、商家系统编码、商家商品编码、总数量、二级单位、最小单位。
   - 填写完成后点击 **保存**，返回入库单列表。
   ![image.png](./images/单据-_入库单_2_9b4425c3.png)

2. **入库单列表操作**
   - **审核**：审核通过后自动同步至WMS。
     ![image.png](./images/单据-_入库单_3_c7162966.png)
   - **编辑**：如需调整信息，点击 **编辑** 修改。
   - **复制**：可复制现有入库单快速创建同类单据。
   - **取消**：用于取消已创建但未完成的入库单。
   - **删除**：物理删除入库单。

3. **入库单回传**
   WMS入库上架完成后，自动回传入库信息至OMS，再由OMS回传至ERP。

## 五、操作结果

- ERP客户：入库单在OMS中自动生成并流转至WMS；WMS上架完成后，OMS和ERP均收到回传入库信息。
- 非ERP客户：入库单在OMS中创建并审核后同步至WMS；最终回传入库信息到OMS，如已对接ERP则回传至ERP。

## 六、注意事项

::: warning 注意事项
- 导入入库单时，若商品二级单位未在WMS中维护，需先在WMS维护单位，同步至OMS后再重新导入。
- 导入的单据号（外部单号）不能重复，如提示“单据【xxx】已存在”，请检查并更新单据号。
- 导入的商品必须属于所选货主，否则会报错“商品【xxx】不属于货主【xxx】”，请核对导入数据。
:::

::: danger 重点提醒
- 对于非ERP客户，新增/导入时必须填写所有必填字段（头表信息和明细信息），否则无法保存。
- 已审核的入库单不可编辑或删除，如需取消请联系管理员。
:::

## 七、常见问题

- **Q1：导入时提示商品二级单位未维护**
  **A**：需先在WMS中维护商品单位，维护后自动同步到OMS，再重新导入入库单。

- **Q2：导入时提示单据【xxx】已存在**
  **A**：当前货主的外部单号已存在，不能重复导入。请更新单据号后重新导入。

- **Q3：导入时提示商品【xxx】不属于货主【xxx】**
  **A**：导入明细中的商品不是当前货主的商品。请检查导入数据，更新后重新导入。
