## 三、场景化标准操作步骤（怎么用？）

### 3.1 场景一：[ERP客户>入库单]

- <strong>系统功能路径</strong>：`登录系统` -> `进入左侧菜单栏` -> `[OMS订单中心]` -> `[入库管理]` -> `[入库单]`
- <strong>快捷直达链接</strong>：👉 [[入库单-鲸天系统](https://wms.ztocc.com/app/#/back/warehousing)]

#### (1) 核心操作步骤

1. <strong>[ERP同步入库单->OMS]</strong> ERP客户：ERP中创建入库单同步入库单到OMS![image.png](./images/单据-_入库单_2_9b4425c3.png)
2. <strong>[WMS对接]</strong> OMS接收入库单->自动同步到WMS中
3. <strong>[入库单回传]</strong> WMS入库上架完成->自动回传入库信息到OMS中->OMS回传入库信息到ERP

### 3.2 场景二：[非ERP客户>入库单]

- <strong>系统功能路径</strong>：`登录系统` -> `进入左侧菜单栏` -> `[OMS订单中心]` -> `[入库管理]` -> `[入库单]`
- <strong>快捷直达链接</strong>：👉 [[入库单-鲸天系统](https://wms.ztocc.com/app/#/back/warehousing)]

#### (1) 核心操作步骤

1. <strong>[OMS创建入库单]</strong> 1.1，客户在OMS中入库单/导入入库单，点击新增/导入1.2，按照要求填写信息：头表信息：货主，仓库，单据类型，外部单号，物流公司，物流单号，备注，联系人，联系电话，寄件省，寄件是，寄件区，详细地址明细信息：商品，商家系统编码，商家商品编码，总数量，二级单位，最小单位1.3，新增/导入文件中必填信息填写完成保存，回到入库单列表![image.png](./images/单据-_入库单_3_c7162966.png)
2. <strong>[入库单列表操作]</strong> 2.1 【<strong>审核</strong>】OMS审核入库单->自动同步到WMS中![image.png](./images/单据-_入库单_4_38169c00.png)2.2 【编辑】新增完成后若需调整信息则点击编辑重新编辑入库单信息2.3 【复制】若需创建相同类型的订单时刻复制新增/导入的入库单，复制后可支持2.4 【取消】若创建入库单有问题则可点击取消按钮，取消入库单2.5 【删除】点击删除按钮，支持物理删除入库单
3. <strong>[入库单回传]</strong> WMS入库上架完成->自动回传入库信息到OMS中->OMS回传入库信息到ERP

## 五、高频常见问题（FAQ）

- <strong>Q1：[导入时提示商品二级单位未维护]</strong>
- <strong>A</strong>：[需要在WMS中维护商品单位，维护后商品单位会自动同步到OMS中，可重新导入入库单]
- <strong>Q2：[导入时提示单据【xxx】已存在]</strong>
- <strong>A</strong>：[当前导入的货主单据号已存在，不能够重复导入，需要更新单据号之后重新导入]
- <strong>Q3：[导入时提示商品【xxx】不属于货主【xxx】]</strong>
- <strong>A</strong>：[导入的商品明细中的商品不是导入货主的商品，需要检查导入数据信息，更新后重新导入]
