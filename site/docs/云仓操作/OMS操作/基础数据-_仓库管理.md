# 基础数据仓库管理

## 一、适用场景

- **新增仓库**：省公司发起开仓流程，需要开启新仓时配置新仓库信息。
- **库存管理**：开启库存管理后，OMS 会根据 WMS 实际出入库商品信息更新库存，库存不足时出库单状态变为缺货。

## 二、前置条件

1. **账号与权限**：使用专属工号登录 **鲸天系统-冷链云仓**，需开通基础数据管理全模块权限。无权限请联系系统管理员配置。
2. **系统链接**：[官方系统登录入口](https://wms.ztocc.com/app/#/base/warehouse)

## 三、操作入口

- **系统菜单路径**：登录系统 → 左侧菜单栏 → **OMS订单中心** → **基础数据** → **仓库管理**
- **快捷直达链接**：[https://wms.ztocc.com/app/#/base/warehouse](https://wms.ztocc.com/app/#/base/warehouse)

## 四、操作步骤

### 4.1 场景一：新增仓库

1. 进入仓库管理页面，点击 **新增** 按钮。
2. 按照要求填写信息：
   - **仓库编码**：ZTO + 省缩写开头
   - **仓库名称**：必填
   - **是否直营**：必填
   - **仓库容积**：必填
   - **仓库联系人**：必填
   - **仓库联系人电话**：必填
   - **启用报价确认**：若需要仓库对网点报价，则开启；否则关闭。
   - **开启库存管理**：若仓库需要管理库存，则开启；否则关闭。开启后，出入库单据完成时 OMS 会根据 WMS 实际信息更新库存，库存不足时出库单状态变为缺货，不推送单据到 WMS。
   - **钉钉 token**：配置后，单据接收或回传报错时会推送钉钉消息到指定钉钉群。
   - **钉钉 secret**：同上。
   - 带 **\*** 为必填项。
3. 填写完成后，点击 **保存**。

## 五、操作结果

- 新增仓库成功后，列表中将出现该仓库记录。
- 若开启库存管理，后续出入库单完成时 OMS 库存会自动更新。

## 六、注意事项

::: danger 重点提醒
- **必填项**：所有带 **\*** 的字段必须填写，否则无法保存。
- **仓库编码**：必须以 ZTO + 省缩写开头，请按规范填写。
- **开启库存管理**：开启后，出入库单必须经过 WMS 实际出入库才能更新库存；OMS 库存不足时出库单状态会变为缺货且不推送单据到 WMS。
:::

::: warning 注意事项
- **启用报价确认**：仅当仓库需要对网点报价时才开启，否则保持关闭。
- **钉钉 token/secret**：配置后仅用于单据异常时的钉钉消息推送，不配置则不会收到推送。
- 如有权限问题，请先联系系统管理员配置基础数据管理全模块权限。
:::

## 七、常见问题

- **Q1：新增仓库后如何管理库存？**
  - **A**：在仓库管理中将 **开启库存管理** 设置为“是”。之后出入库完成时，OMS 会根据 WMS 实际出入库商品信息更新 OMS 库存。OMS 库存不足时，出库单状态会更新为“缺货”，不再推送单据到 WMS。
