# MD 优化 Skill 完善 Plan

## 问题总结

### 问题 1：标题格式不统一
**现状**：标题中嵌套 `**` 包裹，如：
```markdown
### **散单拣货实操演示**
## **发运**
## **1、中通冷链(小程序)**
```
**期望**：清理标题中的 `**`，只保留纯文本：
```markdown
### 散单拣货实操演示
## 发运
## 1、中通冷链(小程序)
```

### 问题 2：文章尾部"更多操作索引"内容需移除
**现状**：文章末尾包含导航索引块，内容如下：
```markdown
✨ 更多操作索引（持续更新中）✨

---

📌必知必会[系统访问、APP下载地址]...
1️⃣网点[01物料购买篇]...🚩[02客户下单篇(推荐)]...
2️⃣中心[01司机操作篇]...
3️⃣云仓[黑眼圈]整理中....
```
**期望**：整段删除（从"更多操作索引"到文件末尾的索引内容）

### 问题 3：图片未下载到本地
**现状**：图片 URL 是带 `Expires` 参数的阿里云 OSS 链接：
```
![image.png](https://alidocs2.oss-cn-zhangjiakou.aliyuncs.com/res/xxx/img/xxx.png?Expires=...)
```
`Expires` 过期后（约 7 天），图片会 403 无法访问。

**期望**：
- 下载所有远程图片到本地 `images/` 目录
- 替换 markdown 中的远程链接为本地相对路径
- 文件名使用原文档名+序号，便于管理

## 实现步骤

### Step 1：标题清理
在 `reformat_md.py` 的 `process_file` 添加 `strip_bold_in_headings` 函数：
- 匹配 `#{1,6}\s+\*\*(.+)\*\*` → 改为 `# \1`
- 支持嵌套（标题内有多个 `**` 块）

### Step 2：移除"更多操作索引"块
在 `process_file` 添加 `remove_trailing_index` 函数：
- 匹配 `更多操作索引` 关键词所在行
- 删除从该关键词到文件末尾的所有内容
- 包括 `📌`、`1️⃣`、`2️⃣`、`3️⃣` 等开头的索引行

### Step 3：图片下载
在 `dws_crawler.py` 添加图片下载逻辑：
- 遍历所有 markdown 文件
- 正则匹配 `![xxx](http...)` 图片链接
- 下载图片到同级 `images/` 目录
- 文件名：`{原文档名}_{序号}.png`
- 替换 markdown 中的链接为相对路径

**注意**：
- 图片 URL 中的 OSS 参数（Expires/Signature）需要保留或去掉？
- 需要处理下载失败的情况（记录到日志）
- 并发下载提高速度

### Step 4：测试验证
- 运行完整流程
- 检查标题是否统一
- 检查是否还有"更多操作索引"内容
- 检查图片是否下载到本地
- 检查 VitePress 预览效果
